<br/>				第二天，旅游团众人乘坐核动力破冰船，开始朝北极进发。<br/>　　<br/>　　按照旅游团的计划，从莫曼斯克抵达北极点，然后返航回来，一共是十天左右的路程。<br/>　　<br/>　　叶尘之前看过路线图，途中会路过狼王岛附近。<br/>　　<br/>　　不过这狼王岛，据说位于一座巨大的冰山之上，一般人根本上不去。<br/>　　<br/>　　……<br/>　　<br/>　　一晃眼，已经过去了三四天时间，<br/>　　<br/>　　此时离北极点，只有千里左右的距离，周围的气温急剧下落，远处还能看到绚丽无比的北极光。<br/>　　<br/>　　破冰船上，已经极少有人出来，不过此刻在一处不起眼的角落里，却有着两道身影。<br/>　　<br/>　　薛梦琪里里外外足足套了三件羽绒服，将自己包裹的严严实实，全身上下只有一张冻的红扑扑的小脸露在外面，哆里哆嗦道：<br/>　　<br/>　　“叶先生，您是在看那座冰山吗？我刚刚听一位苏国本地人说，那座冰山之上是一座仙岛，上面住着他们苏国的神仙，你说好笑不好笑？”<br/>　　<br/>　　薛梦琪见一身单衣的叶尘，一直在盯着那座巍峨的冰山观望，忍不住开口笑道。<br/>　　<br/>　　叶尘微微一笑，没有多说什么，<br/>　　<br/>　　从某种角度来讲，那个苏国人并没有说错，因为那座巨大冰川之上的小岛，便是狼王岛了。<br/>　　<br/>　　而狼王岛上住着的狼王科洛夫，也的确算得上是苏国曾经的守护神。<br/>　　<br/>　　薛梦琪接着又道：<br/>　　<br/>　　“不过话说回来，这座冰山离我们至少也有数百里的距离吧，我们居然还能看的一清二楚，那得有多大啊！”<br/>　　<br/>　　“如果不是周围冰山太多，船只无法抵达，我还真想过去看看！”<br/>　　<br/>　　叶尘微微叹了口气，道：<br/>　　<br/>　　“或许要不了多久，这座冰山便不存在了！”<br/>　　<br/>　　薛梦琪听到这话，顿时美眸一瞪，<br/>　　<br/>　　“你这话是什么意思？”<br/>　　<br/>　　叶尘正要开口，眉头猛然一皱，<br/>　　<br/>　　‘不好！魏无极他们似乎遇到了麻烦！’<br/>　　<br/>　　那日，和魏无极等人分开之时，叶尘在他们身上留下了一道精神印记，便于掌握他们几人的行踪。<br/>　　<br/>　　而就在刚刚，他忽然感觉到一丝强烈的精神波动，应该魏无极等人，遇到危险了！<br/>　　<br/>　　“罢了，好歹也都华夏之人，总不能见死不救！”<br/>　　<br/>　　自言自语了一句之后，叶尘直接冲着一旁的薛梦琪摆了摆手，<br/>　　<br/>　　“我还有事要办，咱们就就此别过吧！”<br/>　　<br/>　　说完这话，叶尘不等薛梦琪回答，直接纵身一跃，整个人便已经飞出栏杆之外，然后迅速坠落了下去。<br/>　　<br/>　　“叶先生！”<br/>　　<br/>　　薛梦琪顿时吓了一跳，当即飞扑到栏杆跟前，朝下方一望。<br/>　　<br/>　　只见，叶尘犹如一片羽毛般，落入下方的海面上，甚至海面上没有出现一丝的涟漪！<br/>　　<br/>　　而且，叶尘并没有在海面上停留，身体微微一晃，人已经奔出数百米开外。<br/>　　<br/>　　薛梦琪无比惊叹的同时，心里也一阵失落，完全没有反应过来，叶尘便已经离他而去。<br/>　　<br/>　　而就在这时，一道熟悉的声音，陡然在薛梦琪脑海之中响起，<br/>　　<br/>　　“薛梦琪，这次多谢你了，以后你若是遇到麻烦，可以去云州苏叶集团！”<br/>　　<br/>　　薛梦琪听到这道声音，顿时欣喜不已，不过紧接着，却又大为失落。<br/>　　<br/>　　“唉！今日一别，也不知道下次见面是什么时候了！”<br/>　　<br/>　　薛梦琪望着叶尘渐行渐远的身影，感慨了片刻，猛然想到了什么，顿时娇躯一震，当即将随身携带的画板和画笔取了出来，然后趴在栏杆之上，冒着严寒，开始迅速绘画起来。<br/>　　<br/>　　薛梦琪的画工确实了得，不一会的功夫，画板之上，已经出现了一副惟妙惟肖的图案，<br/>　　<br/>　　一道潇洒的少年身影，在辽阔的冰洋上方，踩着冰洋上的一座座冰山，朝北极光的方向狂奔而去……<br/>　　<br/>　　等薛梦琪一气呵成，将整幅画作绘画完毕，此刻的叶尘，早已经淹没在远处的极光之中。<br/>　　<br/>　　薛梦琪望着眼前这幅，自从学画以来，自己最为传神的一次创作，顿时喜不胜收，不过紧接着，却又莫名的一声长叹。<br/>　　<br/>　　……<br/>　　<br/>　　狼王岛，方圆数十里，虽然不算太大，却也不算小。<br/>　　<br/>　　而且，此岛位于冰川之上，岛上更是冰川密布，不但崎岖无比，那些冰川更是犹如一把把巨大而锋利的匕首，便是那些训练有素的超级特种兵，一旦进入这种地方，也是九死一生，根本不可能进入中心地带。<br/>　　<br/>　　而此时，<br/>　　<br/>　　小岛西北部，一处冰谷之中，<br/>　　<br/>　　“魏无极！说出叶狂仙的下落，否则今日这里，便是你们的葬身之地！”<br/>　　<br/>　　说话之人，是一名满头金发的白人男子，手里拿着一把硕大的银色巨锤，不但悬空而立，通体上下竟然被一道道细微的雷电所笼罩，犹如雷神降世。<br/>　　<br/>　　而在山谷的另外一边，还有一名身材高瘦的白人女子，足尖轻轻落在一座冰川的峰顶之上，任凭周围的寒风呼啸，却稳如泰山，<br/>　　<br/>　　至于下方冰谷之中，一片狼藉，赫然正是魏无极一行七人，而且个个身上挂彩，显然之前应该经过了一番苦战。<br/>　　<br/>　　与此同时，周围远处的冰川之上，还有数十名围观之人，个个气势极强，显然都不是泛泛之辈。<br/>　　<br/>　　赵无极抬头望着上方的那白人男子，冷冷一笑，<br/>　　<br/>　　“米国超能局，雷神宙斯！暴风之神安娜！果然名不虚传！可惜啊，你们还远远不够资格挑战叶先生，叶先生没来，你应该感到庆幸，否则你们还有命在吗？”<br/>　　<br/>　　原来，<br/>　　<br/>　　眼前的这两个人，分别是米国超能局排名第一、第四的雷神宙斯和风暴之神安娜。<br/>　　<br/>　　雷神宙斯具备操控雷电的能力，加上米国超能局为其量身打造的秘银锤，是公认的仙人一级的高手。<br/>　　<br/>　　至于风暴之神安娜，则具备操控暴风的能力，实力虽然比宙斯要弱上不少，可是速度奇快无比，在神境之中，也算是佼佼者。<br/>　　<br/>　　相比之下，赵无极等人，在神境之中，也只能算是中等水平，又岂是两人的对手？若非对方的目标不是他们，此刻早就把他们杀了。<br/>　　<br/>　　说起来，赵无极等人也很郁闷，他们自知实力太弱，所以踏入狼王谷的时候，故意绕到小岛西北角一个最为偏僻的位置，却怎么也没有想到，才刚一登岛，就暴露了行踪。<br/>　　<br/>　　不但米国超能局的两位主神来了，连其他各方势力，也全部被吸引了过来。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是我意如刀，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。请您关注微信公众号：dazhuzaiyuedu（长按三秒复制）书友们快关注起来吧！<br/><br/><br/>
