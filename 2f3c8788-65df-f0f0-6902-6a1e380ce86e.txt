        “话已至此，阿景你好好考虑我所说的话。”傅夜沉接着说，然后拽着童筱颖的手，将她强行带走。
<br/>
<br/>        童筱颖感觉自己是被傅夜沉连拖带拽地塞进副驾驶座上的，就连他替她扣安全带的动作也十分粗鲁，就像是在绑着一个不可饶恕的犯人。
<br/>
<br/>        他该不会误以为，她试图安安分分地当他的太太，是因为她刚刚所说的那些话吧？
<br/>
<br/>        “大哥，我……”
<br/>
<br/>        “我不想听！”傅夜沉呵斥住童筱颖欲要说话的话。
<br/>
<br/>        以她之前的性子，估计她又会说，她就是那样的想法，希望他能放过她，跟她离婚！
<br/>
<br/>        “大哥，你听我说，我其实……”
<br/>
<br/>        “我现在需要安静。”傅夜沉再次打断童筱颖的话，“哐”地一声，关上了副驾驶座的车门。
<br/>
<br/>        童筱颖深吸了口气，决定暂时闭嘴。
<br/>
<br/>        他在生气，证明她猜得没错。
<br/>
<br/>        如果她这个时候强行解释，说不定会适得其反，那还是等他气消了，她再解释吧！
<br/>
<br/>        这一路上，两人彼此沉默不语，僵持得过分安静。
<br/>
<br/>        到了私宅后，童筱颖抱着小爱下了车，傅夜沉将小爱的东西全部挪下车后，二话不说地重新回到了驾驶座上，脚下一踩油门，风一般地开车急驰而去。
<br/>
<br/>        童筱颖只能独自一个人，将小爱那一大堆的东西，慢慢地往屋里搬。
<br/>
<br/>        傅夜沉这一走，就是一整晚，彻夜不归。
<br/>
<br/>        她主动给他打电话，他只说他很忙在加班，便将她的电话给挂断了。
<br/>
<br/>        而这一夜，童筱颖发觉自己竟然失眠了，而且还是辗转难眠，满脑子都是傅夜沉。
<br/>
<br/>        翌日，傅氏集团大厦。
<br/>
<br/>        偌大的会议室里，坐在主位上的傅夜沉剑眉微蹙，俊脸上满是严肃地看着手里的策划案。
<br/>
<br/>        这顶级大Boss的严肃，紧逼得在座的每一位高管也跟着绷着一张脸，一个个正襟危坐，连个屁都不敢乱放地端坐在会议桌前。
<br/>
<br/>        “啪”地一声，傅夜沉将手里的策划案往会议桌上一扔，呵斥道：“重做！”
<br/>
<br/>        简短的两个字震慑得所有人不寒而栗。
<br/>
<br/>        也不知道傅大Boss今天是怎么呢？跟吃了火药似地，一上午都在实施冷暴力，一点瑕疵都不允许，一旦发现所有的文件就要回炉重造，让全公司上下叫苦连天。
<br/>
<br/>        会议结束的时候，等所有人都出去了，落以柔才犹豫着走过来，对依旧坐在主位上发电子邮件的傅夜沉，颔首询问：“傅总，夫人她在一楼大厅等你，需不需要把她带到总裁办去？”
<br/>
<br/>        “打发她走。”傅夜沉冷着脸说。
<br/>
<br/>        落以柔顿时一脸诧异。
<br/>
<br/>        虽然，她能继续留在这里为傅夜沉工作，听师父郑忠威说，是童筱颖开了金口跟傅夜沉求的情，但是她一点都不想谢谢她童筱颖。
<br/>
<br/>        因为，她的直觉告诉她，她先前之所以被傅夜沉调走，十有八九也跟童筱颖在傅夜沉身边嚼舌根有关。
<br/>
<br/>        想到这里，落以柔毫不犹豫地转身准备去回话，却被傅夜沉突然叫住了：“等等。”
<br/>
<br/>        落以柔转身看向他，见周遭已无旁人，不假思索地亲昵起来：“大哥还有什么吩咐？”
<br/>
<br/>        “跟她说，要她以后没事，就不要来公司。”
<br/>
<br/>        “诶？”落以柔微微诧异，但转念一想，忍不住嘴角微扬地应道，“是。”
<br/>
<br/>        落以柔现在虽然在童筱颖面前没有以前那么嚣张了，但是她却一字不漏地把傅夜沉刚刚交代给她的话，有板有眼的复述给了童筱颖听。
<br/>
<br/>        她知道落以柔不喜欢她，但是落以柔也不会“假传圣旨”。
<br/>
<br/>        叫她以后没事不要来公司，这一定是傅夜沉的原话。
<br/>
<br/>        童筱颖微微垂着眸子，悻悻地离开了傅氏集团大厦。
<br/>
<br/>        总裁夫人失宠，总裁夫人被打入冷宫，这种谣言蜚语，瞬间在公司里的女同事们的私密群里传开。
<br/>
<br/>        大家都心知肚明，她们的傅大Boss一定不缺女人，虽然，被傅夜沉公开的女人只有童筱颖一个，确实惹来几个暗恋傅大Boss的女人的羡慕嫉妒恨，所以当她们得知童筱颖被傅夜沉冷落时，也格外大快人心。
<br/>
<br/>        落以柔就是其中的一个，但结果，她的这份开心还没尝到甜头，傅夜沉便让她帮他预订餐厅，晚上他要陪“新欢”吃晚饭。
<br/>
<br/>        至于到底是不是“新欢”，傅夜沉没有明说，是落以柔心里乱猜的。
<br/>
<br/>        童筱颖回去后，思前虑后，总觉得自己和傅夜沉之间不能就这么“冷战”下去。
<br/>
<br/>        她和昼景的婚姻，就是在彼此的“冷战”中结束，这次，她不想重蹈覆辙。
<br/>
<br/>        他今晚肯定又不会回私宅，那么他一定是在公司过夜。
<br/>
<br/>        那她等夜幕降临之后，再去找他好了。
