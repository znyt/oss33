　　“黑蛇”监狱，表面上隶属于戴伦军事资源公司，而事实上，它是美国政府的产业。
<br/>　　“黑蛇”监狱由摩洛哥政府军看守，除了大量的摩洛哥政府军看守外，还有戴伦军事资源公司旗下的雇佣兵，以及各种陷阱和机关，监狱墙壁是20英寸厚的钢筋混凝土结构，墙壁上方是瞬间能把人电成焦炭的高压电网。
<br/>　　监狱下是一层10英寸厚的脱氧钢，想要通过挖地道的方式越狱，根本就是不可能的事情。
<br/>　　因为被关进“黑蛇”监狱中的人，全都是被美国政府通缉的要犯，他们或者是极度危险的职业杀手，或者是高智商的金融罪犯，或者是利用病人进行活体病理试验的医学疯子，或者是手眼通天的军火贩，以及策划、甚至进行过恐怖活动的恐怖分子！
<br/>　　“黑蛇”监狱中的犯人没有任何一个是简单的角色，他们中间任何一个如果越狱出去，都会造成巨大的震动。
<br/>　　这是Derrick第一次如此迫切的想要进入监狱！不过，Derrick现在还缺少一个被送进“黑蛇”监狱的理由。
<br/>　　“维伦恩”生物制药公司不仅自己制毒，而且还贩毒，掌握着北美洲绝大多数的贩毒网络，Derrick进入“维伦恩”生物制药公司也不是一天两天，对“维伦恩”生物制药公司所掌握的贩毒网络也有一定的了解，所以Derrick很快就想到了一个主意，一个能让自己被送进“黑蛇”监狱，躲过玛丽科娃手下杀手的办法。
<br/>　　Derrick本身就是在通缉的要犯，如果再涉及到贩毒的话，那就很有可能被送进“黑蛇”监狱。
<br/>　　在第一批“水晶珠”被投入到市场，观察效果的时候，Derrick记下了几个收货人的电话。躲进商场厕所后，Derrick按照记忆，拨通了一个电话，“刀疤熊，有一批刚下生产线的‘水晶珠’，有兴趣吗？”
<br/>　　刀疤熊，是玛丽科娃的一个下线，也是距离Derrick最近的一个下线。刀疤熊是个俄罗斯人，因为身体健壮像一头熊，加上胸前有一道长长的刀疤，所以有了“刀疤熊”这个绰号。
<br/>　　靠在沙发中，正闭着眼睛享受来自跪在自己面前女人服务的刀疤熊，听到电话中Derrick的话，眼睛突然睁开，迸射出一道精光来。“水晶珠”刚被送到他手里的时候，他还是保持着怀疑，可是当有人在使用后所表现出来的欲罢不能后，刀疤熊心中所有的怀疑全都被打消了，这玩意儿实在太强了，刀疤熊没忍住也服用了一颗“水晶珠”，当了一回“战神”！
<br/>　　贩毒的人一般不会允许自己，还有自己身边的人吸毒，他们贩毒，所以也比一般人更了解毒品的危害。刀疤熊知道“水晶珠”是一种全新的合成型毒品，可是这种东西对于一个男人的诱惑实在太大了，以致于刀疤熊没有能抵抗住“水晶珠”的诱惑，沉沦在“水晶珠”带给他的成就感中。
<br/>　　刀疤熊一把按住自己面前女人的头，坐直身体，问道：“有多少量？”
<br/>　　“大概150粒左右。”Derrick回答道。
<br/>　　刀疤熊皱眉，显然是对这么一点数量的“水晶珠”感到不满意，这些量实在太少了。刀疤熊最开始的时候一周只用一次“水晶珠”，可是后来，刀疤熊发现一星期一粒的量已经无法满足自己，他自己一个星期就需要两粒“水晶珠”，而他还有下家也对“水晶珠”格外的渴求。
<br/>　　电话那头的沉默，并没有让Derrick失去耐心，等待了几秒钟后，继续说道：“这个数量已经不少了，而且是升级后的‘水晶珠’，效果只会比之前的更好。”
<br/>　　“时间？”刀疤熊问道。
<br/>　　“两个小时后，老地方交易。”Derrick回答道。
<br/>　　“这似乎跟以前的规矩不一样。”刀疤熊很谨慎，虽然对“水晶珠”非常渴求，但是刀疤熊依旧保持着该有的冷静和谨慎。
<br/>　　“如果你怀疑，可以选择不要这批‘水晶珠’，但是我可以保证，在此以后，你绝对不会再能用这么低的价格拿到‘水晶珠’！”Derrick说道。
<br/>　　刀疤熊在犹豫了几秒钟后，最终他的冷静和谨慎还是没能抵御住“水晶珠”对他的诱惑，答应了Derrick的交易。
<br/>　　Derrick从“维伦恩”生物制药公司离开的时候，带走了一些升级后的“水晶珠”，现在正好可以用来对玛丽科娃捅一刀。结束了跟刀疤熊的通话后，Derrick拨通了DEA（缉毒局）的电话，以举报人的身份举报了两个小时后，要进行的毒品交易。
<br/>　　……
<br/>　　西南部，某废弃食品加工厂。
<br/>　　Derrick驾驶着汽车进入工厂，拉开储物格，从里面拿出一把手枪，轻轻拉开套筒，看了眼枪膛中已经上膛的子弹，把手枪插在后腰，坐在汽车中安静的等待着，没过多久，几辆汽车依次驶入工厂。
<br/>　　中间那辆防弹路虎中，刀疤熊搂着那个刚跟他酣战完，双脚还发软的女人走下走了下来；其余几辆汽车中，也是走下十几名携带着自动武器的保镖。
<br/>　　Derrick推开车门走下汽车，看了眼刀疤熊和被他搂在手里的女人，脸上露出一丝了然的神色。他是“水晶珠”的创造者，是对“水晶珠”最了解的人，所以看到刀疤熊，就已经知道他已经沉沦在了“水晶珠”带给他的不一样的征服感中。
<br/>　　“怎么是你？”刀疤熊看到来跟自己交易的是Derrick，立刻皱起了眉头，问道：“布莱克呢？”
<br/>　　“布莱克现在还拿不到‘水晶珠’。”Derrick说道：“目前能接触到‘水晶珠’的，除了霍斯曼德和queen以外，就只有我了。钱呢？”
<br/>　　刀疤熊对着跟在自己身后的保镖勾了勾手指，那个保镖把一个银色手提箱放在汽车引擎盖上，打开手提箱，里面整齐的放着一沓一沓100面值的美金。
<br/>　　Derrick从口袋里拿出一个瓶子，对着刀疤熊晃了晃；刀疤熊朝着身边的保镖递了一个眼神，保镖合上手提箱，拎起手提箱向Derrick走去。
<br/>　　就在保镖从Derrick手中接过瓶子，而Derrick从对方手里接过钱箱子的时候，几颗闪光震爆弹被扔进工厂中……
