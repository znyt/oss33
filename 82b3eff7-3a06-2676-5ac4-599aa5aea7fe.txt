陆寒霆精健的胸膛像一堵墙，任她怎么推搡都纹丝不动，他低眸看着她恼怒的澄眸，勾了一下薄唇，“我哪里讨厌了，你说清楚。”
<br/>
<br/>        就是…整个人都讨厌！

<br/>        夏夕绾嗅到了他身上那股干净清冽的男人气息，被他禁锢在怀里，听他问这样的混账话，她的双腿竟然不自觉的发软，想要向下滑。
<br/>
<br/>        “陆寒霆，我们已经离婚了，是你先出轨的，是你先不要我的，我现在有没有跟别的男人跟你有什么关系，你自己不要的东西也不许别人动，究竟是怎样阴暗的想法…唔！”

<br/>        陆寒霆低头，狠狠堵住了她的红唇。
<br/>
<br/>        他吻得很用力，像凶猛的野兽扑向了他觊觎已久的可口猎物，恨不得将她吞入食腹。

<br/>        他没有不要她。

<br/>        他也没有出轨。

<br/>
<br/>        他什么都给不了她，只能看着她为了他试毒，一次次的来讨好迁就他，他怕了，怕自己给不了她明天和未来。
<br/>
<br/>        夏夕绾要气疯了，她觉得这个男人真是不可理喻，那时他那样绝情的将她推开，等离婚了又来招惹她。
<br/>

<br/>        夏夕绾狠狠的咬下了他的舌尖。

<br/>
<br/>        很快，他就吻到了一股苦涩的味道，夏夕绾哭了。

<br/>
<br/>        他把手缓缓松开。
<br/>

<br/>        夏夕绾进了电梯，迅速消失在了他的眼前。

<br/>        这时他觉得喉头一股腥甜涌了上来，下一秒一口血就喷了出来。
