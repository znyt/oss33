        &lt;h3&gt;第三卷第九十二  章  陈子昂的危机&lt;/h3&gt;
<br/>
<br/>        陈子昂将全国都跑了一个遍。
<br/>
<br/>        对于耗材她是很重视的，这是一场划时代的改变，南方虽然有些小作坊，但是没有成气候，她将品牌推出市场，像她所想，我们也有了自己的耗材品牌，会占领一席之地。
<br/>
<br/>        耗材的品牌名称一直还没有确定下来，商标注册也没有下来，她一直在等着江俞轩，征集的名称好几个，其实都很有意义。
<br/>
<br/>        在各个分店，她都会拿一些给客户免费试用，耗材从研发和投入已经花费了不少资金。
<br/>
<br/>        而且他们研发的系统也花费了不少的资金，前几天陈思宇给她打电话说资金吃紧了。
<br/>
<br/>        接到陈思宇的电话，她才惊觉他们的投入太大，场面铺的太开了，可是手中的项目哪一个也不能停下来。
<br/>
<br/>        她想到了投资五姐的超市的哪一点资金，看能不能抽回来，但是根本就是杯水车薪，解决不了问题。看来只能再找银行借贷了。
<br/>
<br/>        她在外地出差，只能等回去再说。
<br/>
<br/>        “思宇，你和盛夏先看看银行，能不能想想办法。我周末回来，我们加班商量一下。”
<br/>
<br/>        “好的，我们先去看看银行，最近银行放款不是很顺利。”
<br/>
<br/>        “你和盛夏核算一下，最近的投入和要回收的情况。”
<br/>
<br/>        “好的，你什么时候回来？”陈思宇在电话那头答应着，电话里面很吵。
<br/>
<br/>        “周五返程，周六加班吧。”
<br/>
<br/>        如果资金不能到位，元旦过后还是不能推向市场，而且名称还没有确定，陈子昂叹了口气。
<br/>
<br/>        谁说事业越大，领导越轻松？
<br/>
<br/>        她每天飞来飞去，研究方案和下面的工作人员打成一片，和大家一起奋斗，有时候忙的晚上才吃饭，中午饿了就啃一块面包就一杯白开水。
<br/>
<br/>        她犯了一个致命的错误，当然大家也是没有经验都是忽及了的，耗材在推向市场之前就应该有名字的，她却想等着江俞轩来决定，想着江俞轩在公司辛苦了这么多年，算是回馈江俞轩对公司付出。
<br/>
<br/>        商标注册其实已经开始了，只是迟迟没有确定名称。
<br/>
<br/>        这不是一个企业领导所为，陈子昂有些感情用事了。
<br/>
<br/>        她知道江俞轩最近很忙，西城的项目是一个大项目，对于江俞轩来说，具有一定的意义，所以当她给江俞轩发邮件，江俞轩没有回她的时候，她也没有催促江俞轩。
<br/>
<br/>        看来她不找江俞轩也不行了。
<br/>
<br/>        江俞轩和江俞琪正在和西城南郊新区和杨老总一起谈论学校的建设情况，他们要在南郊建立一个第一学校，最起码要在建筑上突出特色。
<br/>
<br/>        江俞轩的电话响起来了，江俞轩看了一下电话，是陈子昂的。
<br/>
<br/>        最近这一段时间很忙，他们基本都是各自忙各自的，都没有联系了，那天看了陈子昂的思路之后本来想着要给陈子昂去电话的，一忙又忘记了。
<br/>
<br/>        陈子昂打电话估计是有事情。
<br/>
<br/>        他看了一眼姐姐，指了指电话，江俞琪点点头。
<br/>
<br/>        “杨总，您们先聊着，我去接个电话。”
<br/>
<br/>        “好，去吧。”杨总是南方人，身材瘦小，但是能力超强，为人和善，但在工作上兢兢业业，比较严厉。
<br/>
<br/>        “子昂，有事情吗？”
<br/>
<br/>        “俞轩，你最近还忙吗？”
<br/>
<br/>        “比较忙，西城的项目下来了，我脱不开身了。”
<br/>
<br/>        “耗材常规的型号基本都出品了，而且试用的情况不错，可以推向市场了，但是我忘记了一件事情。”
<br/>
<br/>        “什么事情？”
<br/>
<br/>        “耗材的品牌还没有确定，商标还没有注册。”
<br/>
<br/>        “这么大的事情你怎么会忘记？”
<br/>
<br/>        “我想等你来确定品牌名称。”陈子昂的声音有点小，感觉像一个做了错事的孩子。
<br/>
<br/>        “你傻子哟，怎么不早点告诉我？”江俞轩虽然嘴上骂着陈子昂是傻子，但是心里还是很感动，他知道陈子昂在等他来做决定。
<br/>
<br/>        “我不是看你忙嘛？”
<br/>
<br/>        “这个是大事情，再忙你也得告诉我。赶紧，将最近的工作情况给我汇报一下。”江俞轩大大咧咧的，最近好多人给她汇报情况。
<br/>
<br/>        “我最近将各地的网店全部铺开了，只等产品推向市场。”
<br/>
<br/>        “没品牌你推什么推？真是服了你，平时决断那么干脆利索的，这一次怎么就忽及了？”江俞轩好一通唠叨。
<br/>
<br/>        “商标注册其实早都在进行了，就是等你来定品牌名称。”
<br/>
<br/>        “我的陈老大，你要学会没有我在身边的日子，不能事事都要想着和我商量。品牌名称有没有合适的？”江俞轩说。
<br/>
<br/>        “我.......”陈子昂听见江俞轩的话，心里一阵难过，这么多年了，她事事和江俞轩商量，她已经习惯了，江俞轩就是她的左膀右臂啊。
<br/>
<br/>        “品牌名称有没有合适的？你有想过吗？”
<br/>
<br/>        “倒是征集了几个名字，等你确定。”
<br/>
<br/>        “那就推迟进入市场吧。等品牌确定了再进入市场，等我去北京，我们见面聊。公司最近运行还好吧？”
<br/>
<br/>        “还好，陈思宇给我打电话说资金紧张，如果不想办法怕资金链会断裂。”
<br/>
<br/>        “子昂，要不引进投资吧？如果你同意了，我来想办法？”江俞轩问陈子昂。
<br/>
<br/>        “好！”陈子昂毫不迟疑的回答，本来在app软件上寒伯安他们也入股了，再引进投资也不是不可以，就看有没有人看中这个市场，毕竟目前还算是空白。
<br/>
<br/>        “那先就这样了，我先忙了，我们回北京见。”江俞轩不能让杨东他们等的时间太长了，就早早结束电话，他的心里有个想法，让寒伯安和郝景文也入股他们公司。
<br/>
<br/>        耗材市场确实是一个很好的行业，后期如果全面打开的话，市场容量也是相当可观的，随着社会的发展，需求量也会越来越大。
<br/>
<br/>        陈子昂只能推迟耗材推向市场的计划。
<br/>
<br/>        还有资金链如果断裂，她所做的一切努力都将会面临巨大的挑战。
<br/>
<br/>        “盛夏，你看看最近最早的回款是在什么时候？”陈子昂给凤盛夏打电话。
<br/>
<br/>        “最早的项目回款都在六个月之后。”凤盛夏最近也是忙的焦头烂额，她们财务部的都成了催款专业户了，可是回来的款杯水车薪，不够耗材折腾的。
<br/>
<br/>        她们的耗材都是才开始，所有的采购都是现款，而每个单位给他们的回款都是有期限的。
<br/>
<br/>        她拆西墙补东墙已经快一个月了。
<br/>
<br/>        “你和思宇在想想办法，顶一段时间，我让我五姐先打一部分款给我们公司。”
<br/>
<br/>        “陈总，就是你让你五姐将你之前投资的款都打回来只能是九牛一毛。”
<br/>
<br/>        “资金缺口这么厉害？”
<br/>
<br/>        “你要将耗材推上市场，现在都是没日没夜的加紧生产。光耗材销售体系软件研发都花费了几百万。”
<br/>
<br/>        “知道了。”
<br/>
<br/>        “另外，所有的证件基本都会下来，就是品牌名称没有确定，你知道的。”
<br/>
<br/>        “名称确定之后，那些东西下来得多久？”
<br/>
<br/>        “最快也得半个月。”
<br/>
<br/>        “为什么提前没有人给我说？”
<br/>
<br/>        “我提醒过你，你说不碍事，等江总。”凤盛夏有些无奈。
<br/>
<br/>        陈子昂头疼了，光开的网店其实也花费了不少钱。她最近确实犯了一个大错，没有江俞轩在的日子还真是不行。
<br/>
<br/>        公司也有高层，但是决定方面全靠他们俩个商量。
<br/>
<br/>        说白了，他们的公司还要完全进入正轨化。
