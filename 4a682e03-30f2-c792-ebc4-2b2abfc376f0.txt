　　第553章
<br/>　　黑袍大汉表情阴沉，脸色极其不好看。当然，他的脸色一向不好看，美其名曰沉默寡言，其实是不太会说话。
<br/>　　某种程度上，程大雷当真猜对了这帮人的来路。他们并非帝国诸侯派出的高手，也并非来自海上的流贼巨寇。他们，来自戎族。
<br/>　　黑袍大汗名唤燕不归，是戎族有数的高手。拜一个高人为师，修得一身惊天艺业。这次千里迢迢来到琅琊城，便是奉命盗取山水经。
<br/>　　所图，自然不小。
<br/>　　而受伤的女子，名叫黑贞，与他师出同门，算是燕不归的小师妹。
<br/>　　黑贞的一身本事，大部分都是燕不归传授，算是代师传艺。朝夕相处中，早已情根深种，只不过燕不归不善言辞，一直未说明而已。
<br/>　　昨夜闯山盗书，一切都还谈得上顺利，偏偏是黑贞中了张六一棍，实在是伤了黑贞的要害。
<br/>　　张六能在海上为非作歹许多年，一身本事自然不弱。他几乎是可以和燕不归打平的，最后一计追魂棍，更是压箱底的绝招。
<br/>　　此刻黑贞趴在棺材板上，赤果的美背出现一抹瘆人的黑。淤血凝结，不得散发，从来医武相通，以燕不归的本事，自然看得出来，若不及时得到医治，她一条命便要交待在这里。
<br/>　　燕不归的脸色愈发难看。
<br/>　　“大人，现在城门已封锁，接下来，定会全城搜捕。山水经十分重要，我们还是及早脱身要紧。”柳褐岩道。
<br/>　　燕不归拧起眉头，心中自然明白眼前情况的危急。昨夜闯山盗书，也见识了书院高手的实力，不早早想办法脱身，或许便脱不了身。但黑贞的伤势极为严重，实在不知还能撑多久，更是受不了长途颠簸。
<br/>　　“找琅琊城最好的大夫，与她疗伤。”燕不归。
<br/>　　“可大王安排下来的事？”柳褐岩提醒道：“我们还是及早将书送回去要紧。”
<br/>　　“人命为大，总不能让她死在帝国的土上。找到大夫，为她排出淤血，她还能活。”
<br/>　　柳褐岩没有说话，心里却不是很认同燕不归的看法。戎族是不太把人命当真的，各个部落打来打去，什么杀父之仇，夺妻之恨，都可以假装没发生过，哪里来得人命关天。
<br/>　　但燕不归是这次行动的指挥者，本身功夫又硬，谁也不敢忤逆他。
<br/>　　“大人，黑贞姑娘的情况，一般大夫怕是治不了。必须由当世名医动手。”柳褐岩道：“我在城门打探消息听说，那华家名医来了琅琊城。如果由他们动手，当能从阎王爷手中，把黑贞姑娘的命救回来。”
<br/>　　“华家……”燕不归眼睛一亮，心底重新升起希望：“现在他们在哪里？”
<br/>　　“应该是已被迎进万卷山。”
<br/>　　“好，我今夜再探万卷山，把华家的大夫抓回来给小师妹治病。”心里有了奔头，燕不归脸上的精气神又活泛起来。
<br/>　　“大人，我们昨夜去过一次，今夜他们必定加强了防范。况且抓人不等于盗书，大人还是要三思。”
<br/>　　柳褐岩琢磨着用词，说完发现燕不归根本没有搭理他。燕不归便是如此脾气，只要话题在黑贞身上时，他愿意多说两句话，其他事他根本不屑于和人商量。
<br/>　　他打定的主意，一般是不会因别人而更改的。更何况，这件事关乎黑贞的生死，他已打定主意要再探万卷山，危险什么的，根本不是他所关心的。
<br/>　　……
<br/>　　初更时分，程大雷也开始了自己的行动。他将绑腿腰带束紧，抓起匹夫剑。
<br/>　　“程当家……”刘发财表情有些无语：“不若您再想想，昨天藏书楼才被偷了，书院就算再没有脑子，也不可能让人再偷一次，您这是玩火呀。更何况，夫子已经说了，没有副本，没有存档，您又何必白费力气。”
<br/>　　“好啦，读书人的话也能信。这世道怎么会变成这个样子，还不是被这帮读书人害的。”程大雷紧了紧腰带：“你也说了，他们昨天才被偷过，怎么会想到今天又有人来偷。”
<br/>　　刘发财知道自己在程大雷面前没有什么话语权，程大雷没有拉着自己一起去，他已经谢天谢地了。
<br/>　　“那么，祝您马到功成，手到擒来，我在这里恭候您的好消息。”
<br/>　　程大雷冲他摆摆手，蒙上了脸，轻轻推开窗户，确认左右无人，一踮脚窜了出去。
<br/>　　程大雷提前已经踩好了点，毕竟是做贼出身，对于这样的活计也并不陌生。如今整个书院的建筑布局已印在他心里。在出了昨夜的事情后，书院的确加强了防守，出动了很多高手。但这对程大雷来说不算什么，他已提早摸清岗哨的位置。
<br/>　　某种程度上，书院的选址也帮了他。当初建造这座书院的人，不知道抽什么风，选择临着大海。夜来涛声阵阵，书院耳聪目明的高手是多的，但他们未必能在阵阵涛声中分辨出风吹草动。
<br/>　　身无彩凤双飞翼施展开来，本是绝世功夫，单比轻身功夫，世上怕没几个人能超过程大雷。他几乎不费吹灰之力，便潜入书院的藏经楼。
<br/>　　接下来的动作便不得不小声些，若这次被人发现，那么昨夜杀人盗书的事情都得赖在自己身上。
<br/>　　藏书楼一共七层，按程大雷的理解，自然越往上，储存的书籍便越珍贵。所以对一二层的藏书他信马观花，扫过就算，越往上，脚步便慢了下来。
<br/>　　楼中的书籍是多的，介质也不一样，有的刻在竹上，有的印在布上，还有些刻在石碑上。程大雷从一楼翻到七楼，结果一无所获，没有寻到山水经的副本。
<br/>　　站在七楼的窗前，程大雷皱眉思索。莫非夫子没有撒谎，世上山水经当真只有一份。这……不符合读书人的脾气啊。
<br/>　　假若夫子并未撒谎，山水经没有副本，那么自己想得到这山水经，有且只有一个法子，便是找到那伙盗书贼。
<br/>　　可想找到他们……谈何容易。
<br/>　　正这个时候，程大雷看到山上一道黑影一掠而过，正奔着山上的楼宇而来。
