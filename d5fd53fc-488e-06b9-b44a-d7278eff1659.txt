
<br/>　　<br/>　　    庄园会所里面，陈重接完他三叔的电话之后，眼睛里就蹿火了，阴阴的盯着地上那青年，抬腿一脚就把旁边的桌子给踹翻了：“真他么窝火，脸都他么丢姥姥家去了！”<br/>　　<br/>　　    陈重确实很憋气，因为他谁的面子都能不给，但他三叔亲自打电话过来了，那他是真一点办法都没有了，没啥，就因为整个陈家都是以他三叔为中心的，他能打电话过来就说明，有不能拒绝理由，哪怕是陈重受了这么大的委屈也不行。<br/>　　<br/>　　    这就是一个大家族的规矩，领导者有着绝对的话语权，是不允许被忤逆的。<br/>　　<br/>　　    午桥脸也耷拉下来了，扭头跟王生说道：“事整复杂了，好像水挺深呢？”<br/>　　<br/>　　    老马拧着眉头跟陈重说道：“怎么回事，必须得放人啊？”<br/>　　<br/>　　    “三叔给我打电话了，没商量的余地，这个人不能留了，得给放了才行……”<br/>　　<br/>　　    地上那青年顿时豪放了，跟一条小狼狗似的蹿了起来：“怎么样，我说什么来着？你们抓我啊，那真不行，怎么抓来的就得怎么给我送出去，多留我一天你们都烫手”<br/>　　<br/>　　    老马叹了口气，懊恼的说道：“早知道昨天晚上就连夜狠收拾他一顿好了，现在给放出去了，这他么的说出去也真难听啊”<br/>　　<br/>　　    几个人正商量的时候，陈重手里的电话又响了，他烦躁的看了眼号码发现是个陌生的号，干脆直接就给掐断了，这时的陈重心里面窝着一股火撒不出去看啥都心烦，但没想到的是他刚掐断电话，那头又打了过来，然后再掐再打，最后给陈重整的都要火山爆发了，他接起来后就开骂了。<br/>　　<br/>　　    “你他么要找死早投胎是不是，打个没完了呢？”<br/>　　<br/>　　    “陈先生，你昨晚是不是抓了跟吴洋在一起的那个人？”电话那头的人根本都没关他的破口大骂，电话接起后他直奔主题的就问了一句。<br/>　　<br/>　　    听见对方的声音，陈重开始有点发愣，因为听着很耳熟却想不起来是谁了，足足停顿了能有十几秒，才犹豫着问道：“是惊蛰啊？”<br/>　　<br/>　　    “唰”旁边午桥眼角顿时抽搐了两下，这是个几乎已经要被他给抛到脑后面去的名字没想到在这突然冒了出来。<br/>　　<br/>　　    王生诧异的问道：“惊蛰？王惊蛰么，不是听说他被关到阴曹地府里出不来了么？”<br/>　　<br/>　　    “不一定，可能是重名吧？听听再说……”午桥狐疑的说道。<br/>　　<br/>　　    王惊蛰在电话里“嗯”了一声，说道：“是我，陈先生你是不抓了跟吴洋在一起的人？”<br/>　　<br/>　　    “对，人是在我这里，不过……”陈重刚叹了口气，要说把人给放了，王惊蛰紧接着就问道：“你在哪里，我过去一趟”<br/>　　<br/>　　    陈重一愣神，有点没太明白王惊蛰是啥意思，他之前早就品出来了，几次事件之后王惊蛰都没有跟他掺和在一起的意思，甚至几次出手都是被动的原因，现在搞不清楚为啥对方怎么还打听起这事来了。<br/>　　<br/>　　    “我在郊外庄园的一家会所里”陈重顺着他的话说道。<br/>　　<br/>　　    “好，地址发给我，我过去找你”王惊蛰说完就给电话挂了，然后拉着菜刀文说道：“开车跟我走，人知道在哪了”<br/>　　<br/>　　    两人随即就从医院里出来往陈重那边赶了，王惊蛰的目的自然很明确，冤有头债有主，谁把丁武砍的生不如死了，他就找谁去。<br/>　　<br/>　　    这要是为啥王惊蛰会主动找上门去的原因，事不关己高高挂起，但关于到他了，王惊蛰也肯定不会含糊的。<br/>　　<br/>　　    会所里，陈重打完电话后，挺狐疑的挠了下鼻子：“怎么有点乱套了呢？”<br/>　　<br/>　　    午桥忽然站起来，跟陈重问道：“陈老板，刚才给你打电话的那个人叫惊蛰？他姓什么……”<br/>　　<br/>　　    “姓王”<br/>　　<br/>　　    午桥当即张着嘴愣神了，片刻后他连忙说道：“短发，一米七十多，人长得很普通不太愿意说话，二十几岁左右？”<br/>　　<br/>　　    “差不多就是这样吧，怎么你也认识嘛？”<br/>　　<br/>　　    午桥心里顿时“哎呀，我擦”了一句，自从上次鬼门开了之后传出王惊蛰被关在阴曹地府里的消息，这一下子让玄门和几个大派的人都被晃了一下，因为谁都想要他手中的连山和归藏，但人在阴间出不来了，这事就只能放下了。<br/>　　<br/>　　    可万万没意料到的是，王惊蛰居然从阴间出来了，然后还跟陈重认识，并且听那意思是马上还要过来一趟？<br/>　　<br/>　　    “真是柳暗花明又一村啊”午桥感慨了一句。<br/>　　<br/>　　    陈重见他没在问，就跟老马说道：“人先等一等再放，我看看惊蛰来是因为什么事”<br/>　　<br/>　　    半个多小时后，王惊蛰跟菜刀文赶到了会所外面，陈重独自一人出来的，王惊蛰从车里下来后，他就问道：“惊蛰，你打听这个人干嘛？”<br/>　　<br/>　　    “我要找吴洋……”<br/>　　<br/>　　    陈重顿时迷糊了，不解的问道：“怎么回事呢？”<br/>　　<br/>　　    “昨天晚上，丁武去跟齐泉抓人，被吴洋在身上戳了十几刀，现在人还在医院里躺着没醒过来呢，陈老板你说我找吴洋干什么？有怨报怨有仇报仇啊”<br/>　　<br/>　　    陈重顿时哦了一声，丁武昨晚受伤的事他到现在还不知情呢，他连忙问道：“人问题大不大？”<br/>　　<br/>　　    “挺过今晚，问题就不大了”<br/>　　<br/>　　    陈重叹了口气，为难的摇头说道：“惊蛰，人确实在这里，但是我现在有点骑虎难下了，这个跟吴洋在一起的人，不知道是什么来历，很多人来找他说情要我放人”<br/>　　<br/>　　    “你把人给放了？”王惊蛰阴着脸问道。<br/>　　<br/>　　    “还没有，但是……不得不放了”<br/>　　<br/>　　    王惊蛰鄙夷的说道：“陈老板，你就这点魄力啊？你一家三口都要被人给坑死了，就这么被人蹬鼻子上了脸，然后一点脾气都没有了？”<br/>　　<br/>　　    陈重眯了眯眼睛，很想反驳回去，但一想好像还真是这么回事，他都感觉自己挺窝火和憋屈的了，可事实呢，他还真不能有什么脾气了。<br/>　　<br/>　　    “我已经推了很多人，但最后是我三叔亲自打电话过来的，他一出面我就得打碎了牙齿往肚子里咽了，这个人我得放，但吴洋我还照样得找！”<br/>　　<br/>　　    “一点余地都没有？”<br/>　　<br/>　　    陈重叹了口气，说道：“确实没有了”<br/>　　<br/>　　    王惊蛰忽然意味深长的说道：“行啊，那你就给他放了吧……”
