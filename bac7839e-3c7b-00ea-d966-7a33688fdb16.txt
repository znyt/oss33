　　寒冰池数百米范围内，五百武者，仿佛置身于一片灵气海洋中，舒服，徜徉。
<br/>　　所有人都沉浸在这美妙感觉中，沉浸在快速提高修为的爽快上。
<br/>　　没有人注意到，一道身影，忽然站了起来，走到寒冰池的另一边。
<br/>　　人影正是萧逸。
<br/>　　聚灵阵，是一阵高深的禁制手段，起码需要破玄境武者才能施展出来。
<br/>　　它的存在，导致范围内的浓郁灵气分布均匀，众人并不需刻意去选位置，大多是随意找个空旷位置便开始修炼了。
<br/>　　于是，寒冰池的另一边，并没有其他武者。
<br/>　　萧逸盘膝坐下，看着自己周围空荡荡的，满意一笑，催动冰鸾剑武魂和控火兽武魂，再次开始修炼。
<br/>　　没有人注意到，他正以一种恐怖而骇人的速度快速吸收着周遭的灵气，甚至产生了数个灵气漩涡。
<br/>　　他的修为，也在飞速跃升。
<br/>　　半天后，第六条主脉，轻松凝结而成。
<br/>　　想起来，他突破先天五重，还是半个月前的事，如今以达先天六重。
<br/>　　两天后，第七条主脉，再次凝聚成功。
<br/>　　到了第五天，第八条主脉也被凝聚出来。
<br/>　　最后一天，第九条主脉，堪堪凝聚成功，修为达到先天九重。
<br/>　　仅仅七天，连跳四重，快得吓人。
<br/>　　而事实上，这七天时间，他修炼的同时，还在温养着血戮剑。
<br/>　　吸入体内的灵气，转化为真气后，一成用于凝结真脉；九成用于温养血戮剑。
<br/>　　若非要温养血戮剑，他早就突破到洞玄境了。
<br/>　　毕竟这里的灵气浓郁程度高得吓人，以他的吸收速度，一日可抵百日之功。
<br/>　　不过，不是他不想突破洞玄境，而是他的修为已经跳得够快了，再贸然去突破，乃修炼大忌。
<br/>　　洞玄境，可是武者修炼之路上的一道分水岭，大意不得，这将影响武者日后一生的走向。
<br/>　　萧逸打算，入了裂天剑派后，得到更系统、更完整的修炼知识，再慎重地去考虑突破洞玄境的事。
<br/>　　而且，萧逸并不觉得自己做错了选择。因为，经过这七天的努力，血戮剑，已经恢复到上品阶位了。
<br/>　　若不是借助这里的功效，想恢复到上品，也不知要何年何月。
<br/>　　上品灵器阿，先天九重武者持之，足以与洞玄境武者抗衡。
<br/>　　当初，对战钟无忧时，自己不过是使用下品灵器烈焰手套，已经如此之强。
<br/>　　如今有上品灵器，战力可想而知。
<br/>　　整个北山郡，怕也找不出几把上品灵器。
<br/>　　七天时间已过，易老回来，叫醒了还沉浸在修炼中的众人。
<br/>　　众人被叫醒，先是面露不满，而后脸色欣喜若狂。
<br/>　　“我突破到先天九重了。”秦飞扬来到萧逸身边，炫耀似地说着。
<br/>　　“切，得瑟。”林劲也走了过来，翻了个白眼，道，“我也突破了一重修为，达到先天八重了。”
<br/>　　铁牛和柳嫣然也走了过来，欣喜道，“我们也突破了一重，先天七重修为了。”
<br/>　　铁牛瓮声道，“才七天时间，我们就突破一重修为了，这寒冰池真是厉害，裂天剑派也不愧是北山郡第一门派，底蕴如此深厚。”
<br/>　　要知道，若是正常修炼，他们要突破一重修为，起码要数月时间。
<br/>　　“萧逸你呢。”柳嫣然眉毛弯弯，问道。
<br/>　　“我？”萧逸笑笑，道，“提升了一点儿吧。”
<br/>　　事实上，萧逸的吸收速度，比他们快了百倍都不止。
<br/>　　和他相比，其他人的修炼速度，简直就是蜗牛般慢。
<br/>　　只不过，众人都沉寂在突破修为的喜悦中，也没有太去纠结萧逸的话。
<br/>　　这时，就体现出武魂的重要性了。
<br/>　　这也是为什么这个世界以武魂的等阶来衡量武者的天赋。
<br/>　　武魂等阶越高，吸收灵气速度便越恐怖，修炼起来也越快。
<br/>　　“好了，随我离开吧。”这时，易老沉声说道，带着一众武者离开。
<br/>　　当五百人离开后，不多时，一个年轻人慢步来到寒冰池旁，紧紧地皱起了眉头。
<br/>　　如果萧逸在此的话，一定会认出，年轻人竟是药堂首席弟子，叶铭。
<br/>　　叶铭看了几眼，皱眉道，“怎么回事，今年的考核武者这么厉害吗？才七天时间，这三个月分量的天材地宝药力就被耗得七七八八了。”
<br/>　　裂天剑派，每三月都要耗费大量天材地宝，以维持寒冰池放出庞大的灵气。
<br/>　　灵气被吸收得越多，里面的天材地宝药力就会消耗得越快。
<br/>　　半晌，叶铭摇了摇头，自语道，“算了，还是找师傅来看看吧。这么大分量的天材地宝，已经超出我能处理的范畴了。”
<br/>　　寒冰池，归药堂管理。
<br/>　　每三月投放一次的天材地宝，数量都庞大得吓人。
<br/>　　这么奢侈的消耗，连一般的大势力都承担不起，也就裂天剑派有这个底蕴。
<br/>　　然而，谁都想不到，这次参加考核的武者中，竟有一人拥有传说中的紫色武魂。
<br/>　　放萧逸来这种‘宝地’，简直就是‘引狼入室’的行为。
<br/>　　当然，紫色武魂这种属于传说的存在，没人见过，也没人知道它恐怖到什么程度。
<br/>　　同时，那被消耗得七七八八的天材地宝，并非是因为萧逸连跳四重的修为，主要还是血戮剑。
<br/>　　灵器内，本身就蕴含极庞大的力量，故跌落品阶后，恢复品阶也需要极庞大的力量去温养。
<br/>　　所需力量之多，一下子就几乎耗光了寒冰池内足以维持三个月的大量天材地宝。
<br/>　　......
<br/>　　在易老的带领下，一行五百人，来到了最开始的巨大广场上。
<br/>　　广场正中央，有一巨大比武台。
<br/>　　第二轮考核，便是个人排名战，真正靠自己的实力。
<br/>　　虽说，第一轮考核已经大致能看出武者们的实力水准。
<br/>　　但，第一轮考核有‘联手’的事情发生。
<br/>　　而且，大部分的战斗，都是混战。一个月时间，考验的是武者的生存能力和应变能力。
<br/>　　裂天剑派的考核，每一轮，都带有目的性。
<br/>　　第二轮，只有单打独斗，才是真正考验武者修为实力的时候。
<br/>　　寒冰池的开放，也是为了让武者们恢复伤势，以最好的状态应对战斗。
<br/>　　这一轮，将采取站桩的方式，一人上场，不断接受挑战，直至战败。
<br/>　　可以说，这将是个人实力发挥到淋漓尽致的一场考核。
<br/>　　.....
<br/>　　巨大的比武场上，被分成十个区域。
<br/>　　五百人，都在等候着。数个执事，正以抽签的方式，从五百个令牌中，抽出十个，作为最先上场站桩的十人。
<br/>　　这十人，将在各自的区域内，不停接受挑战。
<br/>　　为了公平，抽签的步骤很是繁琐，也很耗时间。
<br/>　　这时，林劲、铁牛、柳嫣然三人，忽然脸色古怪地走到萧逸身边，支支吾吾。
<br/>　　“你们做什么？”萧逸面露疑惑之色。
<br/>　　“那个，我有件事想和你们说。”柳嫣然面带歉意地看了眼萧逸、林劲和铁牛。
<br/>　　“我也有事要和你们说。”林劲同样面带歉意。
<br/>　　铁牛没有说话，但脸色一样。
<br/>　　“你们三个，怎么回事？”萧逸皱起了眉头。
<br/>　　“嫣然你先说好了。”林劲道。
<br/>　　铁牛也看向柳嫣然。
<br/>　　“好。”柳嫣然迟疑地点了点头，“其实...其实，这些天，我骗了大家，其实，我没有那么弱的。”
<br/>　　说着，柳嫣然看向萧逸，道，“那天，我不要你的灵器，是因为，我自己就有。”
<br/>　　说着，柳嫣然手上光芒一动，一条红色长鞭凭空而现。
<br/>　　长鞭上灵气涌动，显然是一把灵器。
<br/>　　“哦，原来说的是这事。”萧逸恍然，同时，他发现铁牛和林劲的脸色变得古怪起来。
<br/>　　“原来嫣然你也隐藏了实力吗？”林劲瞪大了眼睛，说道。
<br/>　　“也？”柳嫣然微微一惊。
<br/>　　“额，那个...其实....”林劲一直是个性格坚毅，豪迈爽快的人，如今竟有些尴尬地支支吾吾。
<br/>　　“其实我也有灵器的，实力很强喔。”林劲说着，手中光芒一动，那是一把重剑，也是灵器。
<br/>　　铁牛忽然瓮声笑了起来，“原来骗人的不止我一个，那么俺的心就安了。”
<br/>　　说着，铁牛手上忽然出现了一个手环，显然也是灵器。
<br/>　　“你们....”三人互相对视起来，满脸惊讶。
<br/>　　“呵呵。”萧逸淡淡一笑，他早就知道这三人都隐藏了实力，也大致猜出了他们的底牌是灵器。
<br/>　　只不过，他眼力过人；但林劲三人，却彼此没有发现。
<br/>　　“看来，我们这里最老实，没有骗人的，就是萧逸了。”柳嫣然赞扬道。
<br/>　　“另外两个家伙也骗人，我就不说了。”柳嫣然愧疚道，“至于萧逸你，抱歉了，我并非刻意想瞒你。”
<br/>　　“只是，我来裂天剑派前，家中长辈教导过，灵器不可轻易拿出来，毕竟，人心叵测，你也知道灵器对武者的诱惑力有多大...”
<br/>　　柳嫣然说着，又连忙道，“当然，我不是说你会觊觎我的灵器...”
<br/>　　“也不是说你们。”柳嫣然看了眼林劲和铁牛。
<br/>　　“我只是不想....”
<br/>　　“我明白你的意思。”萧逸笑笑，打断道。
<br/>　　“明白就好。”柳嫣然松了口气，“我很珍惜我们这段友谊，我很怕看到因为灵器而导致...”
<br/>　　“我真的明白，别担心。”萧逸说着，摸了摸她的脑袋，表示无所谓。
<br/>　　他当然明白，柳嫣然隐瞒灵器的原因，就跟自己当初误会三人要抢夺自己的‘地阶武技’一样。
<br/>　　这段难得而真挚的友谊，互相都格外珍惜着，不希望因为外物而导致失去。
<br/>　　害怕品尝对这段情感的失望。
<br/>　　林劲和铁牛也是这个原因。
<br/>　　当初在第一轮考核，他们想着，能通过考核就算了，所以不需出全力。
<br/>　　可现在，个人排名战，必须实力全开。
<br/>　　他们也即将无法隐藏自己的所有实力，便在此刻一一叙说出来。
<br/>　　但，他们又怕因为自己的隐瞒，而导致彼此误会，故支支吾吾的，满脸愧意。
<br/>　　说起来，四人都是16、7、8岁，处于人生中最精彩的年龄，自是对这些情感格外珍视。
<br/>　　林劲和铁牛也叙说了一番，理由都一样。
<br/>　　同时，都愧疚地看着萧逸，说了句，“萧逸兄弟，你是老实人，抱歉了。”
<br/>　　萧逸笑笑，表示无所谓，毕竟，其实最不老实的就是他。
<br/>　　这时，秦飞扬忽然走了过来，惊讶道，“哟，你们几个家伙，怎么这副脸色，做了什么亏心事吗？”
<br/>　　众人白了他一眼。
<br/>　　这家伙，从一开始，就在另一边和几个女子搭讪，相谈甚欢，现在才回来。
<br/>(本章完)
