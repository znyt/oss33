　　小兰似乎也意识到，自己留下来，只会连累叶尘，最终含着泪离开了。
<br/>　　“叶枫，你现在可以把你修炼的功法，交出来了吧？”
<br/>　　叶家家主满脸激动之色，迫不及待的开口道。
<br/>　　叶尘嘴角微微一扬，指了指自己的脑袋，道：
<br/>　　“功法就在我的脑袋里面，就看叶家主有没有这个本事拿到了！”
<br/>　　“小畜生！你敢耍我？”
<br/>　　叶家家主听到叶尘这话，顿时两眼一瞪，怒气冲天。
<br/>　　眼看叶家家主就要出手，一旁一名老者，忽然闪身而出，
<br/>　　“对付一个毛头小子，何须家主亲自出手？我这就把他擒住，交给家主处置！”
<br/>　　那老者声音落下的同时，整个人犹如一只看到猎物的老鹰一般，自天空之上，朝下方的叶尘俯冲了过来！
<br/>　　“青藤索！”
<br/>　　老者冲到叶尘跟前数十米，忽然长袖一挥，一道散发着璀璨光芒的绿色绳索，自他的长袖之中飞出，携带一股惊人的造化之力，瞬间便爆射到叶尘眼前，要将他缠住。
<br/>　　造化之力虽然在大多数情况下，是用来疗伤的一种力量，但同样也可以用来攻击，只不过杀伤力相对较弱而已。
<br/>　　但是在造化世界之中，只有造化规则，所以这个世界的修炼者，只能修炼出造化之力，叶尘也不例外。
<br/>　　嗖嗖！
<br/>　　叶尘虽然早就有所防备，在第一时间就利用精妙的步法，躲开了对方这一击，但是那根粗壮的青藤，在空中打了一个转，便又继续找他追赶了过来。
<br/>　　这老者的修为，虽然不如叶家家主，却也是道基境的高手，而叶尘却才刚刚突破到丹成境而已，两人之间的修为着实不小。
<br/>　　叶尘凭借着丰富的战斗经验，与老者纠缠了一阵之后，顿时便开始落入下风，被那根青藤逼的节节败退。
<br/>　　突然！
<br/>　　嗖！
<br/>　　叶尘一个躲闪不及，脚踝被那青藤缠住，继而迅速沿着他的腿，缠到了他的身上，很快便犹如裹粽子一般，将叶尘全身都死死缠住。
<br/>　　“哈哈哈！”
<br/>　　那老者见擒住叶尘，顿时高兴的哈哈大笑起来，
<br/>　　“小畜生！你也不过如此嘛！真是不堪一击！”
<br/>　　老者得意的嘲笑了一句之后，然后将手中的青藤索猛然一拉，
<br/>　　“过来！！”
<br/>　　呼！
<br/>　　叶尘被青藤索包裹着，整个人似乎毫无任何反抗之力，顿时便被老者拉了过去。
<br/>　　然而，老者根本没有注意到，就在叶尘看似已经青藤索被擒，似乎败局已定，嘴角却微微扬起，露出一抹冷笑。
<br/>　　“青帝长生体！吞噬！”
<br/>　　眼看叶尘已经老者拉至跟前，叶尘忽然一声低喝，
<br/>　　轰！
<br/>　　下一刻，叶尘身上，陡然发出一道璀璨的青色光芒，
<br/>　　滋滋滋~~
<br/>　　缠绕在叶尘身上的那根青藤索，仿佛一下子枯萎了一般，竟然瞬间变成了干黄色。
<br/>　　“什么！！”
<br/>　　那老者明显一愣，显然没有反应过来这是怎么回事。
<br/>　　“彭！”
<br/>　　迅速枯萎之后的青藤索，猛然破碎开来！
<br/>　　轰！
<br/>　　叶尘从青藤索中挣脱出来，此时又刚好被老者拉到了跟前，于是直接一拳，狠狠砸向老者的丹田位置，
<br/>　　“噗！”
<br/>　　那老者根本没有反应过来，而且本以为自己胜券在握，所以也没有丝毫的防备，丹田瞬间便被叶尘生生震碎，狂喷了一大口鲜血之后，一脸不甘的倒飞了出去，重重栽落在数十米外的地上！
<br/>　　“什么！”
<br/>　　不光那老者本人，一时间没有反应过来怎么回事，就连周围的其他叶家高手们，也全都被眼前的一幕惊呆了。
<br/>　　要知道，那老者可是叶家的三大长老之一，一身修为早已经达到了道基后期，对付一个只有区区丹成境的小子，本以为是手到擒来的事情，怎么突然发生了如此变故？不但败的稀里糊涂，而且败的如此凄惨！
<br/>　　身受重伤！丹田被废！
<br/>　　这位叶家长老即便捡回一条命，恐怕也将彻底沦为一个废人了！
<br/>　　“好你个小畜生！！”
<br/>　　叶家家主反应过来之后，先是大怒，继而眼眸之中，透着一抹浓浓的兴奋之色，心中不由得暗暗寻思起来，
<br/>　　‘这小畜生不过才区区丹成境的修为，竟然可以击败一名道基境后期的高手，肯定是施展了某种逆天秘术，倘若我能得到这门秘术，实力必然可以大大提升……’
<br/>　　就在叶家家主暗暗寻思之时，其余那些叶家的高手们，已经纷纷朝叶尘出手了。
<br/>　　“不可伤他性命！本家主要活的！”
<br/>　　叶家家主当即出口提醒了一句。
<br/>　　他这话一出，原本暴怒的叶家众人，顿时全都为之一滞。
<br/>　　而叶尘则抓住这个机会，身体一晃，直接杀向那几个修为较弱的叶家高手。
<br/>　　轰轰轰！
<br/>　　施展出青帝长生体之后的叶尘，不但可以迅速的吸收造化之力，来修复身上的伤势，同时防御也大大提升，再加上他丰富的战斗经验，简直犹如虎入狼群般，转移之间便斩杀了几名丹成境的叶家高手，一下子冲出了叶家众人的包围圈，然后催动精妙身法，直奔山谷外飞驰而去。
<br/>　　“可恶的小子！”
<br/>　　等到叶家家主反应过来，叶尘已经逃出山谷，并且奔出了十多里。
<br/>　　叶家人多势众，又有一名破神强者坐镇，以叶尘如今的修为，根本不可能与之正面抗衡。
<br/>　　不过，叶尘才刚刚奔出数十里，顿时便感应到一股磅礴的气息，从后方朝他袭来。
<br/>　　虽然他的身法精妙无比，可是修为毕竟太弱，与叶家家主相比，差的不是一点两点，很快便被此人追了上来。
<br/>　　“小畜生！乖乖束手就擒！”
<br/>　　叶家家主追至跟前，直接大喝一声，手掌猛然拍落下来，
<br/>　　“万木囚牢！”
<br/>　　轰隆！
<br/>　　随着叶家家主这一掌拍落，九天之上顿时陡然出现了数百道青藤，形成了一个巨大的牢笼，将下方的叶尘一举笼罩其中，让其无处可逃。
<br/>　　紧接着，叶家家主身体一晃，便穿过那些青藤，也进入到了牢笼之中，俯视着下方的叶尘，犹如俯视自己的猎物一般，缓缓道：
<br/>　　“小子！本家主再给你最后一次机会，乖乖将你修炼的功法交出来，本家主可以饶你不死！”
