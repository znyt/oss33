　　丧礼结束，一行人回到山上，曲波召集大伙，纷纷大殿前的广场上，由几位长老训话。这涉及人家门派内部的事，叶少阳本想回避，却被曲波硬拉着留下。
<br/>　　“诸位师兄弟和晚辈，就在我们现在站立的地方，昨晚有过一场激烈的战斗，在这里，我们有好几个兄弟牺牲了，步了道渊祖师和无生掌教的后尘……他们是龙虎山的英雄，我们都应该感到骄傲，龙虎山浩气长存、英勇不屈，对得起道门宗主的地位！”
<br/>　　曲波的发言，激起了诸位弟子的情绪，作为龙虎山的弟子，他们很骄傲，纵然如今山门凋敝，他们还是很骄傲，因为他们是龙虎山弟子，他们都不怕牺牲，不管对手再强，也不可能让他们屈服。
<br/>　　这是龙虎山千年传承下来的精神，永远不可能改变。
<br/>　　曲波接着又说了一些鼓励的话，然后吩咐了一些具体事情，然后……他要求大家肃静，听他说今天最重要的事。
<br/>　　“昨天，我还有几位长老一夜没睡，我们讨论了这次灾难的始末，外因是轩辕山突然袭击，事前没做好准备，但内因也是前任掌教慕寒刚愎自用，不信任法术界同仁……事情经过大家都知道，多余我就不说了，也得亏叶掌教不计前嫌，前来搭救，才免于龙虎山千年基业毁于一旦……
<br/>　　山门不可一日无主，昨晚我与诸位长老商量，大家一致认为，应当请叶天师担任我龙虎山掌教，不知诸位意见如何，今日谁都可以发言。”
<br/>　　叶少阳听到这里，吓的从椅子上跳起来，忙摆手说道：“这怎么行！使不得使不得！”
<br/>　　曲波上前，语气诚恳地说道：“叶天师，之前，我们龙虎山对你是有误解，也一直想跟茅山较劲，但这么做的结果是什么？慕寒走了极端，造成法术界分化，差一点就被轩辕山灭门……你能不计前嫌前来帮忙，这份恩情，如同再造……
<br/>　　这是其一，其二，你杀了星月奴和影魅，为道渊祖师和无生掌教报仇，凭这份功德，坐我龙虎山掌教更是没得说。其三，慕寒走了，龙阳师兄也殁了，如今龙虎山再没有合适的人担当掌教之位……就请叶天师不要再推辞了。”
<br/>　　叶少阳涨红着脸，解释道：“这怎么可以，从古到今，都没有这样的事……”
<br/>　　“那都是门户之见！”一个长老尖声说道，声音带着强烈的愤怒，“就是这种门户之见，害惨了我们龙虎山，慕寒倒是张家嫡传，但结果呢？瞧瞧他都干了什么，什么天选之子，龙虎山再也不信这个了，叶天师你是如今人间法术界中流砥柱，你来掌管龙虎山，我们心服口服。”
<br/>　　几个长老跟着一起点头。
<br/>　　叶少阳道：“摈弃门户之见，这个我支持，法术界就是不够团结，才让轩辕山钻了空子，我们要是铁板一块，真的谁来都不怕！不过……几位长老在上，你们是龙虎山的老人了，威望也够，你们比我合适的多啊。”
<br/>　　那长老笑道：“叶天师，我就知道你要用这个理由推脱，实话跟你说，要在慕寒即位之前，还真轮不到你，我们几个老家伙也不是没这个心思，但龙虎山向来有个规矩，掌门之位，只能在同辈和晚辈之中流转，绝不可逆传长辈，慕寒辈分在我们之下，他当了掌教，那不管怎样，我们几个长辈都是没机会的了……”
<br/>　　叶少阳张口结舌，一把又拉过曲波，道：“那就曲波啊，掌门大印在他手上，慕寒亲口说过传位给他的不是吗？”
<br/>　　又一个长老笑道：“慕寒临阵逃脱，已被长老会开除名籍，连龙虎山弟子都不是了，他说的话，自然不管用，曲波是个好苗子，但担当掌门大任，实在有点力不从心……曲波我这么说你别生气啊。”
<br/>　　曲波拱手道：“今天的事就是我发起的，我哪敢生气，叶天师，我自己水平我是清楚的，我实在不是当掌门的料，叶天师你就别推脱了，非你莫属。”
<br/>　　叶少阳还在推辞。
<br/>　　“我看不拿出杀手锏你是不行了！”曲波笑着，突然拿出一张黄纸，展开成A4纸那么大，交给一个弟子，让他给所有弟子传看，
<br/>　　“你们可看好了！这是无生掌教的亲笔信，是他舍身之前亲笔所写，他当时就预料到慕寒可能当不起大任，是因此给我留下了这封遗书，上面有掌门之印，做不了假！”
<br/>　　纸张最后传回到曲波手中，他送给叶少阳看。
<br/>　　上面只有一句话：若慕寒不当用，则请叶少阳摄位。
<br/>　　下面有他的签名和印章。是用毛笔写的。
<br/>　　叶少阳瞠目结舌，仔细一想，便理解了张无生的苦心。之前他内心也吐槽过张无生怎么选了慕寒这么个人来当接班人。
<br/>　　这其中，怕也是无可奈何。
<br/>　　慕寒是天选之子，生来天赋就极高，这一点，张无生当然知道，因此从小就栽培他。等他性格成形，张无生怕是也发现了他性格上的缺陷，但多年心血，也没有办法，而且那又是自己亲侄子，出生就被立为接班人的。
<br/>　　因此在他决定舍身的时候，将掌门之位传给了他。
<br/>　　那时候的张无生，只怕内心是纠结矛盾的，一方面担心慕寒会搞出大事，一方面又不想浪费自己多年来培养的心血……况且除了慕寒，实在再没有任何合适的人选了。
<br/>　　他想到了自己，想到打破门户之见，立自己为掌门，但就算他是掌门，这命令只怕也过不了长老会那一关，他又即将赴死，没法去亲口跟他们解释，干脆就传位给慕寒，同时写下这封密信交给曲波——假如慕寒胡搞，引起长老会对他的彻底失望，到时候再公布密信，长老会也是无话可说。
<br/>　　真是用心良苦啊。
<br/>　　叶少阳设身处地地想到这里，心中感慨万千，默默对张无生说道：张师叔，你可真是看得起我。那我便努力不让你失望吧。
