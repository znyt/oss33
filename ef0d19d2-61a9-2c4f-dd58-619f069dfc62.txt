<br/>　　寻常的生死境武者，在他们的手中都无法接下几招，就要败北。

<br/>　　荒铁血看着楚风眠，冷声说道，言语中的话音威胁之意不言而喻。
<br/>　　“我们身为监察使，有资格盘问你的来历，你要是再不回答，我等就要呼唤宗门强者了，倒时候你可就是我们七大宗门的通缉犯，知道吗？”

<br/>　　楚风眠眼神中露出几分不屑，大笑着开口道。
<br/>　　“告诉你们也无妨，我叫楚风眠，就是这个名字，不知道你们有没有听说过了。”
<br/>　　“楚风眠？”
<br/>　　听到这个名字，三名监察使的都是目光闪烁，似乎是在思考这些什么。
<br/>　　突然之间这三人的脸色，突然大变，眼神中尽是杀意。
<br/>　　“楚风眠？居然是你！”
<br/>　　这个名字，在几日前，突然在七大宗门之中，都是引起了轩然大波，据说整个归墟天之中的七大宗门弟子，全部都是被一个叫楚风眠的武者，全部斩杀了。
<br/>　　他们虽然在外，却也听说过这件事，没想到他们居然会在这里，碰到了楚风眠。

<br/>　　荒铁血看着楚风眠，突然大笑起来道。
<br/>　　“本来以为你躲在北域之中，当个缩头乌龟苟且偷生也就罢了，没想到你居然还敢踏足中域，胆子够大的啊。”
<br/>　　“楚风眠，你本来就是我们七大宗门的通缉犯！我们七大宗门下了命令，只要是谁碰到你，都可以杀了你，拿你的人头去索要好处，没想到啊，居然是让我们碰到了。”
<br/>　　天剑宗的剑越，也是突然之间似乎是脸色大喜，大笑着开口道。
<br/>　　“好！好！来到这鬼地方，本来就没什么油水，现在居然冒出来个楚风眠！很好，现在你可是无数老祖的眼中钉，肉中刺，杀了你，可以换取无数奖赏！”

<br/>　　归墟天这种满地是宝的地方，他们三人也曾经前去过，自然清楚。
<br/>　　杀了楚风眠，不亚于得到一座巨大无比的宝藏。
<br/>　　“杀！”
<br/>　　三名监察使，几乎是毫不犹豫，不由分说全部出手。
<br/>　　一股股灵力全部爆发出来，引得周围的无数武者，都连连后退，漫天血光，无数杀招，轰然之间全部向着楚风眠轰击过来。
<br/>　　“楚前辈小心！”
<br/>　　古锦一时间还没有想到楚风眠的来历，但是看到这一幕，她还是急忙提醒道。
<br/>　　楚风眠毕竟是救了她一命，古锦也不愿意看到，楚风眠死在这三位监察使的手中。

<br/>　　听到这个称呼，在场无数武者的目光都是各异。

<br/>　　这得是多么狂妄的人，才能说出这句话，一些超越生死的老祖，都不敢说出这句话。
