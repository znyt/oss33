　　叶少阳一直在琢磨的是，崔府君在听了萧逸云的话之后，究竟是信还是不信。从他用这种方式赶走自己、并且说出那么一段话来看，应该是信的，至少也是怀疑。不过，最让叶少阳想不通的是，崔府君为什么不跟自己见面求证，而是自己赶人。
<br/>　　他为什么要这么做？
<br/>　　叶少阳想了半天也想不到，不过这些其实都不重要了，反正结局都是一样：人家不愿意见自己。从他最后跟自己说的那番话里，叶少阳感觉到，自己可能连再去阴司的机会都没有了。
<br/>　　本来他还想再想什么法子去一次阴司，找徐文长或者黑白无常等人帮忙，现在看来，根本没有必要了，崔府君的态度，几乎也等于了别人的态度，再说自己现在连阴司都去不了，想这些根本就是多余了。
<br/>　　计划失败了……
<br/>　　叶少阳仰望天空，长叹了一口气。对这个结果，他内心十分失望，但是并没有绝望。既然阴司这条路走不通，自己还是继续进行计划，去找道渊真人！虽然他能感到，这么做的希望也不大，且不说道渊真人也极有可能不相信自己的身份，退一步说，就算相信，他能不能帮到自己还是一回事。
<br/>　　不过，自己真的没有其余的办法可想了，总不能什么都不做，就这么干等着徐福来找自己。
<br/>　　万一他十年八年不来，自己难道还得在这个世界住个十年八年不成？
<br/>　　主意打听，信心也加强了几分，叶少阳起身朝小陈家里走去，去找翠云。
<br/>　　接到翠云之后，叶少阳谢过小陈，跟着翠云一起回家去。
<br/>　　路上翠云问他干啥去了，叶少阳哪里有心情说，关键他一个普通人，就算说了她也不懂，叶少阳也就随便找老了个理由打发过去。
<br/>　　“翠云姐，我还是要去江西，找我那个亲戚。我想今天就走。”叶少阳成功把话题引到这上面来。
<br/>　　翠云正在赶路，听见他这番话，一下就站住了，有些失神地说道：“今天就走？”
<br/>　　叶少阳点点头，“反正都是要走的，我在这也没什么必要住下去了。”
<br/>　　翠云咬了咬嘴唇，说道：“你怎么走？”
<br/>　　“我……我不知道啊。”叶少阳这才发现，自己一直嚷嚷着走，其实并没有真正考虑过这个问题：这个时代火车飞机是不用说了，这种小地方，估计长途汽车都没有，自己总不能靠着双腿走到龙虎山去吧？
<br/>　　翠云道：“我还是给你雇一辆马车吧，马车挺快的，一般人都做不起的，好在我们现在有钱了。不过……你说的江西在哪啊？”
<br/>　　“就在安徽南边一点吧，相邻的省份，你不知道？”叶少阳有点吃惊，自己已经是个地理够渣的了，也知道江西大致的位置。
<br/>　　翠云摇摇头，“我从来都没出过安徽，最远就是到下蔡县，还是那年送我家公婆去找小叔子，后来他们都死在那儿了……”
<br/>　　翠云絮絮叨叨地说起往事，突然想到什么，皱眉对叶少阳道：“我听说现在一个省都是一个督军管的，出省的话有些麻烦的，而且听说南方时局不好，老是打仗，还闹土匪什么的。你……非去不可啊。”
<br/>　　叶少阳点点头，“非去不可。”
<br/>　　翠云也没再说什么，让叶少阳在家里呆着，自己去上街买菜。
<br/>　　叶少阳把自己东西收拾了一下，然后开始吐纳，继续修复丹田的强度，三个周天之后，叶少阳睁开眼睛，发现天已经黑了，厨房传来煮饭和拉风箱的声音，知道是翠云在做饭，于是过去看了一眼。
<br/>　　翠云正在拉风箱，抬头看到他，笑了笑说道：“你去外面等着吧，我多做两个菜，一会就可以吃了。”
<br/>　　叶少阳上去从她手中接过风箱的把手，说道：“我给你打下手，你安心做菜就行。”
<br/>　　翠云立刻有点不好意思。“哎呀，哪有让男人下厨的道理啊，你快起来吧。”
<br/>　　叶少阳道：“帮个忙不是很正常吗，大惊小怪什么。”
<br/>　　翠云道：“厨房是女人才进的啊，你又不是厨子，不可以干这种活的。”
<br/>　　“分这么清楚干啥。”
<br/>　　“不是啊，我们这都是这个规矩啊，男人是绝不进厨房的，你们家那边不是吗？”
<br/>　　叶少阳笑道：“没关系，我又不是你男人。”
<br/>　　说完这句话，叶少阳意识到有点那啥，抬头看了一眼，翠云的脸色被炉火映得通红。
<br/>　　翠云跟他客气了一番，见他实在要帮忙，也只好由他，自己去一旁洗菜切菜。
<br/>　　叶少阳拉着风箱，忍不住说道：“几十年后，会有一种叫鼓风机的东西，会取代风箱，比风箱好用的多，关键是不用动手。”
<br/>　　翠云扑哧一笑，道：“几十年后的事，你怎么会知道。”
<br/>　　叶少阳不由苦笑，“是啊，几十年之后的事，我怎么会知道。”
<br/>　　“好傻啊你。”翠云看着他这个样子，忍不住笑起来，说道：“这是你自己相像的吧，不用人动手，怎么会出风呢。”
<br/>　　叶少阳道：“用电啊。”
<br/>　　翠云听了之后，更加不以为然，说道：“电啊，电不是用来点灯的吗，我在县城见到过点灯，那东西可亮了，比煤油灯亮得多。”
<br/>　　叶少阳道：“电不光能照明，还能干很多事。”望着炉子里烧的噼里啪啦的柴火，说道：“不过再过几十年，鼓风机也用的少了，那时候都是电磁炉，煤气灶，根本就不需要烧柴了。”
<br/>　　“越说越离谱了，没柴火用什么煮饭啊。”
<br/>　　叶少阳笑笑，这次没有再解释，他知道就算自己解释得再详细，翠云也不可能接受几十年、一百年之后的科技。
<br/>　　叶少阳内心深处再度涌现出一种孤独感，这种孤独感不是针对翠云，而是眼前这个世界。对这个世界来说，自己是一个彻头彻尾的陌生人，也许还是唯一的陌生人。
<br/>　　翠云就是个农家小媳妇，不会做什么大菜，菜肴看上去也没什么花样和色泽，都是家常小菜，叶少阳尝了一下，口味真心不错。
<br/>(本章完)
