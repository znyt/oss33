　　翌日，李承乾照旧去给老货们授课，内容依旧是九九乘法口诀，老货们依旧没有背完，总之一切都和几天前一模一样，没有任何变化。
<br/>　　只不过在下了课之后，李承乾被程老货拉到了一个没人的角落。
<br/>　　老家伙四下看了半天，确定没人注意之后才神秘的说道：“小子，你的那个手炮还有没有多余的？”
<br/>　　“手炮？”李承乾狐疑的看着老程，脑袋摇的风车一样：“没有。”
<br/>　　“没有？那囡囡手里那两只是从哪来的？你可别说你不知道，那东西除了你小子根本不会有人拿得出来。”程咬金撇撇嘴，一副老子什么都知道的表情。
<br/>　　“我有什么好处？”听到是程小四露了底，李承乾也不再辩解，直瞪着程老货反问。这老货就是个奇葩，****都要咬一口的主儿，只要被他看到的东西，就没有他不想要的。
<br/>　　“你还想要什么好处，老夫连闺女都给你了，你竟然还跟老夫要好处。”程咬金习惯性的牛眼一瞪，虽然没什么作用，但是多少也能壮一壮声势。
<br/>　　“可是我父皇下了彩礼的。”李承乾不甘示弱的瞪回去。
<br/>　　“不够。”
<br/>　　“不够你找我父皇去啊，找我干什么，诏书又不是我下的。”面对无耻的老程，李承乾开始撒泼。
<br/>　　而且他说的也是事实，当时下诏书的时候他并没有在长安，如果说和他有关系的确不假，但关系还真就不是很大。
<br/>　　老程一时间也被他呛的不知说什么好，瞪着眼睛犹豫了半天，最后说道：“你想要什么好处？”
<br/>　　手铳的诱惑实在是太大了，尤其是在程小四有意炫耀之下，更是让他心里痒痒的。
<br/>　　扣动扳机之后那巨大的声音，以及大量铁砂被打出去之后靠成的破坏，让老程觉得自己这样的大将军如果不配上两支手铳，那就是大大的丢人。
<br/>　　可是程小四太抠门，竟然连摸都不让老程摸一下，两只手铳宝贝的跟什么似的，‘哐哐’放了两枪之后就收起来了，弄的老程不上不上特不是滋味，于是自然而然的就想到了李承乾。
<br/>　　“乘法口诀，三天背下来。”李承乾梗着脖子说道。
<br/>　　面对老程的要求，李承乾知道自己拒绝不了，而且手铳这东西，随着将作监那边越来越熟练，制作起来已经不像刚刚开始的时候那么费力，给统军将领配备手铳已经慢慢提上了日程。
<br/>　　所以如果给这些老家伙们无条件配发，还不如换点好处回来。
<br/>　　“你这是趁火打劫知道么，小子。”听到背乘法口诀，老程的脸顿时垮了下来。
<br/>　　这东西不是他背不下来，实在是因为脸面的问题不能背下来，他们十几个老货可是提前都约好的，半年之内绝不背完，急死这个敢让他们重回学堂的小混蛋。
<br/>　　“随你怎么说，反正想要手铳，就满足我的要求，否则免谈。”李承乾嘿嘿一笑，感觉似乎抓住了老程的命门，琢磨着是不是回头再多给程小四一些火药和铁砂，让她没事儿在老程前面显摆显摆呢。
<br/>　　“行，你小子最好说话算话。”程咬金也看出李承乾不见兔子不撒鹰的手段，索性不再追问，硬着头皮答应了下来。
<br/>　　至于说和其他老货之间的约定？嘿嘿，什么约定能有腰上挂着两只手铳来的重要。
<br/>　　“小侄说话自然算话，如果到时候没有兑现，程伯伯尽可以去父皇那里参我一本。”李承乾邪邪的笑着，他只是答应老程三天背下乘法口诀就给他火铳，可没有说给几支。
<br/>　　“嘿，量你小子也不敢不守信用。”老程嘿然出声，大咧咧的说着，拂袖而去。
<br/>　　李二这一辈子可是把承诺看的比什么都重要，如果李承乾敢不兑现承诺，到时候那还真就是一参一个准儿。
<br/>　　当天晚上，老程夫妇休息的房间。
<br/>　　“三六一十八，三七二十一，三八二十四……”
<br/>　　“老爷，您这嘀嘀咕咕干什么呢这是？”程夫人看着老程背着手在屋子里一个劲的嘀咕，不由问道。
<br/>　　“诶呀，休要呱噪，老夫正在背口诀呢。”老程被老婆打断思路，有些不耐。
<br/>　　“什么口诀这么麻烦，您这可都弄了一个下午了，怎么还是没进展呢？”程夫人继续问道。
<br/>　　“你个妇人懂得什么，这叫乘法口诀，难着呢。”被老婆问的急了，老程牛眼一瞪牢骚道。
<br/>　　“很难么？囡囡四天前用了半个时辰就能背到九九八十一了，偏偏老爷您背了一个下午还没有背完五九。不是妾身说您，背书这东西……。”程夫人到底是大家族出身，说起背书那绝对不是老程能比的，完全有理由来鄙视老程。
<br/>　　“好了好了，老夫知道你能，你聪明行了吧。”被老婆念叨的有些烦躁，老程干脆一甩袖子离开了房间，打算找个清静点的地方。
<br/>　　结果出了房间没多久，老程就听到‘嘭’的一声。
<br/>　　手铳！又是手铳！顺着声音，老程晃悠着走了过去，时间不大便来到后宅的一个演武场。
<br/>　　“花娘，再拿两个火药包来，顺便把铁砂也拿来一些……”演武场的一角，四个身穿四二式军服的女兵被程小四指使的团团转。
<br/>　　“嘭嘭”又是两声呛响，刺鼻的火药味再次弥漫，离着程小四所立位置二十余步之外的草人身上多了数个小小的黑洞。
<br/>　　“囡囡，又练习手炮呢？”程老货涎着老脸，凑到闺女身边，看着程小四手中的手铳直流口水。
<br/>　　“阿爹，这个叫手铳，不叫手炮。”程小四纠正了一下程咬金的称呼问题，然后将手铳收进了腰间的枪套，防老程如同防贼偷。
<br/>　　“不管这东西叫啥，囡囡，你可一定要注意安全，千万别伤了自己。”对于闺女的不信任，老程的语气中多了一丝的幽怨。
<br/>　　儿大不由爷啊，也不知道李承乾那个混小子给自己闺女施了什么法术，竟然让闺女如此死心塌地的跟着他，说什么都听，连自己这个老子的面子都不给，这女闺……白特么养了。
