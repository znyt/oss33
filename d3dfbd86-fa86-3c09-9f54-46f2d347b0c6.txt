　　不过山河真人把自己的修所建在这里可不是为了看风景，而是没到极端天气，风霜雪雨的时候，自己就在这里静静打坐吐纳，锻炼自己的心志不被外界的一切影响，多年来受益匪浅。
<br/>　　李浩然站在亭子里，左右张望了一番，说道：“你这桃花山的风水是不错，不过其中有断相，待我回头禳补一番，化险为夷，至少为你宗门增添百年香火。”
<br/>　　山河真人心头大喜，急忙拜谢，躬身站在李浩然身后不远处，一直小心翼翼地偷看他的脸色，见李浩然一直站在悬崖边上不动，心中有很多疑问，也不敢说。等了好半天，李浩然一直不开口，山河真人只好战战兢兢地说道：“不知道祖师有什么吩咐，小人好去照办……”
<br/>　　“你是不是很好奇，我从哪里来，来你桃花山做什么？”李浩然头也不回地说道。
<br/>　　山河真人内心忐忑了一下，道：“祖师的事，小人不敢过问。”
<br/>　　李浩然道：“你桃花山算是我之后人，有些事，我却要找你们帮忙……我来桃花山之事，吩咐下去，不许任何人透露。”
<br/>　　山河真人急忙应承下来。
<br/>　　李浩然沉吟片刻，道：“我来这里，是要借助你桃花山的力量，帮我找两个人……”
<br/>　　山河真人数着耳朵听着。
<br/>　　“一个叫叶少阳，是茅山第三十九代掌教……”
<br/>　　山河真人一听，眉头立刻皱起来，拱手说道：“祖师，您……”
<br/>　　“我知道你想说什么，他是百年后后茅山掌教。”李浩然虽然不想说这些，但是为了让人家把事情办成，也只能尽量提供一些信息。
<br/>　　“徐福用逆转时空的山海印，将他从百年后送到了这里，目前下落不知，我必须找到他。”
<br/>　　“百年之后……”山河真人傻傻地看着他。
<br/>　　但是李浩然并没有更出更加细致的解释，实际上，他自己也不太清楚这是怎么回事。
<br/>　　山河真人自己琢磨了半天，虽然这件事太过不可思议，让他难以接受，但他相信青牛祖师不可能随便乱说，认真思考了半晌，忍不住问道：“这么说，祖师您也是……”
<br/>　　“不错，我也是从百年后而来。这件事你无须知道太多。”
<br/>　　“是！”山河真人不敢再问。
<br/>　　“你要尽最大努力，帮我寻找叶少阳。”
<br/>　　山河真人抱拳说道：“弟子从命！只是……不知这个叶少阳何等样人？”
<br/>　　李浩然想了一下说道：“二十出头，灵仙牌位，身负七星龙泉和一干极品法器。”
<br/>　　“这……”
<br/>　　山河真人当场震住，傻傻地看着李浩然，“祖师恕罪，不是弟子怀疑，只是……二十出头的少年，怎可能是灵仙牌位，会不会……”
<br/>　　李浩然转头看着他，冷笑一声：“你以为他是你们这样的普通人？他是百年之后的法术界第一天才，茅山单传，十六岁便受封天师，前途无量。”
<br/>　　尽管叶少阳算是他的仇敌之一，不过在李浩然的心中，叶少阳也算是一等人物，尤其对人间法术界来说，他是鹤立鸡群般的所在，其余那些凡夫俗人，不论是道心还是实力，都没有办法跟他相提并论。如果抛开私仇来说，在百年后的末法时代，叶少阳是当时他在人间所见的唯一的一位亮色。
<br/>　　至于道风……李浩然从就没把他当成人类看待。他已经大致猜到了道风的身份，只可惜自己现在回到了百年之前，一切都无法查证了。
<br/>　　“天下竟有这般才子！”山河真人倒吸了一口凉气，说道：“敢问老祖，还有什么关于此人的信息没有？”
<br/>　　李浩然道：“没了，你记住他的名字便可。”
<br/>　　“这……天下之大，只知道他的名字，这却从何找起？”
<br/>　　“他是法师，就算来到百年之前，所做之事，也一定会跟捉鬼降妖有关……”李浩然笑了笑，“以他的天赋和实力，势必会声名远播，你通过法术界的关系，多帮我打听着便可，若有他消息，便来告诉我。”
<br/>　　李浩然说完，抬起左手，凌空画了几下，手指所过之处，留下了一抹淡淡的灵力组成的印记，最终形成了一副奇怪的双鱼图，看上去仿佛有一汪清水，中间两只像是蝌蚪又像是小鱼的黑点，在清水中朝着一个方向缓缓游动着。
<br/>　　李浩然道：“这是上古八卦，你能悟出多少，看你机缘了。”
<br/>　　山河真人一听，内心颤了一下，朝着凭空出现的这幅图画细看过去……
<br/>　　图画是李浩然的灵力所编织，随着他把手放下来，灵力没有了源头，开始缓缓消散，最多也就过了两分钟，图像完全消失了。
<br/>　　山河真人仍然怔怔地望着图像之前存在的位置，表情如痴如醉，过了很久，总算回过神来，跪倒在李浩然面前，感激涕零，“谢祖师栽培！”
<br/>　　李浩然道：“这地方我要了，你去吧。若有叶少阳的消息，及时通报，若无消息，任何人不许上山。”
<br/>　　“弟子遵命！”
<br/>　　山河真人爬起来，脑海中还不断回味着之前从图像中领会到的有关道法的奥义，失魂落魄一般地走下山去。
<br/>　　李浩然望着悬崖下方，抬起了自己的右手，握紧了拳头。
<br/>　　“道风！！”李浩然喝了一声，他这条右臂，看上去完好无缺，却是后来自己用法力再生而出，虽然也能做一切用处，但根本不能跟自己本来的手臂相比。
<br/>　　星宿海一战，对他的影响也是极大。
<br/>　　在徐福作法的那一刻，他根本就不知道那是怎么回事，当时只看到叶少阳等人消失在一片光华之中，以为徐福要把他们带走，逃离战场，为了追杀他们，这才奋不顾身地冲了进其余……
<br/>　　出来之后，叶少阳等人都不见了，自己身在一片荒野之中，他跟叶少阳等人一样，也郁闷了好久，到处寻找和求证，最后才知道自己穿越到了将近一百年前……
<br/>　　李浩然冥思苦想了很久，也不知道徐福为什么要这么做，他也很想离开这里，回到自己熟悉的那个世界，在青冥界，尸族正在入侵，作为阐教的一份子，对于战局进展他也是十分关心，在这里一天，他都有如坐针毡的感觉。
