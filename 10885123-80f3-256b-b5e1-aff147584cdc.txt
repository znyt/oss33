　　第223章
<br/>　　程大雷猛地一惊，看看老者这年纪，再加上和李婉儿有几分相似的脸颊……
<br/>　　该不会，该不会……
<br/>　　想想当初微服的李行哉，喜欢男扮女装的李婉儿，还有面前的老者……
<br/>　　我擦，原来你们一家祖传这个。
<br/>　　程大雷深呼吸，深呼吸，让自己砰砰乱跳的心脏镇定下来，一切就当做没发生，刚才怼的是不是有点太狠了，就当自己没认出来。
<br/>　　“老头……”老者的脸色明显已经有些不好看。
<br/>　　“是呐，我们都这么叫他。”
<br/>　　“还都这么叫？”脸色更加难看。
<br/>　　“万年长寿为老，天下之首为头，对明帝陛下如此尊称，有什么不对么？”程大雷一脸懵懂。
<br/>　　“呃，好吧。”老者面色稍霁，伸出手帮程大雷擦了擦汗：“你脸上为什么一直冒汗，是又不舒服么？”
<br/>　　“热，太热。”程大雷心砰砰直跳，堂堂明帝竟然给我擦汗，可别把我的胡子擦下来。
<br/>　　好在老者只是象征性的表示一下，接着就将手帕放下了。
<br/>　　“你这次救驾有功，老头一定会重赏你。”
<br/>　　“老头？”
<br/>　　“明帝陛下呐，你们不是都这么叫么。”
<br/>　　“呃，呵呵。”程大雷干笑两声：还挺记仇。
<br/>　　“这次你必定加官进爵，想好要什么赏赐了么？”
<br/>　　“赏赐！”程大雷轻叹口气：“舍身护驾，不是身为臣子该做的么，若说赏赐，又有怎样的赏赐比身家性命重要。”
<br/>　　“呵呵，看来你要拒绝封赏？”
<br/>　　“不，我不会拒绝。”程大雷坚定道：“我要明帝派我到最危险的地方，仗最难打的地方，身为军人，马革裹尸，战死沙场，方才是给我最大的封赏。”
<br/>　　“咦……”老者明显顿了顿：“没想到你还挺忠心。”
<br/>　　“谈不上忠心，我认为这只是一个军人应该做的，只是能这样做的人太少了，才显得我有点忠心。”
<br/>　　老者又顿了顿，叹道：“西北那边的仗应该很难打吧？”
<br/>　　“很难打。”程大雷下意识想起青牛山一战。
<br/>　　“是呐，很难打，屡战屡败，偌大帝国，竟挡不住一群蛮夷。”
<br/>　　程大雷经历青牛山一战后，有时也在琢磨这些事，他随口道：“大武以农立国，戎族逐草而生，帝国打败戎族一百次，其实都伤不到戎族元气，而帝国只要败一次，就会伤筋动骨。攻不可攻，只好据城而守，可从东到西漫长一道防线，对帝国国力是一种巨大消耗，如此只能越守越弱，越弱越守。”
<br/>　　“越守越弱，越弱越守，你说得有几分道理。”老者道：“那帝国就只能这样下去么？”
<br/>　　“倒也不是没有法……”程大雷忽然想起，面前坐着的可是帝国天子，自己也不是信口胡诌的键盘侠，如果一句话说错，可就是掉脑袋的事情。
<br/>　　“说呀，为什么不说下去，你有什么法子？”
<br/>　　“请一支精兵，直捣黄龙，将戎族彻底击溃。”
<br/>　　“帝国没有这样的精兵，也支撑不起这样一场北伐。”老者心里有些失望，也不过如此嘛。
<br/>　　“强有强的法子，弱，也有弱的法子，倒还是有个法子，只是不敢说。”
<br/>　　“为何不说，朕……随便说说嘛。”
<br/>　　果不愧是操纵数千万人命的一国之君，一抬眼程大雷就感觉到如山的压力。
<br/>　　“这个法子有些疯狂。”程大雷道：“在西北之地放开一道口子，引大军深入，以帝国复杂的地貌为战场，关门打狗，围点打援，逐渐将入侵的戎族吞掉，重挫戎族元气。”
<br/>　　“关门打狗？你是说以数百万百姓的生命为代价，主动将戎族引进来，甚至，放弃长安。”
<br/>　　程大雷轻轻点点头。
<br/>　　“不行，这太疯狂了，简直是残忍。”
<br/>　　明帝有些失神，此刻他大概已判断出来，程大雷应该已知道自己是谁。
<br/>　　“除此之外，就没有别的法子了。”
<br/>　　明帝沉默一会，目光看着程大雷，缓缓道：“你比我想象得要强，我想明帝那个老头，不会再让你做个小校尉的。最后，我还有一个问题。”
<br/>　　“请讲？”程大雷觉得自己已经混过去了，明帝明显没有生气么。
<br/>　　“人有病，药可医，国有病，可有药医，你觉得，帝国还有药救么？”
<br/>　　程大雷一激灵，这话是一个太医和一个病人该谈的么，怕只怕，明帝已经明白自己识破对方身份。
<br/>　　戎族环侍，皇权衰落，一眼望去，豪族世家争权，诸侯阳奉阴违……这情况傻子也能看得出，明帝自己会不明白。
<br/>　　答案简单到只需要一个字，不过上嘴皮碰下嘴皮的事情，可面对当朝天子，你敢说真话么。
<br/>　　程大雷抬头迎上明帝灼灼的目光，此时二人目光碰撞，心里都如明镜相仿。
<br/>　　程大雷装作什么都不知道的挥挥手。
<br/>　　“没救啦。”
<br/>　　明帝从房间离开时，苍老的身躯仿佛又佝偻几分，看着他的背影，程大雷心底忽然升起一丝悲悯。
<br/>　　这自然是很可笑的事情，可那一刻，程大雷真的觉得他其实挺可怜的。
<br/>　　院外，刘阿吉忙迎过来，伸手扶着老者，二人沿着街道往前走，穿过斑驳的树影，一乘小轿跟在后面。
<br/>　　“陛下，我扶您上轿吧。”
<br/>　　“不了，我想走走路。”
<br/>　　“是不是他说什么话，惹您不开心了。”
<br/>　　“没有，没有，说得挺好，有些地方也说得挺对。”
<br/>　　“那么陛下，您看这个人可用么？”
<br/>　　明帝沉默着，没有说话。
<br/>　　“要不将他赶出长安，回他来的地方。”
<br/>　　“可用，不仅要用，还要大大的用，狠狠的用。”
<br/>　　……
<br/>　　未央宫，御书房。
<br/>　　一个人影鬼鬼祟祟的小太监，向里面探进来脑袋，其他太监看见了，也装作没看见的样子。
<br/>　　“既然来了，为何不进来，在外面鬼鬼祟祟的做什么？”明帝轻叱一声。
<br/>　　小太监这才走进书房，只见明帝案前放着几本摊开的纸笺，不过他此时并没有看，而是端详着案上一柄铁枪。
<br/>　　明帝扫了她一眼，微怒道：“怎么又穿成这个样子，传出去像什么话？”
<br/>　　“我穿成这样怎么了，还不是和父王学的。”
<br/>
