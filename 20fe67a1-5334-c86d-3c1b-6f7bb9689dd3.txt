　　这些邪物和法师，与他们本命法器之间都有神识中的联系，陆续地也都找到了自己的法器。也有很多没找到的，被小马叫到一边等着。
<br/>　　法器和魂器，也有很多无主的——要么是主人没加入联军，要么就是死了，小马做主，让他们一人挑一件称手的，不许贪多。
<br/>　　叶少阳看自己的茅山弟子清波空着两手，于是挑了一把七段光的拂尘给他，让他回去好生祭炼。清波美滋滋地道谢。
<br/>　　十几分钟后，所有人都挑选到了合适的法器或魂器，一个个爽地不行。小马趁势鼓动情绪：“我说诸位，吃水不忘挖井人，如果没有叶天师，谁能带你们找到这些法器？”
<br/>　　大伙深表赞同，如果说昨天晚上见到叶少阳，那种臣服是精神上的，那眼下的收获就是物质上的，在场的法师和邪物，没一个不对叶少阳真心臣服。
<br/>　　叶少阳推倒了空空的货架，这才看到了自己的七星龙泉剑，连着剑鞘反插在地上，心头一颤，上去拔了下来。
<br/>　　锵的一声，长剑拔出，立刻发出一道恢弘的紫气，熠熠生辉。
<br/>　　四周那些邪物，都慑于七星龙泉剑的凌厉，四散退开了十几米远，但又好奇地一个个瞪大眼睛观察起来。
<br/>　　法术界传说中头一等的法器啊，人间五大法器之一、茅山三大神器之首，七星龙泉剑宝剑！
<br/>　　叶少阳将罡气灌入剑中，熟悉的感觉立刻传来，他兴奋地用中指在剑锋上弹了一下，宝剑立刻发出嗡嗡的声音，有如龙吟。
<br/>　　大伙纷纷赞叹。
<br/>　　突然，叶少阳感到握剑的那只手一阵钻心的疼痛，急忙还手，摊开手一看，掌心多了三块红斑，与此同时，自己掌心的肉里面似乎有什么东西在飞快移动，转眼进入了经脉，叶少阳大惊失色，也是他反应快，左手在肩膀上连点了几下，封锁住经脉，才将那个东西拦截在胳膊的经脉之中。
<br/>　　整条胳膊都麻了。
<br/>　　“快，中计了，小马你们快走！！”
<br/>　　小马懵逼地看着叶少阳，刚要开口，突然感觉脚下一软，低头看去，坚硬的土地如同流沙一般向下陷落，与此同时，有碧绿色的水中从土地的缝隙里渗透出来，蔓延开来。
<br/>　　人群大呼小叫起来，邪物们立刻起飞，想要逃离出去，结果飞到了五六米高，突然一阵轰轰雷鸣，一道道的电弧，从上方蔓延开来，呈现出一个半圆形，电弧流动，交织在一起，整个就像是电弧编织成的一张网，锅盖一般将他们所有人罩在了中间。
<br/>　　那些飞起来的邪物，都被打落下来，跌入水中。
<br/>　　场面立刻乱作一团，众人大呼小叫起来，四下奔走，却是一旦碰到电弧立刻就被弹回来。
<br/>　　“冷静，不要乱，他妈的，谁乱我先灭了谁，都围在一起！”小马大声喊叫，勉强将这些人呵斥住，跳到水里去，游到叶少阳身边，问他怎么样了。
<br/>　　“我中了某种法术，有东西进入我经脉了，应该是道化虫一类的东西，还好给封在胳膊里了！”
<br/>　　如果自己猜的没错，自己中的是一种“灵虫”，这东西不是生灵，但却如同生灵一样，能移动，能自主的执行命令。
<br/>　　这就像是电脑病毒一样，不是生命，却有着跟真实生命一样的特征。
<br/>　　灵虫有无数种，各式各样，是一种极其厉害但是阴损的法术，一旦进入人体内，有的吞噬罡气，有的啃食内脏，有的扎根在魂魄里……虽然目的不同，但有一个共同的特性，就是一旦入体，十分难以清除。
<br/>　　叶少阳一瞬间就明白了怎么回事，心中直骂对方太阴损了：这只灵虫，是附着在剑柄上的，自己之前就算再再再再谨慎，在拿到七星龙泉剑的一刻，注意力自然会被剑锋吸引，狂喜之下，警惕心也会消失无踪。
<br/>　　恰恰就是这一个空子，让这灵虫找到了机会，钻进了自己体内……
<br/>　　灵虫比大多数蛊虫都厉害十倍，不是一般法师玩的，至少得都地仙以上的实力，才能饲养灵虫，一旦出错，还容易反噬主人，因此人间这玩意十分稀少——当初后卿身上长得那个蝎子一样的东西，其实就是灵虫的一种，主的是封印。连尸王那种实力的，尚且都会着了灵虫的道，可见有多厉害。
<br/>　　叶少阳左手握着右臂，乱七八糟地想了一通，突然觉得脚下的地面越来越远，很快就双脚彻底悬空，人也悬浮在了水里。
<br/>　　水是碧绿色的，彻骨生寒，不断地往上冒着寒气。
<br/>　　叶少阳起初倒没感觉这水有什么异样，但他很快就发现了不对劲，一把拉住面前的吴嘉伟说道：“白眉，你感觉到这水不对劲了吗！”
<br/>　　“这水……我怎么感觉有一股吸力，好像体内的真气在流失？”
<br/>　　那就没错了。叶少阳深吸了一口气，的确，这水是在流动着的，一点点将他们体内的真气带走，但真气流失的速度很慢，难以察觉，叶少阳才不敢确定。
<br/>　　“小叶子，你手咋样了！”小马追问着。
<br/>　　“没事，但一时间不能作法了。”
<br/>　　他将右手经脉封住，灵虫虽然被封住了，但真气也没法再灌入右臂中，否则让那灵虫进入心脉，后果不堪设想——方才他感知了一番，那只灵虫虽然恶毒得很，但他毕竟是上仙，灵虫又被封印在了右臂中，只要有足够的时间，他可以将真气一点点压入右臂，依靠自身强大的修为，将这只灵虫逼出去。
<br/>　　前提是打坐入定，按照他的推测，大概得有两个周天的时间，就能将这只灵虫逼出来，但是……眼前的局面都乱成了一锅粥，水声火热的，哪里有两个周天的时间给他调息？
<br/>　　叶少阳强迫自己冷静，朝前后看去，这才发现，他们所在的位置，正好是两座山的中间，身下的碧绿色的液体，就像是一条河，从尽头处流淌下来，地陷之后，这里就成了一条河，而头顶上空那电弧组成的“罩子”，正好就把这几百号人罩在了河里。
