　　洪荒星空深处，雷闪电鸣，天雷地火，汹涌滚滚。
<br/>　　于那恐怖的天劫之中，盘坐这一尊伟岸的身影。不管天劫如何打击在他身上，他都不动分毫，好似永恒如此，亘古不变一般。
<br/>　　千锤百炼，整整百年时光，就这么过去了，玄天就这么在天劫中度过这些时光。天劫一旦变弱，便会被他演化洞天吞噬，所以他也不缺能量。
<br/>　　这是何等可怕的事情，从来没有听说过有几人，能在天劫中持续百年时间。
<br/>　　百年雷火，百年煎熬，在生与死之间徘徊，最终渡劫成功，玄天的道行在激增，这等于是他的一次蜕变与升华，不管是意志的坚韧，还是人体的强度，，他都较之之前强大无数倍。这也务实了他转轮武皇境的根基，并拓展向上。
<br/>　　之前因为创法的缘故，他都没怎么注重自己的根基，根基也都没机会去筑牢，留下许多隐患。如今好了，经过这次天劫的洗礼，他全身的暗伤，根基上的隐患，全都不见了，而且还更上一层楼。
<br/>　　百年时间，抵得上平日苦修数万年，因为，这是渡天劫中过来的。
<br/>　　“这条路，这种法的开创者，成功熬了下来，得到的好处是不可想象的。”这是许多大神通者在感叹，很嫉妒。
<br/>　　“该回人族了。”
<br/>　　玄天起身，看向九天之上，之后转身离开，踏入不周山山谷。
<br/>　　“恭迎玄天武祖回归！”
<br/>　　无数人族见到玄天，倒头就拜，他们看向玄天的眼中，充满了尊敬与崇拜，那是一种狂热的崇拜。
<br/>　　“武祖么？有意思！不必多礼！”
<br/>　　玄天轻笑一声，他大袖一甩，众人只觉得一阵清风拂过，便不由自主地站了起来。一念之间，便将山谷中数十万人族全部扶了起来，不得不说，这份控制力也是极为惊人的。
<br/>　　“大哥！”
<br/>　　人族三祖来到玄天身边，高兴道。
<br/>　　“不错，你们都开辟出了自己的洞天，修为都到了武尊境了，非常不错。”
<br/>　　玄天微微一笑，有些惊诧道。他实在没有想到，传道不过百年时间，他们就修炼到了洞天境了，洞天境已经可以称尊了，可以堪比洪荒中的太乙金仙了。
<br/>　　看来我还是小看了人族的潜力，或者说小看了人族三祖的气运了。玄天这样想到。不过，他并不知道，其实他身上的人族气运更盛，毕竟他是第一个被创造出的人族，人族气运他最多。
<br/>　　如若不然，他也不可能在人族被创造出来一千多年时间中，便创出适合人族的武道法门，虽然这其中与他重修有关，但重修就算再快，再加上新创一法的原因，也不可能如此之快。
<br/>　　所以说，气运还是非常重要的。更何况他身上不仅有人族的气运，他还有女娲圣人的一点气运以及前身的气运，虽然很少，但不可否认它的确是存在的。如若不然，他的修行虽然会很快，但却绝不会如此之快。
<br/>　　“让大哥见笑了！”
<br/>　　“无妨，此次传道我大有所悟，还需要去闭关，你们还需努力修行，须知，在这洪荒之中，比我人族强大的人有很多，我人族还是很弱的。”
<br/>　　“好的，大哥，你尽管闭关，人族之中，我会照看的。”燧人氏说道。
<br/>　　“希望如此。”
<br/>　　“嗯？”
<br/>　　玄天好似突然间想起了什么，他伸手截取一段不周山山体，之后祭出武道神火，将这节不周山炼化成九块石碑，不周山乃盘古脊背所化，其中蕴含着盘古大神的气息，自是十分不凡。
<br/>　　之后，玄天分别在这九块石碑上刻下九条道纹，它们分别代表着金、木、水、火、土、风、阴、阳、星辰。
<br/>　　这些都是组成世界最基本的元素，也是最基本的法则，其他一切的运转离不开它们。他在每一块碑中留下自己对这些基础元素的领悟，但并没有留下功法，有的只是最基础的道韵。
<br/>　　之后，玄天将它们放在人族祖地之中。
<br/>　　“这是我留下的武碑，以后可以让人族修士来领悟，能领悟多少就看他们自己的了。这武碑只是一个引子，我希望他们可以借此创出属于自己的功法。”玄天对着燧人氏三人叮嘱道。
<br/>　　“好的，大哥，你放心。”
<br/>　　“太好了，有了这些，我人族也算是有底蕴了，差点只是时间，以后我人族必能崛起！”
<br/>　　他们每一个人都对未来充满希望，玄天都看在眼里，他并没有多说什么，人族不经历战火的洗礼，不经历更多的困难的磨砺，想要就此崛起，这怎么可能？他们将这个世界想得太好了，真的有那么好吗？未必！
<br/>　　“但愿如此！”
<br/>　　玄天淡淡的说了这么一句，说完，不等他人回答，他便直接来到女娲庙旁的人祖殿中，开辟出一个道场，闭关了。此次传道，他的收获才是洪荒之中最多的，他需要好好闭关，整理一番。
<br/>　　女娲庙是在人族新生后，为了感谢女娲娘娘创造了人族，而建立起的庙堂，里面供奉的只是人族圣母女娲娘娘。每年都会有人去进香火，祭拜女娲娘娘。
<br/>　　在女娲庙旁，还有一庙，为人祖庙，供奉的就是人族四祖玄天、燧人氏、有巢氏与缁衣氏。玄天的道场就是开辟在人祖庙中，在庙中随便开一方小世界，他还是能够做到的。
<br/>　　洪荒之中，一般来说，到了准圣，便能开辟小世界了，到了圣人一般都能开辟中千世界，或者大千世界。
<br/>　　世界分为最初的洞天，洞天的极限是九千九百九十九里。之后是小世界，小世界的极限是十二万九千六百里，之后才能晋升中千世界，三千个小世界为一中千世界。而中千世界的极限却是十二亿九千六百万里，之后才是大千世界。
<br/>　　如洪荒，就是一方大千世界，而且还是一个最为顶级的大世界，以后更可能还是一方完美世界。所谓的完美世界，就是天地法则俱全，天、地、人三才大道平衡，万道平衡的世界。
<br/>　　在洪荒之中，有这么一句话：“条条大道可证混元”，这就是说洪荒的完美。虽然说每方世界都有三千大道，但，三千大道的构成却是不同的。实际上在其他一些世界中，成道的方式是固定的，只有领悟某种或者某几种特定的法则，最后，才能成道。其他法则一概不行。
<br/>　　如，某个名为星辰变的世界中，只有领悟空间法则才能成为神王，领悟时间加速、时间停顿、时间倒流，才能成为天尊。神界中，每个神王都需要领悟空间法则与时间法则。修行其他法则的也有，但是很少。这几乎是成为某种定律了。
<br/>　　这种世界，还不算完美世界。
<br/>　　听说，在完美世界之上，还有永恒世界，那种世界才算是真正的永恒不朽，不论经历多少无量量劫，它都不会破碎。不过，这一切都是传说，谁也不知道是否永恒世界真的存在？
<br/>　　世界一般是有寿命的，只不过这种寿命是极为悠长的，不说一般人，就连许多大神通者都直接感受不到，所以才会认为天地恒久。
<br/>　　但，实际上这是不对的，当世界寿命到头之时，修士若是不能超脱，便会随着世界一起毁灭。
<br/>　　所以，每个修士的目标都是超脱，尽管有许多修士都不真正明白“超脱”一词的真正含义，但并不妨碍他们为了这个目标而努力。
<br/>　　他们又怎么知道，修士皆求超脱，但最后真正能够超脱的又有几人？就算是在他们眼中高高在上的圣人，也不敢说自己能够超脱。
<br/>　　圣人又如何？在天道眼中，圣人一样是蝼蚁，只不过圣人是一个较大一点的蝼蚁而已。
<br/>　　超脱？何时能够真正超脱？证得大道圣人不朽境，就真能够超脱吗？谁也不知道。
<br/>　　这一切，还等着未来人去一一验证。
