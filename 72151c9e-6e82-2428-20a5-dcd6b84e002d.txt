　　“啊啊啊。”
<br/>　　也就在这个时候，有一个士兵忽然被什么东西给拖了过去。
<br/>　　林峰和周华也是被惊了一下，侧头瞧去就看到一个人不人鬼不鬼的玩意穿着个黑色的阿迪卫衣，此时此刻那个士兵已经被他大卸八块了。
<br/>　　“丧尸？六阶的？”
<br/>　　刷。
<br/>　　这个丧尸再次张开了嘴巴，他的舌头就好像青蛙的舌头一样不断的延伸，再次把一个士兵给包裹了然后拖了过去。
<br/>　　哒哒哒
<br/>　　其余士兵几乎在瞬间反应过来纷纷扣动扳机去开火，但是这个丧尸速度很快，直接消失不见了。
<br/>　　“林峰，你看到没有，那个丧尸似乎变异了。”周华道。
<br/>　　“嗯，看来丧尸之中出现变异丧尸了。”林峰点了点头，这可不是什么好消息，林峰之前遇到的丧尸都还是个人，但这个已经不算人了，居然舌头可以像青蛙捕捉昆虫一样的突袭。
<br/>　　嗡嗡
<br/>　　也就在这个时候忽然远处飞来了一群密密麻麻就好像是乌云一样的东西。
<br/>　　“不好，是变异蝗虫群！”
<br/>　　林峰立即面色微变，汉城这个地方之前闹过蝗虫。
<br/>　　“喷火兵！快！”
<br/>　　周华大手一挥，立即喷火兵全部上前对着空中喷火，只看到那空中扑簌簌的狂掉小火球，在这一瞬间，最起码也死了几百只变异蝗虫了。
<br/>　　一群群蝗虫就好像小火球一样扑簌簌的往下掉。场面颇为的壮观。
<br/>　　但是更多的蝗虫直接冲入了大部队，许多士兵都是下意识的扣动扳机，作为军人，唯一值得信赖的自然就是他手中的步枪了。
<br/>　　然而，蝗虫太多了，这最起码也是上万之多，那些没有戴任何防护装备的军人直接被蝗虫钻入了耳朵钻入了鼻孔之中，然后一个个都倒在地上抽搐，没过多久，脑袋里的东西就已经被蝗虫给吃的干干净净，然后蝗虫在里面产卵，产完卵后，再从耳朵或者鼻孔之中爬出来，继续寻找下一个目标。
<br/>　　林峰见此直接是拿出火焰符箓对着半空之中的蝗虫群就是来了那么几下，然而也无济于事，蝗虫群足足数万的存在。
<br/>　　不过很快，这群蝗虫群就直接飞走，因为后面出现了几百头的变异鸟。
<br/>　　“难道说，这群变异蝗虫群在逃生？也是啊，鸟吃蝗虫的，哪怕变异了也基本上是这个规律。”
<br/>　　林峰顿时想到了什么。
<br/>　　这群变异鸟追着蝗虫追着好好的，忽然看到了林峰这么一堆人，顿时放弃了蝗虫，而是朝林峰他们俯冲而来。
<br/>　　而部队在付出三百个尸体后，全都反应过来，扛起自己的步枪就是扫射，子弹打蝗虫不好打，但是打鸟却格外的轻松，尤其是就这么一百头都不到的变异鸟。
<br/>　　哒哒哒一片片弹幕朝空中射去。
<br/>　　“林峰兄弟，这一次，轮到我们两个干瞪眼了。”
<br/>　　周华无奈道，打地上的东西他们可以出手，但天上飞的，他们没枪的也无能为力。
<br/>　　“那可不一定。”
<br/>　　林峰掏出98K，直接瞄准了半空之中的一头五阶变异鸟。
<br/>　　“哦？林峰小兄弟还会狙击？”
<br/>　　周华眼睛亮了，然后他也拿了一把狙击，作为兵王，他肯定样样精通，狙击不在话下。
<br/>　　“林峰小兄弟，既然如此，我们两个就比比看谁打的鸟多如何。”
<br/>　　周华道。
<br/>　　“没问题。”
<br/>　　林峰拿起98K就是砰的一下一枪打爆了一个要俯冲下来猎杀士兵的五阶变异鸟，顿时这个变异鸟脑门子炸开。
<br/>　　“好枪法！’
<br/>　　周华自然不知道林峰是因为狙击精英才会变得如此犀利的。
<br/>　　然后他也一枪打爆了一个一个变异鸟的脑袋。
<br/>　　然而林峰已经打下来了第二只。
<br/>　　看的周华是佩服不已，他就感觉林峰就是个天生的战士，小小年纪，居然狙击如此厉害，这简直就是匪夷所思的事情。
<br/>　　然后全队人就看到林峰一个人在那里秀操作，林峰基本上都不需要瞄准，就好像开了自瞄挂一样的，一枪一个。
<br/>　　“不好，变异鸟群里面有个八阶的存在！”
<br/>　　砰！
<br/>　　下一刻，这个八阶的变异鸟眼珠子爆了。
<br/>　　顿时变异鸟吃痛的要飞走，但下一刻，砰又是一颗子弹射入了对方的眼珠子之中顿时爆头。
<br/>　　成为一具尸体从空中掉落。
<br/>　　一群人都看的瞠目结舌。
<br/>　　周华也是表情格外的复杂，这么多头变异鸟，他一共狙杀了大概十头不到，有三十多头是被其余士兵乱枪打死，剩下的四十多头包括这个八阶的存在，全都是死在98K之下。
<br/>　　“林峰小兄弟，你还真的是让人佩服！”
<br/>　　周华此时此刻是佩服的五体投地，不只是他，其余士兵也是如此，一个个看林峰的眼神之中除了狂热就是信服。
<br/>　　一个初中生，狙击如此了得，如此出众，他们这些当了两年三年甚至七八年的兵，简直无地自容。
<br/>　　变异鸟的出现，反而是救了大部队许多人一条性命，被变异蝗虫带走几百条生命的部队，在变异鸟的袭击下，却是零伤亡。
<br/>　　原本一千多人的部队，也只剩下八百多人。
<br/>　　剩下的八百多人队伍继续挺进。朝医院不断的赶去。
<br/>　　“这个陈博士，为了救他，牺牲了这么多人的性命，希望他真的是个专家，而不是砖家。”
<br/>　　林峰看着那些死掉的士兵遗体心中五味陈杂，这些遗体之中很多也就是个小伙子，比林峰也大不了多少岁。

<br/>　　一家医院内。
<br/>　　一个护士站内，这里有十几个民警和几个拿着步枪的武警。

<br/>　　在她身边则是好几个五十多岁的中年人。

<br/>　　“就这么点食物，也是几位警察同志拿命换来的。”
<br/>　　“我们几个老古董，手无缚鸡之力的，实在是不好意思再去让他们为我们牺牲了，他们已经牺牲的足够多了。”
<br/>　　“没错，陈博士，若是基地的救援这几天还没到的话，你就和王队长他们走吧，别管我们了。”
<br/>　　这几个中年人都是物理学家。在国内也算是小有名气的，而这个大概二十七八岁的女人，就是赫赫有名的陈博士，非常的年轻。

<br/>　　武警中队队长王队长就端着枪回来了，他摇了摇头。
