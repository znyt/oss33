　　第289章
<br/>　　程大雷突然这么热情，搞得牧长庚有些摸不着头脑。
<br/>　　目前，程大雷还揣测不明白牧长庚得了来意，事实上，牧长庚的确是从琴川关而来，目的是刺探，或者是刺杀。
<br/>　　“俺家里遭了灾，父母都死了，没有活路，特意来投奔大王爷爷。”牧长庚。
<br/>　　“你家里是什么地方，遭了什么灾跑出来的？”徐神机。
<br/>　　“不重要，一点都不重要，以后张兄弟就是我们兄弟，你叫我大当家就行。”程大雷。
<br/>　　牧长庚眼睛一亮，心道这程大雷蛮好糊弄啊。事实上，为了这次潜入计划，牧长庚准备多多，而他又是一个格外心细之人，关于自己的背景，来历都编好了说辞，真如果蛤蟆寨派人去查看，也绝不会有任何破绽。
<br/>　　却是没想到，程大雷连问都不问，倒是让他省了很多力气。
<br/>　　徐神机也不明白程大雷这突然的热乎劲儿是从哪里来，他顿了顿道：“那么，考核一下吧？”
<br/>　　“还需要考核？”这倒是牧长庚没有想到的。
<br/>　　事实上，慕名而来的每个人都要进行考核，擅长捕鱼的就去捕鱼，擅长打铁的就去打铁，什么都不会有膀子力气的，就编入战斗小队。
<br/>　　考核无非就是测试一下个人气力，搬举石锁，会拳脚的可以耍上两套，有时候也可以和山寨的弟兄对练。
<br/>　　“我来，我亲自和张兄弟过过手。”
<br/>　　“大当家，这个不太好吧？”徐神机。
<br/>　　“有什么不好的。”程大雷站在牧长庚对面，道：“来，尽管打我。”
<br/>　　牧长庚真想试一试程大雷的底细，当然，他不会亮出真正本事，不然岂不是会被程大雷看出破绽。他向程大雷打来，姿势笨拙，脚步趔趄。
<br/>　　啪！
<br/>　　程大雷伸出手来，响亮亮一个耳光打在牧长庚脸上。
<br/>　　牧长庚和徐神机都愣住了。
<br/>　　“这这……张兄弟，你没问题吧，手快了，手快了。”
<br/>　　“没，没问题。”牧长庚左脸发烫。
<br/>　　“没问题就好，咱们再来。”
<br/>　　啪！
<br/>　　牧长庚的右脸又肿了起来。
<br/>　　“这个太不好意思了，我这样一打，你应该这样一拦，你怎么没有拦啊。”
<br/>　　“我……”牧长庚恨得牙痒痒，但此刻只好道：“大当家你太快了，我没反应过来。”
<br/>　　“再来，再来。”程大雷道：“我这次会慢一些。”
<br/>　　“这个，这个还是不要再来了。”牧长庚赶忙摆手。
<br/>　　“是啊，大当家你太快了，我看就这样吧。”徐神机。
<br/>　　程大雷露出很遗憾的表情：“那就先让张兄弟去铁匠铺帮忙吧。”
<br/>　　山寨有铁匠铺，现在还没开始制造兵器，只是帮山寨弟兄修补兵器之类的。徐神机安排人送牧长庚过去，看着牧长庚离去的背影，程大雷嘴角抿起。
<br/>　　“大当家，这个人有问题？”徐神机。
<br/>　　程大雷眼前一亮：“你能看出来？”
<br/>　　“你那两耳光打那么重，傻瓜才看不出来。”
<br/>　　你难道以为自己不是么……程大雷默默想着，道：“给他好好安排一下，对了，晚上安排他和你住在一起。”
<br/>　　大意了，大意了……牧长庚的出现提醒了程大雷，现在山寨有诸多人加入。会不会其中就有别方势力混进来打探情况的，自己忙碌着治兵，倒是疏忽了这件事。
<br/>　　接下来，该好好排查一下。
<br/>　　赵子龙、关鱼、张肥几个陆陆续续回来，他们回来得忒快了些。程大雷询问他们各自的情况，发现事情比自己想得顺利。
<br/>　　云中龙被送回去，果然没有什么意外，云中龙一出现就压住了场子，乖乖向赵子龙交付了赎金。
<br/>　　而高飞豹几人本意是去抢劫其他头领，但其实根本没动起手来，他们亮出蛤蟆寨的招牌，这些人就举起手来，把他们要的东西拿了出来。
<br/>　　高飞豹几人显得颇为遗憾，这完全发挥不出各人本事么。
<br/>　　第二天，程大雷让五人继续去抢劫其他头领，说是抢劫，倒不如说是去公然向十八家头领索要保护费。
<br/>　　这些人可能是被打怕了，丝毫没有反抗的心思。
<br/>　　而程大雷则开始排查山寨中的内奸，还真被他找到七个有问题的，于是就一块儿丢进了铁匠铺。
<br/>　　蛤蟆寨的人如一阵旋风般扫荡过大河两岸，十八家头领都成了他们眼里的肥羊。直到十天后，这场抢劫比武才算是结束。徐神机、刘悲、李行哉清点财物，确定各队胜负。
<br/>　　消息传到琴川关。
<br/>　　莫鸣米一张脸黑得像锅底一般，自己不去打他们也就算了。这帮人竟然主动出击，开始找自己麻烦。
<br/>　　“将军，这帮人不能留啊，长此以往，怕他们会越来越难对付。”严笛。
<br/>　　莫鸣米点点头，示意自己明白。他开口道：“牧长庚那边情况怎样，有没有消息传过来？”
<br/>　　“目前还没有，会不会是被识破了？”
<br/>　　“没有消息，有时候也是好消息，说明他目前还没有被识破。”
<br/>　　牧长庚不是不想传消息过来，而是他没办法传消息过来，确切来说，他是陷在了这里。
<br/>　　第一天他就被丢进了铁匠铺，开始抡铁锤，牧长庚的确是有把力气，可每天工作超过八个时辰，整日与铁锤，火炉为伍，牧长庚整个人都黑了一圈，而且还瘦了一圈。
<br/>　　他倒是想打探一些情报，或寻找机会，直接杀了程大雷。可奈何，实在累得够呛，每晚从铁匠铺回来，一粘枕头就着。
<br/>　　牧长庚有时候也想，自己会不会是被识破了，他们故意用这个法子折腾自己。可看铁匠铺都是这么工作的，倒不是单独针对自己。
<br/>　　他却是不知道，铁匠铺的几个人，都是程大雷觉得有问题的人。
<br/>　　如此苦不堪言的日子，牧长庚过了半个月，每天最幸福的事情，就是忙完后在床上躺一会儿，至于自己来这里的目的，他都快忘了。
<br/>　　这一日，牧长庚筋疲力尽的回到宿舍，看到徐神机守着油灯洗脚。
<br/>　　他眼睛一亮，忽然道：“徐军师，咱们山寨会不会有人来打？”
<br/>　　“来打怕什么，不是有大当家在么。”徐神机随口道。
<br/>　　“大当家有法子……”
<br/>　　“那是当然了。”
<br/>
