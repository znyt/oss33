<br/>				    她一怔，脚步停了下来，心头一动，清眸盯着空无一物的手掌，忽的心中一明：“心有通天道，何惧风雪阻仙道。”<br/>　　<br/>　　    伴随着她的声音一落，那飘荡着的雪白消失无踪，周围仿佛又恢复了原来的样子，她定了定神，看了看自己，依旧是一袭红衣在风中飘扬，身上不沾半点风雪，原本觉得的疲惫以及口渴也全消失了，就仿佛只是她的错觉一般，不曾出现过。<br/>　　<br/>　　    “九公子？九公子？”<br/>　　<br/>　　    “小友？小友？”<br/>　　<br/>　　    两道声音传来，她回过神来，朝前面看去，见老者和何殊两人正站在离她约十米的距离喊着她，见她抬头看去，两人便再次问着。<br/>　　<br/>　　    “你怎么上来了啊？你不是来看热闹的吗？怎么也上来了？”<br/>　　<br/>　　    听到这话，凤九露出了一抹笑容来，她迈着脚步往前走去，道：“闲着无事，便上来看看，也许，这上面的风景会更好。”<br/>　　<br/>　　    周围的声音传入她的耳中，身后有打斗的声音，也有惊呼的声音，而上前，依稀间可见几道身影缓步在那烟雾间走着。<br/>　　<br/>　　    老者和何殊见他缓步上来了，一步一阶，还轻轻松松，不免有些诧异。待他来到两人身边时，老者看着他叹道：“小友，你真是深藏不露啊！”<br/>　　<br/>　　    凤九笑了笑，道：“还没走到顶端，如何知我深藏不露呢？”她迈步往上走去，一边道：“两位，走吧！”<br/>　　<br/>　　    两人见他迈步往上走着，两人微顿了一下，便也跟着往上走着，他们已经走了近一半的路了，越是往上，越是难行，尤其是何殊出渐渐出现难受的情况，体内的气息也开始有些混乱。<br/>　　<br/>　　    感觉到身体已经达了极限，何殊停下了脚步，对他们道：“我已经到极限了，再上去我怕承受不住了。”<br/>　　<br/>　　    上面的凤九和老者看了他一眼，点了点头，没再多言的继续走着，而何殊则往下走去。<br/>　　<br/>　　    “小友，你没感觉吗？”老者忍不住问着，见凤九神色一直如初，没有出现半点难受的神色，不免奇怪着。<br/>　　<br/>　　    “有啊！”凤九轻叹一声。<br/>　　<br/>　　    老者一听，一双神采奕奕的眼睛紧盯着凤九，连忙问：“什么感觉？”<br/>　　<br/>　　    “这条路怎么那么长？还有多少到顶呢？”凤九轻声说着，言语间有着一丝无奈，听得老者一怔一愣的，半响没缓过神来。<br/>　　<br/>　　    好半响，老者深吸了口气，道：“我是问其他感觉，没感觉哪里不舒服？你没感觉到这仙梯上弥漫着的威压吗？没感觉体内的血气在沸腾吗？”<br/>　　<br/>　　    凤九突然间停下脚步，静顿了一下后，转头对老者道：“还真没有，我只感觉这条路太长了，爬上来有些费劲。”说话间，她从空间中摸出了两个果子，递了一个给老者：“呐，吃个果子润润喉。”<br/>　　<br/>　　    老者动了动嘴唇，却是不知说什么好了，最后只是接过果子，道了一声谢，便继续往上走着。<br/>　　<br/>　　    凤九边吃着果子边走着，也不知过了多久，前面已经没有看见人了……<br/>
