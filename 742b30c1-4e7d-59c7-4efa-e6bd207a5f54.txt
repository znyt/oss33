　　云州，黑森林，
<br/>　　一晃眼，叶尘已经回来半月有余。
<br/>　　在这半个多月的时间里，叶尘除了潜心修炼，将修为已经稳定在了金丹境。
<br/>　　还用当初从药王宗万毒窟得来的珍贵药材，配合扶摇仙果，炼制出了更高级的神魂丹药。
<br/>　　不但彻底治愈了曦月神魂损伤问题，并且神魂力量有了质的飞跃，已经开始凝聚出新的元神，离脱离肉身也已经不远。
<br/>　　原本，
<br/>　　按照叶尘的计划，等曦月的神魂能够脱离肉身而独自存在，就会带着她离开地球，前往修真界。
<br/>　　可是如今，他已经和蓬莱仙界交恶，再加上妖界和魔界，也和他诸多纠葛，难保等他离开地球之后，这些势力到时候不会找他家人的麻烦！
<br/>　　而且，虽然他已经找到进入修真界的入口，可是前途未卜，自然不能带着所有人，和他一起冒险。
<br/>　　所以，在离开之前，他必须要彻底解决掉这些威胁。
<br/>　　除此之外，苏岚母女、楚飞烟等人的修为，也必须要尽快提升。
<br/>　　唯有如此，叶尘才能放心离开地球。
<br/>　　……
<br/>　　一晃眼，又过去了将近一个月，
<br/>　　在这一个月的时间里，叶尘亲自教导监督，再加上各种海量丹药，众人的实力都有了质的飞跃。
<br/>　　楚飞烟已经达到炼气巅峰，离筑基境只有一步之遥，唐清雅和苏蔓两人，也达到了炼气六重境界，就连苏岚一向不喜修炼，竟然也突破到了炼气四重。
<br/>　　至于原本实力就不弱的雪姬、李贺年等人，更是隐隐要追上叶尘的脚步，随时都要突破金丹之境。
<br/>　　当然了，即便是他们的修为，真的可以追上叶尘，实力也不可同日而语。
<br/>　　要知道，如今的叶尘，虽然只有金丹境初期，可是如果手段尽出的话，便是化神强者都有一战之力！
<br/>　　至于如今在狂仙门中，实力仅次于叶尘的祝明飞，自从叶尘传授了他一门火系功法之后，实力也是突飞猛进，已经足以跻身蓬莱仙界的天榜前十！
<br/>　　……
<br/>　　这一日，
<br/>　　叶尘从闭关状态中出来，发现楚飞烟、李贺年以及祝明飞等人，全都在门口等候着，居然连易山河也赫然在列。
<br/>　　叶尘顿时意识到不妙，
<br/>　　既然连易山河都来了，肯定是遇到了大事，
<br/>　　“出了何事？”
<br/>　　易山河先是躬身行了一礼，然后沉着脸道：
<br/>　　“叶帅，出大事了！”
<br/>　　“最近华国突然冒出一名无名强者，一口气斩杀了我华夏数十名武道高手！”
<br/>　　“南方武林盟主魏无极，和北方武林盟主鱼化龙，全部被此人一招秒杀！”
<br/>　　“此后，龙虎山的天师符，茅山派，以及梵音寺，等隐世宗门，也在短短三天之内，全部被此人踏灭！”
<br/>　　“而且此人已经放出话来，接下来就要来找狂仙门的麻烦！”
<br/>　　“此人实力太强，而且行踪诡秘，神龙卫已经使出了浑身解数，却愣是没能找到此人的行踪，也不知道他的来历，只知道此人金发金眸，似乎并非世俗之人……”
<br/>　　易山河说完之后，脸色越发沉重，
<br/>　　短短几天时间，整个华夏武道界，几乎被这名神秘人给一窝端了，此事已经引起了华夏高层的高度重视。
<br/>　　如今整个华夏都人心惶惶，可是却都束手无策。
<br/>　　无奈之下，易山河也只有前来请叶尘出山了。
<br/>　　叶尘听完之后，不由得眉头微微一皱，
<br/>　　“莫非是蓬莱仙界的人出手了吗？”
<br/>　　不料，一旁的祝明飞忽然开口道：
<br/>　　“应该不是！我昨天前去检查过那些被杀之人的尸首，这些人的体内，包含着一股极为邪恶而诡异的力量，和仙族传承大为不同，倒像是神族的手段！”
<br/>　　叶尘顿时一愣，
<br/>　　“神族？”
<br/>　　他怎么也没有想到，率先向他出手的，居然不是仙、魔、妖三族，反而是和他几乎没什么瓜葛的神族！
<br/>　　仔细想来，叶尘除了当初在狼王冰岛，斩杀了两名西方教廷的红衣大主教，除此之外似乎并没有其他任何恩怨？
<br/>　　纵然西方教廷的背后是神族，可是对于神族来说，区区世俗界的两名红衣大主教，应该只是小角色而已，何以会让他们如此疯狂报复？
<br/>　　叶尘思绪飞转了片刻，也没能想出一个所以然，不过既然对方都已经欺负到家门口了，他自然也没有退缩的道理。
<br/>　　“那些被杀之人的尸首可曾带来？”
<br/>　　叶尘沉吟了片刻，忽然开口道。
<br/>　　众人均不解其意，
<br/>　　不过，随着易山河拨出一个电话，不一会的功夫，几名身穿军装的男子，便将一具尸体送到了众人面前。
<br/>　　叶尘来到那尸体跟前，双眼微微闭上，眉心处闪过一道赤色光芒，没入那具尸体之中……
<br/>　　过了片刻之后，叶尘猛然一声低喝，
<br/>　　“神明索魂术！”
<br/>　　呼！
<br/>　　那道赤色光芒，忽然又自那具尸体之中跳出，继而朝东南方向而去。
<br/>　　叶尘嘴角微微一扬，露出一抹冷笑，低声喃喃道：
<br/>　　“还好最近我的神魂力量，又有所精进，否则还真未必能施展出这门神魂秘术！”
<br/>　　“我倒要看看，你到底是何方神圣？”
<br/>　　说完这话，叶尘身体一晃，整个人已经循着那道赤色光芒飞遁的方向而去，瞬间便消失的无影无踪。
<br/>　　……
<br/>　　而与此同时，华夏东南，武当山，
<br/>　　在普通人看来，如今的武当山，早已经只是一个普通的旅游景点，却根本不知道，在武当山的背后，其实一直保留着古修丹道一脉。
<br/>　　而所谓古修丹道，便是上古修真道法中的一支。
<br/>　　譬如，那名满京都的燕真人，便是出自其中。
<br/>　　只可惜，此时的燕真人，半跪在大殿之中，狼狈无比。
<br/>　　而在大殿之上的座位之上，正坐着一名金发金眸的青年男子，一脸的鄙夷之色，
<br/>　　“老头，听说你是东方世俗界中最顶级的人物，如今一看，简直如蝼蚁一般！”
<br/>　　“看来那个叶狂仙，比你也强不了多少吧？柯莱特那个老家伙，果然是在夸大其词！桀桀！”
<br/>　　燕真人听到这话，不由得眉头一皱，
<br/>　　“原来你是西方教廷的老教皇派来的人，我武道山和你们教廷，似乎没有什么恩怨吧？”
<br/>　　那金发青年听到这话，顿时哈哈大笑，
<br/>　　“看来你还没有搞清楚状况啊！柯莱特那个老东西，连给本神子提鞋的资格都没有！”
<br/>　　燕真人听到这话，顿时脸色大变。
