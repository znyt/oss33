　　既然要上岛自然少不了秦琼和尉迟恭两个，这两人是李二最信任的两个手下，传说中门神的原型那可不是空穴来风。
<br/>　　不过钢铁战舰是开不到岸边的，作为尖底船，它的吃水线远比想像中要深得多，如果不想游回大唐，他们只能放小船下水，然后靠人力划到岸上。
<br/>　　小岛之上的确如李二所说，真的有烟升起，而有烟则证明有人。
<br/>　　只不过与李二所想的不同，这岛上依旧还是蛮荒，所谓有人也只是一群土人。
<br/>　　所以当巨大的战船驶进沿海的时候，他们不出意外的受到了攻击。
<br/>　　一个个光着屁股的土著，划着传说中的独木舟蜂拥而出，带着毒液的木质长箭雨点般射向战船。
<br/>　　“太上皇，要不咱们还是先等等？”长孙无忌看着下面一个个黑漆漆猴子一样的土著，叫着，跳着，发了疯一样攻击着战船，面上闪过一丝犹豫。
<br/>　　“这有什么好等的，灭了他们弹指间的事情。”尉迟恭虽然这段时间一直在家里修身养性，但火爆的脾气却一点没改，对于敢挑衅大唐的敌人向来只有一个杀字。
<br/>　　李世民看着那些不知死活，前仆后继冲上来，用手里的木头片用力往船上戳的土著无奈的叹了口气，愚昧至此想来岛上也不会有什么神仙存在了，否则怎么可能有人会用血肉之躯挑战钢铁战舰。
<br/>　　最终，兴趣索然的太上皇看了一眼独孤青云：“青云，南海你比较熟悉，这事儿你看着办吧。”
<br/>　　“喏！”独孤青云此时早就有些等的心急，这头南海老龙在南海称王称霸，何时被人如此围攻过，如果不是考虑到太上皇在，早就揍他们丫挺的了。
<br/>　　现在有了李二的许可，自然不会手下留情，一声令下打出旗语，周围数十战船上涌出无数光着膀子的汉子，手里拿着已经被正规部队淘汰掉的钢弩向那些土著发起攻击。
<br/>　　这些人都是他曾经在南海招募的手下，既不属于‘海狼’也不属于水师，硬要说的话这些人最多能算是民团。
<br/>　　不过就算是民团战斗力也不是那些土人可比的，三棱弩矢在钢弩强大的弹射力下，粗暴的撕裂那些土著的身体，带走他们那卑微而脆弱的生命。
<br/>　　而随着那些土著的大量死亡，海面上逐渐翻腾起来。
<br/>　　那是大量被血腥味吸引过来的鲨鱼，面对大量新鲜的美味，它们也开起了杀戮派对。
<br/>　　绝望的惨叫声在海面此起彼伏，被血腥味刺激到疯狂的鲨鱼开始攻击那些独木舟，用他们强有力的身将船撞翻，然后再将船上的人拖入海底。
<br/>　　惨，太惨了！就连见惯了死人的尉迟恭此时也是面色发白！牙齿竟也有些不受控制。
<br/>　　死人不可怕，死的再惨的他也见过，可是特么活生生被鱼吃掉就有些让人难以接受了，看着那些浮在水中的土著被鲨鱼活生生扯掉胳膊、大腿甚至脑袋，身为一个旱鸭子的老尉迟很为自己的老命担心。
<br/>　　“嘭嘭……”十艘钢铁战舰上突然响起了枪声，将那些还在水中挣扎的土著一一杀死。
<br/>　　这是秦琼下的命令，他实在无法接受人被鲨鱼活生生吃掉的场面，就算是连话都说不明白的土著也不行，所以他让那些跟着李二出海的禁军用子弹结束了那些土著的生命，让他们死的痛快一点。
<br/>　　……
<br/>　　半日之后，一切归于平静，海面上的浮尸被鲨鱼吞食的干干净净，夕阳下，蔚蓝的海水反射着粼粼波光，丝毫看不出中午时候这里还曾经发生过一场完全不成比例的战斗。
<br/>　　近二十余只小船被放到了海里，两百全副武装的禁军作为先头部队开始登岸，负责清扫岛上的障碍。
<br/>　　而伟大的太上皇则需要在船上再待一个晚上，等到明天一早先头部队清扫出一片安全的地域之后，才能登岸。
<br/>　　这是规则，就算是李二再心急也没有用。
<br/>　　部队登岛之后天色已经彻底暗了下来，岛上的丛森就像一只张开大口的野兽，等待着猎物的到来。
<br/>　　不过两百人的队伍登岛之后并没有急着进入丛林，而是在海边就地驻扎了下来，搭起帐篷，围起一片干净的沙滩。
<br/>　　土著的武器虽然简陋，但他们却是丛林中天生的猎手，黑暗中两百人的队伍如果贸然进入丛林估计连死都不知道自己是怎么死的，这是以近三个月来，禁军通过近百人的死亡得出的结论。
<br/>　　不过，不能进攻并不等于不能活动，至少在丛林的边缘，他们还是安全的，他们可在以这里进行一些有限的活动，比如说砍一些树木，又或者搭一些简单的防御工事。
<br/>　　而在从林的深处，那些白天进攻过战舰又侥幸没有被攻击到的一些土著正在讨论着如何应对这支能让他们万劫不复的军队。
<br/>　　“族长，他们应该是大唐的军队，我们根本不是他们的对手，如果可以的话，我们最好还是投降，他们现在在海上的活动越来越频繁了，如果再不投降归顺，只怕以后我们很难在这片海域生存下去。”
<br/>　　先知，岛上最有文化的人之一，在某些情况下，他的权力甚至可以大过族长，所以当他提出建议的时候，就算是族长也要好好考虑。
<br/>　　“先知大人，我并不赞成你的看法，我们世世代代都生活在岛上，完全没有必要去抱唐人的大腿，而且我们的勇士也不怕死，如果他们敢来，我们一定会让他们知道我们的厉害。”族长的儿子，也就是传说中的少族长眉头紧锁，似乎对先知的建议很不满。
<br/>　　“少族长，你的想法是什么我很清楚，但是白天的事情你也看到了，一百多人冲上去，结果连大唐的船都没有摸到便差点全军覆没，这已经不是不怕死可以解决的问题了，如果真的惹怒了他们，很有可能我们会有灭族的危险。”
<br/>　　“怕什么，就算是灭族，也比被他们欺压来的痛快，而且他们来这里的目的应该是为了神仙草吧，你觉得就算是我们投降了，如果他们发现了神仙草，就会放过我们？”少族长依旧不同意先知的说法，纵然他白天已经领教了大唐军械的厉害。
