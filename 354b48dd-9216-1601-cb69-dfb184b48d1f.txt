　　花费了一些时间，刘哔回到了镇上。
<br/>　　从义庄出来的时候天还没亮，现在却已经艳阳高照了。
<br/>　　镇子上也逐渐开始热闹了起来，这个时间，大部分人都已经吃完早饭，准备工作了。
<br/>　　镇门离义庄还有一段距离，刘哔赤裸着上身走在街上，自然引来街上行人的纷纷侧目，经过怡红院的时候，几个衣着暴露的小姐姐甚至对他吹起了口哨。
<br/>　　刘哔没有多加理会，径直走回了义庄，正巧碰见文才正在院子里摆弄那些花花草草。
<br/>　　“师弟，你回来了。”听见门口的动静，文才转过头来，看到刘哔的状态，顿时一愣：“你受伤了吗？发生了什么事？”
<br/>　　摆了摆手，刘哔示意自己没什么大碍，看到九叔没在院子里，开口问道：“师父呢？”
<br/>　　盯着刘哔胸口处被鲜血染红的布条，文才眼中满是担忧，没有回答他的话语，而是进屋拿了一个药箱出来。
<br/>　　“师父有些事情，出门去了，我先帮你包扎一下伤口吧，怎么搞的这是。”将药箱放在地上，文才拉着刘哔坐下：“本来师傅交代我们刻录一些符纸，你却把自己搞伤了，你还是先回房间休息吧。”
<br/>　　点了点头，刘哔推开文才的双手，他不知道自己的身体恢复成了什么样子，这幅样子，不太适合让别人看见。
<br/>　　“哎呀，我帮你重新包一下吧，用衣服包也不知道干不干净，如果被感染了……”固执的伸出手来，文才嘴上止不住的絮叨着。
<br/>　　出了这么多血，伤的一定很重吧，虽然只是认识了几天时间，但他还是非常担心自己这个师弟的身体。
<br/>　　“没事，只是被树杈不小心给刮了一下，还上什么药啊。”文才的啰嗦，让刘哔感觉有些烦躁，拒绝文才后，便也没再理会，径直向自己的房间走去。
<br/>　　回到自己的屋子里，他将所有门窗关好，解开自己先前用衣服制作的布条，扯下来的时候动作有些粗鲁，带起了一层血痂。
<br/>　　没有痛感，也不会痒，一点知觉都没有，看着胸前的透明窟窿，刘哔眉头紧皱，伤口边缘已经结痂了，但却没有新的血肉重新长出来。
<br/>　　这样说的话，‘不死’技能只是可以保证自己受到除了脑部之外的任何伤害都不致命，却无法起到恢复伤口的作用。
<br/>　　也对，曾经那些被自己搞残的丧尸，当时也没有任何恢复的迹象。
<br/>　　但是这样的话，想要伤口恢复如初，就只能想办法赚取积分，然后用系统进行修复了。
<br/>　　突然，像是察觉到什么，刘哔扯过一件干净衣服，随便套在了身上，将伤口暂时遮挡了下来，随后，门外传来了一阵脚步声。
<br/>　　“师弟，我把药箱先放在门口了，你还是使用的话，就出来拿吧。”停顿了一下，文才又继续说道：“你还是先包扎一下吧，要不，感染了就麻烦了。”
<br/>　　说罢，门外又是一阵越来越远的脚步声，看样子，文才似乎已经离开了。
<br/>　　刘哔目光微动，犹豫了一下，开门拿起了药箱。
<br/>　　他的伤口确实需要包扎一下，倒不是害怕感染，而是不包扎的话，如果被人发现了也是麻烦。
<br/>　　……
<br/>　　拿完药箱，刘哔重新回到屋里，找出几条纱布将伤口简单包扎了一下，便再也没有管它，而是认真钻研起先前紫女给自己传授的功法。
<br/>　　这套功法的名字叫做‘紫气东来’，据紫女自己说是紫府的秘传功法，具体真的假的，他也不太知道。
<br/>　　当然，至于这个紫府，他也压根不知道是什么，只能简单的猜测，这是一个势力，规模，大小，实力一概不知。
<br/>　　闷头钻研了片刻，刘哔从识海中退了出来，这上面的东西，他没看懂……
<br/>　　倒不是说这套功法有多么繁奥，而是整篇功法都是用繁体字书写的，如果只是单独一两句，他倒是也能猜出来，但是整篇都是，他就有些理解无能了。
<br/>　　晃了晃有些发胀的头颅，刘哔展开画像，伸手在紫女胸前，敲了敲。
<br/>　　“歪，有人没有，出来一哈。”挠了挠头，刘哔有些尴尬，他注意到了紫女恨不得将他生吃活剥了一般的眼神。
<br/>　　他也不知道紫女是怎么感应外界的，但是要找人家，怎么也得先打个招呼吧，这不，他一敲对方立马就出来了。
<br/>　　“什么事，说！”紫女脸上带着愤怒，眼神中一片冰冷：“还有，以后找我，直接说事，别随便摸我！”
<br/>　　讪笑了一下，刘哔有些尴尬的开口：“那个，你之前不是传授给我半套功法吗，我有些不懂的地方，想请教请教你。”
<br/>　　“哪个地方不明白？”知道刘哔也不是无心，冷哼一声，紫女的脸色缓和了下来。、
<br/>　　她还指望刘哔帮自己解除封印，自然不会再这种事情上为难他，刘哔的实力提高了，对她而言也是一件好事情。
<br/>　　“额，全不明白。”挠了挠头，刘哔解释道：“主要是上面的字我有点看不懂，你能不能给我讲解一下是什么意思？”
<br/>　　眼角抽动了几下，紫女抬手，做了一个扶额的动作，有些无奈的看着刘哔：“你……不识字？”
<br/>　　张了张嘴，刘哔犹豫了一下，点了点头。
<br/>　　她认为自己不识字就不识字吧，他也懒得解释什么。
<br/>　　翻了个白眼，紫女也没继续嘲讽刘哔，而是耐着性子，一点一点的向刘哔讲解了起来，一些地方，还加上了自己的一些心得。
<br/>　　有了紫女的指导，刘哔对脑海中这套功法的理解也飞速提升着，很快就了解了这套功法的作用和修炼方式。
<br/>　　和他想想中的不同，与其说这是一套功法，倒不如说是一种强身健体的方式，具体的功效，和地球上的气功差不许多，但是效果却要更加强大一些。
<br/>　　这套功法总共分为九层，紫女传授的只有半部，从第五层之后就没有继续修炼下去的方式了。
<br/>　　但根据紫女所说，前半部分是修炼最容易，也是提升最快的，资质一般的人，也仅仅需要十年左右的时间，就能到达第五层的巅峰。
<br/>　　而到达这个境界，一般的千年鬼物就可以轻松应对了。
