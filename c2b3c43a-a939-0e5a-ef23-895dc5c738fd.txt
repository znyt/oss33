　　杀剑凶狠，业火焚空，灵山晃动，天地皆惊！
<br/>　　灵山上下，血流成河，星辰倒转，天翻地覆，冥河老祖双剑锋利，杀道惊人，经历一番艰苦战斗，终于将准提圣人的尸身斩杀。
<br/>　　“哈哈，我血海冥河自立教以来，杀天、杀地、杀人、杀仙、杀神、杀妖、杀魔，如今杀圣，天地万物，无物不杀！今日吾当证道混元！”
<br/>　　冥河老祖一声大喝，拖着他那有些残破的身躯，心中杀道经过如今杀圣，算是彻底完善了。
<br/>　　轰隆！
<br/>　　一股强悍的气息自他身上爆发，与此同时，洪荒天地之内，无尽雷霆汇聚而来，雷海震荡，雷龙翻滚，一道道澎湃雷霆组成九天十地，汇成一片雷霆的世界。
<br/>　　风声大作，雷声惊天，整个西方世界，都变得明亮起来，无数妖魔以及修为弱小之人，都被这巨大的雷霆吓得匍匐在地，瑟瑟发抖。
<br/>　　雷霆向来是天威的代表，如今执掌雷霆的乃是天罚圣主原始，他的话，便代表着天。
<br/>　　与此同时，万劫魔尊那边再次灭杀一道接引圣人的尸身，以他混元的强悍，面对足足五位准圣圆满的圣人尸身，他一人便斩杀三人，冥河斩杀一人。
<br/>　　万劫魔尊就此罢手，因为他知道，西方二圣已经往这边赶来了。他如今还不想与圣人大战，只因那样没有利益的事情，不同不会做。如今西方教大败，数万年都难以大兴，这对于他劫魔殿来说，是最好的选择。
<br/>　　万劫魔尊看了一眼九天之上的狂暴雷霆，他是经历过一次证道雷劫，此雷劫并没有当初他所渡之劫强大，这说明冥河的积累还没有他当初深厚。
<br/>　　这雷劫是遇强则强，当然，他当初还有天罚圣主在背后出手，送给他一场机缘，如今冥河这才是正常的证道雷劫。
<br/>　　“走！”
<br/>　　万劫魔尊大喝一声，灵山上的劫魔便纷纷随着他向着太华山方向走去，他并没有兴趣来看冥河渡劫。
<br/>　　在东方人族九州，界牌关前，当然，此刻界牌关早已荡然无存了。此地如今已是百孔千疮，山脉破碎，大地断裂，空间混乱，变得脆弱不堪，经过一场圣人大战，此地就快有重演混沌的趋势了，若不是有元始在一旁定住虚空，恐怕此处早成了废墟。
<br/>　　接引与准提两人一番大战，俱皆受到重创，最后终究是将阐教众仙放了出来，且各自送上一件灵宝，他们不得不如此做，因为他们的尸身就快要被灭杀完了。
<br/>　　就在两人刚走之时，也正是冥河屠了准提尸身，证道雷劫来临之际，准提与接引各自喷出一口逆血，面色无比难看地看了一眼元始两人，随后向着西方急速飞去。
<br/>　　“没想到冥河那厮，还真有两下子，竟然还真让他成功了。”通天说道。
<br/>　　“冥河资质并不算太差，能够成为一方教主的人物，又有几个是简单之人？他能够证道，我并不奇怪。”元始淡淡看了一眼西方，西方一切景象，尽在他眼底呈现，他淡淡开口道。
<br/>　　“如此一来，证道之人数目增多，恐怕对于圣人的地位，是一个不小的冲击。”通天教主道。
<br/>　　“无妨，洪荒沉寂太久了，高手多了，岂不更加热闹。”
<br/>　　元始语气依旧很是平淡。他自从知道了诸天万界以及永恒一界之后，目光便看到了更高远之处，对于洪荒之内的小打小闹，已经不放在心上了，多几个混元强者，也无所谓，他不是鸿钧，想要控制洪荒天道，便需要控制洪荒界内的强者。
<br/>　　鸿蒙紫气，以元始如今目光来看，它是洪荒天道本源的一部分，鸿钧以此让他们成圣，恐怕早已心存他意。并以此便断了他们攀向大道之巅的道路，此生只能在洪荒之中，而且鸿钧分化鸿蒙紫气，便是为了削弱天道力量，更加容易地控制天道，最后进行合道，一举突破天道境。
<br/>　　可惜，他没想到出了元始这个变数，从原始成圣，就一切变了，原始、通天与太上的成圣，以及天柱不周山的断裂，的确是削弱了天道的力量，但却多了一个更加难缠的原始，如今正在天道空间与他争夺天道本源法则。
<br/>　　而且，他更没有算到，天道空间之中，有一团最为核心的本源，是无论如何也不能炼化的。
<br/>　　有了一个变数，那便会有更多变数，变数的作用，便是改变命运轨迹。如今，洪荒之中好多人的轨迹都变了，便是因为多了元始这个最大的变数。
<br/>　　“也是时候设法将通天与太上身上的鸿蒙紫气剥夺了，虽然这会让他们实力下降，但为了未来，这必须做。”元始看着通天，心中默默想到。
<br/>　　大战终结，双方都回各自的洞府，虽然封神榜上名额看似已经足够了，但仗，还是要继续打的，于是乎，阐教众仙继续进行自己的伐商兴周大业。
<br/>　　界牌关万仙大阵一战，截教群仙一百零九万，近两百魂入封神台，二三十万妖仙趁乱逃生，数万修士死亡魂入轮回，剩下六十多万，还在截教。截教可以说是，并没有损失多少。
<br/>　　这场针对西方二圣的大战，可以说是自三清分家便计划了。元始早知量劫过后，截教不存，西方当兴，但那又如何，他可以谋划。
<br/>　　经过二人的谋划，截教存了下来，西方大兴的天机却是还没有显现，显然是被延迟了。经此大战，截教的众多问题也显现出来了，截教之内，三教九流，不是什么人都有资格问道的。而且如今通天教主也知道了此乃天道之意，天道在于平衡，截教的存在，已经挡住了许多修士的路。
<br/>　　明白此问题后，通天教主当即便遣散了五十多万截教外围妖仙，只留下十多万左右的核心力量，这算是大战后的截教吧。
<br/>　　道不可轻传，法不可轻授！
<br/>　　因为一旦洪荒修士数量增多，天地量劫还是迟早会来的，明白这些问题后，通天教主也开始控制门下弟子数量了。有些人，可以收作弟子，但有些人，却是无论如何也不能收。
<br/>　　教主当即便宣布封岛不出，截教退出封神大战。
<br/>　　没了截教的阻拦，这一场大战可以说是注定的，申公豹挥师东进，纣王虽然还有一些能用的手下，调军前往五关阻挡，但周军气势恢宏，势不可挡，一路上连破青龙关、佳梦关、穿云关、潼关、临潼关。武王发兵渡过黄河，西、北四百多路诸侯会师于孟津，以武王为首，直攻朝歌。
<br/>　　如今南方之地还在进行人魔大战，东方之地也被魔族入侵大半，只有少数商军还在抵挡，天下九州，商朝已经失去了八分。
<br/>　　周军东进，一路上遇到梅山兄弟七人，名袁洪、金大升、戴礼、杨显、朱子真、常昊、吴龙，俱为千年老妖。七人正是截教门徒，在万仙大阵前接引渡人之时逃走，因为截教封岛，所以并没有回去，此次代商朝领军，与周军大战。
<br/>　　一番大战，梅山七兄弟被杨戬、哪咤等人所俘，之后三鼓通响，袁洪将于辕门斩首示众。但奇怪的是，无论如何，众仙都难以杀死这袁洪，只能将其逼出原形，乃是一只白猿。
<br/>　　最终，还是申公豹以绝望之刃一刀斩首，灭杀袁洪，至此，天地间少了一只四大灵猴之一的通臂猿猴。其余梅山六兄弟，却是被杨戬收服。
<br/>　　没了这等拦路之人，各路诸侯大军直逼商朝都成——朝歌！
<br/>　　然而，到了朝歌，众人却又面临新的问题。
