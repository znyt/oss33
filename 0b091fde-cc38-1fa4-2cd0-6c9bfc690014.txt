
<br/>　　<br/>　　    陈三岁出来的时候，说话虽然还是那幅揍性，但其他人看他的眼神都略微有些变化了，哪怕王惊蛰也是的，从这一刻起，也许那位推油青年就该谢幕了，迎来的将是苯教这一代的大圆满传承。<br/>　　<br/>　　    陈三岁出来后就跟黄九郎交代了下，说是二战纳粹的那批珍宝确实在天坑里，他说的声音不大不小，三井美奈也听到了，但心里却一片苦涩，搞了半天她拿着半本笔记其实是在给王惊蛰，陈三岁和黄九郎做嫁衣了。<br/>　　<br/>　　    黄九郎得到准确的消息，神情放松的吐了口气，这番运作总算是没有白费，就询问了下那批珍宝在哪，陈三岁朝着獒王勾了勾手指，没想到那条狗子很亲昵的就走了过来，用巨大的，仿佛狮子头一样的狗脑袋在他的腿上蹭了蹭。<br/>　　<br/>　　    陈三岁拍了拍他的头，说道：“小鬼你虽然伤势未愈，但也辛苦一下，领他们过去吧”<br/>　　<br/>　　    “mmp”王惊蛰一顿腻歪，直翻白眼。<br/>　　<br/>　　    黄九郎郑重的朝着陈三岁拱了拱手，说道：“稍后忙完，我会让宗教局的人来一趟藏区，为您登记落字”<br/>　　<br/>　　    藏区的生根活佛，法王还有上师地位极其尊崇，这都是要在宗教局备案的，以后就是出了藏区在外袒露身份的话，官方也是要正经对待的。<br/>　　<br/>　　    王惊蛰又翻了翻白眼，这厮的便宜是跟着自己捡的，但是真他么呵呵了，黄九郎把人情又算在了陈三岁的身上。<br/>　　<br/>　　    黄九郎走了，让鲁靖尧将三井美奈也带走了，神庙前就剩下了王惊蛰和陈三岁，两人坐在石阶上，仰视夜空。<br/>　　<br/>　　    陈三岁先叹了口气，搓了搓脸蛋子，说道：“真是世事难料，我是没想到，自己居然还会有这么一天，活佛？以前我都骂过和尚是秃驴”<br/>　　<br/>　　    王惊蛰很诧异的问道：“你到底是转世的大圆满大师，还是说你具备了承接大圆满的条件？”<br/>　　<br/>　　    这一点王惊蛰很好奇，据他所知藏区的活佛都是转世身，苯教的大圆满法师自然也应该不列外，可要说陈三岁真是转世身的话，这可就太巧太离奇了一点，难道还真是冥冥中自有注定？<br/>　　<br/>　　    关于此点，陈三岁没有解释，只说了一声以后再说，然后他直接岔开了话题，说道：“我暂时就不走了，再有一段日子扎西降泽就该圆寂了，在这之前我要熟悉苯教的一些事物，还得要接受传承，所以很长一段时间咱们就见不上了”<br/>　　<br/>　　    “呵呵，厉害了？”王惊蛰斜了着眼睛说道。<br/>　　<br/>　　    陈三岁淡淡的说道：“哪里有？我只是感慨世事无常罢了，对于这个什么大圆满传承，说实话我没有任何的感觉，我活着不过就是活着而已，早已经没了什么奔头，哪怕是多个法师，活佛的称号，对我来说又能意味着什么？毕竟，失去的终归已经失去没办法在寻回来了”<br/>　　<br/>　　    “可能也正是因为你的这种心境，让你成为了苯教大圆满的传承……”<br/>　　<br/>　　    陈三岁和王惊蛰坐在一起聊了挺长的时间，说的无非是以后的一些事，陈三岁告诉他自己将会留在这里一段时间，之后就会出天坑了，按照每一代大圆满传承者的规矩，他也是需要在外面走一走的，王惊蛰就告诉他好好学习天天向上吧，以后还有用得着的一天呢。<br/>　　<br/>　　    聊到后来，没什么说的，自然就告辞了。<br/>　　<br/>　　    “拜拜！”<br/>　　<br/>　　    “再会！”<br/>　　<br/>　　    王惊蛰走向通道准备出天坑，陈三岁在他身后，忽然说了一句：“以后，请叫我三岁上师……”<br/>　　<br/>　　    王惊蛰脚步一顿，嘴角抽搐了两下，心中感叹其实还是那个推油的家伙比较招人喜欢。<br/>　　<br/>　　    王惊蛰从天坑里出来后，黄九郎他们已经在外等候了，天坑下面那批珍宝的事情落实了，却没办法这么轻易的就给捞出来，然后就会有部队的人过来接管。<br/>　　<br/>　　    一天后，他们离开附近，来到了山脉地势平坦的地方，被一架直升机给接走了。<br/>　　<br/>　　    飞机直接飞往拉sa，黄九郎问他要不要一起回京，军用机场里有一架飞机待命，可以送他回去。<br/>　　<br/>　　    王惊蛰摇头拒绝了，说自己打算在这里修整一段时间，静静心再说，反正没什么事，不至于这么着急走。<br/>　　<br/>　　    来到藏区有两个月的时间了，他一直都在奔波着寻找龙骨草，说实话他还没有体验过藏区的神圣呢，之前在拉sa也不过就停留了一天而已。<br/>　　<br/>　　    在一家旅馆里，王惊蛰好好睡了一大觉，整整休息了两天他才算缓过这阵的疲劳来。<br/>　　<br/>　　    此时，季节已经到了六月份后，藏区的这个时节天气还是不错的。<br/>　　<br/>　　    这里似乎离云彩和蓝天很近，仿佛伸手就能够得到，王惊蛰宛若一个平常的游人散布在布达拉宫前的广场上。<br/>　　<br/>　　    两月前来，他们三人在车里匆匆一瞥，没有进过布达拉宫，这次没什么事他得进去看看了，确实如孔良所说的，这座藏区最神圣，大气磅礴的宫殿里还真的是珍宝如云，价值连城，看的让人眼花缭乱的，不过在王惊蛰看来宫中多了人气，神气却少了不少。<br/>　　<br/>　　    “想来，大昭寺也是如此吧”王惊蛰有些遗憾的摇了摇头。<br/>　　<br/>　　    从布达拉宫出来后，王惊蛰就去了大昭寺看看，据说这里是藏人心中的圣地。<br/>　　<br/>　　    “来都来了……”这就是王惊蛰的念头。<br/>　　<br/>　　    大昭寺又名祖妈庙，在拉sa的老城区，这座1300多年历史的寺庙在藏传佛教中拥有至高无上的地位，其实藏区的活佛成百上千，有真也有冠名的，但真正的活佛，受藏区人认可，国家承认的，名望最大的其实没有几个。<br/>　　<br/>　　    而据说，每年都有活佛在大昭寺为人赐福，这位活佛基本就是藏区的No.1。<br/>　　<br/>　　    大昭寺广场前人很多，黑压压的一群人，八成左右都是各地来旅游的，只有一少部分是藏区的民众。
