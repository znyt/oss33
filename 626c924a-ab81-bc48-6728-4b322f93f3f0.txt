        最终，他傅炎廷才是“鹬蚌相争，渔翁得利”中的渔翁。
<br/>
<br/>        傅炎廷知道，自己的长孙傅夜沉喜欢的人是童筱颖，所以，他不能让童筱颖成为傅夜沉的软肋。
<br/>
<br/>        童筱颖和傅昼景留下来，迟早会是傅夜沉的绊脚石。
<br/>
<br/>        傅夜沉可以对恶徒心狠手辣，却无法对自己身边的亲人下狠手。
<br/>
<br/>        傅炎廷了解自己的长孙，所以他的所作所为，只是在为自己的长孙清除所有障碍。
<br/>
<br/>        然后，让傅夜沉安安稳稳、妥妥当当地接手他的“霸业”。
<br/>
<br/>        傅昼景对爷爷傅炎廷的心思，只猜对了一半。
<br/>
<br/>        爷爷傅炎廷果然忌讳他会“吞了”大哥傅夜沉的傅氏集团。
<br/>
<br/>        如果，真的离开了傅氏集团，从此以后，他和筱颖会相依为命。
<br/>
<br/>        其实，筱颖那个时候说得对，不属于他的东西，他果然不该去强求。
<br/>
<br/>        大哥傅夜沉是天之骄子，而他……
<br/>
<br/>        傅昼景苦笑。
<br/>
<br/>        无论他怎么做，都改不了爷爷心中嫡庶有别的观念。
<br/>
<br/>        另一边，苏漫雪正洋洋得意地在ATM机前从童筱颖的银行卡里取出季思妍给她的那些钱时，突然被两名便衣警察给“逮住”。
<br/>
<br/>        苏漫雪被两名便衣警察先送去了派出所辗转又送去了看守所，就连那些莫须有的罪名，都是让她哑巴吃黄连，有苦说不出。
<br/>
<br/>        而苏漫雪被抓，最得意的虽然是季思妍，但是傅昼景的反应过于平静，反倒让季思妍疑惑不解。
<br/>
<br/>        这件事情，明明已经是波涛浪涌的声势，但结果却过于异常的风平浪静，甚至一点风声都没有走漏。
<br/>
<br/>        半个月后……
<br/>
<br/>        傅氏集团大厦顶楼，总裁办。
<br/>
<br/>        傅昼景手里拿着辞呈，推门而进，他俊脸上挂着难得的笑容，似乎心情不错。
<br/>
<br/>        而傅夜沉正靠坐在皮椅上，手肘抵在檀木扶手上，手背撑着脸颊，淡定自若。午后的阳光，从落地窗外射进来，映在他五官上，俊逸出尘，让人看不透。
<br/>
<br/>        傅昼景走近一看，才发现，大哥傅夜沉正在召开跨国会议。
<br/>
<br/>        笔记本电脑里的画面是傅氏集团在M国的分部会议室，里面坐着上那边的各位大佬。
<br/>
<br/>        他知道大哥虽然曾经在部队里当兵，但他在当兵期间，因为特种部队的需求，他大哥修了五国语言。
<br/>
<br/>        但是，直到今天，他真正见识大哥的语言本领。
<br/>
<br/>        傅夜沉开完会后，见傅昼景能安然无恙地出现在自己面前，就说明，傅昼景身上的那件事情已经解除了。
<br/>
<br/>        爷爷终究还是保了阿景？！这一点都不像爷爷平日里的作风！
<br/>
<br/>        傅昼景见大哥傅夜沉的会议已经结束了，便向大哥傅夜沉递交了自己的辞呈。
<br/>
<br/>        傅夜沉看到傅昼景递过来的辞呈时，一脸茫然。
<br/>
<br/>        傅昼景释然一笑：“大哥，谢谢你，一直无条件地相信我，帮着我。”
<br/>
<br/>        “你这是？”傅夜沉欲言又止。
<br/>
<br/>        傅昼景随之笑容一止，微笑瞬间收敛，深沉地说道：“我是来跟大哥告别的。爷爷说，筱颖学习成绩优异，决定送筱颖出国留学，而我也随筱颖一起去澳洲。”
<br/>
<br/>        “你要跟筱颖一起走？！”傅夜沉眉眼微冷，心口莫名其妙地隐隐作痛。
<br/>
<br/>        傅昼景点了点头，接着回答道：“嗯，爷爷已经帮我和筱颖把那边都安排好了，我们今天就得离开。”
<br/>
<br/>        “好。”傅夜沉接过傅昼景手中的辞呈，薄唇微启，嘴角勉强扯出一抹淡淡的笑意。
<br/>
<br/>        傅昼景临走前，忽然问道：“大哥，我妈妈，到底是怎么死的的？”
<br/>
<br/>        傅夜沉身心一怔，完全没料到傅昼景会突然提及此事情。
<br/>
<br/>        沉默了片刻，傅夜沉才压低了嗓音，开口道：“殉情。”
<br/>
<br/>        傅昼景不再吭声，黯然神伤地转过身去。
<br/>
<br/>        童筱颖被关在依山别苑里的这些天，也终于是守得云开见月明。
<br/>
<br/>        同时，她也终于明白，自己为什么每天晚上都睡得那么安稳了，原因是刘管家在她的饮用水里下了安眠药。
<br/>
<br/>        童筱颖被老老实实地“囚禁”了半个月之久，当刘管家告知她，她可以离开的时候，她还一脸狐疑地看着刘管家。
<br/>
<br/>        此时，刘管家递给童筱颖一部崭新的手机，说道：“二少奶奶，这是傅老太爷送给您的新手机，密码是您的生日。手机里有支付卡，里面有钱。待会，您还能见到二少爷。”
<br/>
<br/>        “你……”童筱颖难以置信地看着刘管家。
<br/>
<br/>        他叫了她这么多天的“苏小姐”，怎么突然改口叫她“二少奶奶”呢？
<br/>
<br/>        童筱颖半信半疑地接过刘管家递过来的新手机，立马拿着手机给傅昼景打了通电话。
