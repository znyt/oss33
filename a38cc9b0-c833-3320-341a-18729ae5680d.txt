总裁办公室。
<br/>
<br/>        这是夏夕绾第一次来陆寒霆的总裁办公室，她站在落地窗前向外俯瞰，好像这座四九城都踩在了她的脚下，这栋商业大楼不愧是海城财富的不二象征。
<br/>
<br/>        这时“叮”一声，她的微信响了。
<br/>
<br/>        打开手机一看，是双双发来的微信。
<br/>
<br/>        双双直接甩了一张照片过来，配文oh，mygod，绾绾，这是谁啊，你跟…陆氏总裁陆寒霆在一起了？
<br/>
<br/>        夏夕绾点开照片，这张照片是刚才在陆氏大厅里，陆寒霆将她托着抱起的画面。
<br/>
<br/>        夏夕绾瞳仁一缩，迅速回了一个双双，这张照片哪里来的？
<br/>
<br/>        此时的双双就在陆氏集团下面，刚才她路过这里就看到外面好大的阵仗，她再往里面一看，简直惊呆了，她当即拿出手机将这一幕给拍了下来。
<br/>
<br/>        本来她还想进陆氏的，但是很不幸，她被赶了出来。
<br/>
<br/>        绾绾，我刚才就在外面，我亲眼看到了，究竟怎么回事，你给幽兰苑那个鬼夫…戴绿帽子了？
<br/>
<br/>        夏夕绾有点头疼，如果她告诉双双陆寒霆就是幽兰苑那个鬼夫，双双会不会吓懵了？
<br/>
<br/>        双双，这件事一两句话说不清，等我明天去研究院再跟你说。
<br/>
<br/>        虽然现在双双好想八卦，但是看到夏夕绾的回复，她只能先忍一忍，靠，绾绾跟海城第一豪门陆寒霆…这想一想就激情四射啊。
<br/>
<br/>        双双将手机放回了包里，准备回家。
<br/>
<br/>        但是这时一辆豪车停了下来，驾驶座上的人是赵立英，“双双，你怎么在陆氏集团这里？”
<br/>
<br/>        赵立英家境很不错的，开了一辆一百多万的豪车，双双跟他有婚约，但是两个人前段时间已经闹掰了。
<br/>
<br/>        双双自然不会将刚才的事情告诉赵立英的，这个赵立英现在已经被夏妍妍给迷晕了，如果让他知道绾绾给幽兰苑那个鬼夫戴绿帽子了，肯定又要大做文章了，她要保护绾绾的。
<br/>
<br/>        “我就是路过这里，正准备回家，赵立英，我现在跟你没有关系了，请你不要跟我说话，免得让别人看见了又丢你的脸！”双双抬脚就走。
<br/>
<br/>        赵立英迅速道，“双双，我们解除婚约的事情也要通知一下家长吧，这样吧，你上车，我送你回家，正好我跟你爸妈说一声。”
<br/>
<br/>        双双想了一下觉得也可以，跟赵立英彻底的了断，于是她拉开车门上了车，“那走吧。”
<br/>
<br/>        ……
<br/>
<br/>        赵立英一直在开车，双双慢慢的就发现了不对劲，“赵立英，你要带我去哪里，这不是我回家的路，你快停车！”
<br/>
<br/>        赵立英将车停了下来，然后将双双拽了出来，他一把抢走了双双的手机。
<br/>
<br/>        打开了微信，他看到了双双刚才拍的那张照片，陆寒霆抱着夏夕绾的照片。
<br/>
<br/>        赵立英得逞的勾了一下唇，然后将这张照片发给了夏妍妍。
<br/>
<br/>        “赵立英，你干什么，快点将手机还给我！”
<br/>
<br/>        赵立英伸手将双双一推，双双直接摔在了地上，他露出了厌恶狰狞的面孔，“双双，这里可是高速公路，我将你丢在这里你自己跑回家吧，也就跑上八个小时，权当减肥了，我从来就没有喜欢过你，而且每次我看到你一身的肉都觉得呕心！”
