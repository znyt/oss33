　　武英，可是以纯粹的武道，便是足以在北域百绝榜上，位列前十的人物。
<br/>　　这种武者，才是最为可怕的，武道一途，苦闷无比，大部分的武者，都不会纯粹的修炼武道，而是也会学习其他武技功法。
<br/>　　但是真正若是在武道一途上，修炼到了大成，那此人的实力，可就真的恐怖无比了。
<br/>　　因为纯粹的武道，根本没有任何弱点，这武英，正是一位纯粹的武修。
<br/>　　除非是真正以实力，才有可能胜过他。
<br/>　　只要是实力不如武英的，就算是只差一重境界，都不可能省的了他。
<br/>　　楚风眠的力量，的确强大，足以是越级战斗，拥有无数奇遇，但是在力量的比拼上，却是没有任何人会看好楚风眠。
<br/>　　“轰隆！”
<br/>　　两拳之威，在半空之中猛然碰撞，肆意的拳风，纯粹的力量，在其中疯狂交织这。
<br/>　　下方无数众人，就算是在这擂台之外，也足以感觉得到，这擂台之中爆发出来的威力，到底是强大到了哪一种程度。
<br/>　　无数眼神，都在盯着其中的一举一动，在这两拳相撞的一刻，两人的身形，也都被这无数灵力，所遮掩。
<br/>　　令人看不清其中，到底是发生了什么。
<br/>　　“碰！”
<br/>　　就在这下一刻，一道身影，却是突然之间，从之中被打飞出去，足足是打退了数十步，才稳定住了身形。
<br/>　　在场众人的视线，都随着看了过去，便是看清楚了此人的模样，一个个的眼神惊骇无比，这被打飞的人影，居然是武英。
<br/>　　在力量的比拼上，武英居然是输了。
<br/>　　灵力散去，楚风眠站在原地，看起来是轻松无比，但是那武英的身上，却是后退了几步。
<br/>　　这结果，不言而喻。
<br/>　　“不错的实力，在力量上，能够将我逼退的，还没有几个人。”
<br/>　　武英后退几步，脸色一下子变的认真了许多。
<br/>　　他并非是以力量为主的武者，但是在力量上，却也是足以超越同境界武者了。
<br/>　　在同境界中，堪称无敌的存在，更何况是面对着跟他还相差不少境界的楚风眠。
<br/>　　“可惜，这种实力，想要在我手中活下去，基本没有任何可能。”
<br/>　　武英看着楚风眠，面露狰狞的说道。
<br/>　　这一招之中，他已经是弄清楚了楚风眠的力量。
<br/>　　“今天，我就当着圣主的面，轰杀了你，也让其他人都看看，敢来与我们圣党为敌的下场！”
<br/>　　武英言罢，身上的武道便是再一次的迸发出来。
<br/>　　旋即，这武道的力量，再一次的凝聚到了武英的拳风上。
<br/>　　武英对自己的武道，极为自信，他跟楚风眠的交手，便是已经将楚风眠的力量，给探察的一清二楚。
<br/>　　楚风眠的实力，很强，可是武英，也没有动用全力。
<br/>　　“小子，你的资质的确很强，甚至是要超越了我，不过可惜，过刚易折，居然是选择跟我们圣党为敌，是你一生中最愚蠢的事，今日你这个天才，就由我武英来扼杀吧！”
<br/>　　武英狰狞一笑。
<br/>　　他的双拳猛然冲着楚风眠轰击过来。
<br/>　　这一拳，可不再是试探，而是真正蕴含杀意的一拳。
<br/>　　“轰隆隆！”
<br/>　　一拳打出，整个擂台上，都是震天雷声，这一道拳风，居然都分化万千，甚至是一股股无上威压，都在笼罩着楚风眠。
<br/>　　这一拳，便是将周围的擂台，都转化为了武道的笼罩之下。
<br/>　　其中的一举一动，都笼罩着武英的武道之中。
<br/>　　在这武道的笼罩之下，楚风眠的一举一动，似乎都受到了极大的影响。
<br/>　　同时这周围的无数拳风，已经是四面八方之间，向着楚风眠轰击而来，似乎是让楚风眠，根本便是无路可逃了。
<br/>　　“这是武英的武道！武英的武道，已经是足以化为威压，被笼罩其中，至少需要以三成的力量来抗衡！”
<br/>　　“糟了，这小子的力量在强大，失去三成力量，又怎么可能还足以跟武英抗衡！”
<br/>　　“此人的境界，还是太低了，若是此人度过了生死大劫，只怕是还是有着跟武英一战的资本的，可是现在，他的一切奇遇，都要为其他人来做嫁衣了。”
<br/>　　下方许多武者，都是一眼看出，其中的局面。
<br/>　　这武英近乎同境界无敌的原因，也正是他的武道，除非是实力比起他强大，不然都会受到他的武道的影响。
<br/>　　失去三成力量。
<br/>　　在这种级别的战斗之中，失去三成的力量，已经是不败无疑了。
<br/>　　更何况武英，根本便是没有丝毫给楚风眠任何机会，武道笼罩下去的一刻，他的拳风也是接连而至。
<br/>　　显然是不给楚风眠任何的喘息机会，要一招之内，彻底的轰杀了楚风眠。
<br/>　　“这种手段，也想要轰杀了我？我已经给过你一次出手的机会了，现在你也算是死而无憾了。”
<br/>　　楚风眠看着武英的无数拳风落下，就在轰击到楚风眠身边的一刻。
<br/>　　就在楚风眠的身躯周围，突然无数金色灵力，显现出来，这金色灵力，开始凝聚，顿时化为了一道道的鳞片龙鳞。
<br/>　　同时就在楚风眠的口中，也是猛然咆哮一声。
<br/>　　一声声龙吟，响彻而起。
<br/>　　这周围笼罩在楚风眠身上的武道，被直接撕裂，只见楚风眠大手一挥，这无数拳风，便是都被瞬间打碎。
<br/>　　“战龙舞苍穹！”
<br/>　　楚风眠化身远古战龙，猛然之间冲天而上。
<br/>　　巨大无比的身躯，猛然冲着武英撞了过去。
<br/>　　“啊！”
<br/>　　一声惨叫。
<br/>　　这武英拼劲全力，想要抵挡，但是在楚风眠的面前，却是无力到了极点。
<br/>　　几乎是被瞬间，他的身躯，都被直接撞碎，几乎就只剩下了一团肉泥。
<br/>　　就在这擂台之上，上一刻几乎还是无敌的武英，瞬息之间，居然被打成了一团肉泥，这让在场众人，几乎都是面露呆滞了起来。
<br/>　　全场一片寂静。
<br/>　　北域百绝榜的第九位武英，如今就在楚风眠这一招之下，化为了一滩肉泥？
<br/>　　本来楚风眠所说的让武英出手，是给他一次机会。
<br/>　　这句话，在无数武者的心中，本来是嗤之以鼻，以为楚风眠是在说大话。
<br/>　　可是现在看来，楚风眠的话，没有任何的夸张。
