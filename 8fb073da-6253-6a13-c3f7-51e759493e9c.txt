　　之前仅仅是对战三只体修傀儡，萧逸自是游刃有余。
<br/>　　或躲或进，轻松无比。
<br/>　　可现今，傀儡达到数十，再加上这无数妖兽，又要护着众人前行。
<br/>　　以他现今剑道实力，显然极强勉强，甚至是难以做到。
<br/>　　不过，他未说让众人放弃，反倒是说‘压过去’。
<br/>　　众人先是一愣，但，很快便明白了过来。
<br/>　　萧逸的剑，已然变了。
<br/>　　原本，仅是剑影重重，剑光飞舞。
<br/>　　此刻，剑影收敛，齐聚剑身；剑光，亦忽然开始变得单一，简单。
<br/>　　对比之前的玄妙剑法，让人看得眼花缭乱。
<br/>　　此刻，萧逸的剑，变为了简单的一劈，一斩。
<br/>　　轰…轰…轰…
<br/>　　嘭…嘭…嘭…
<br/>　　每一剑劈下，必是一声轰鸣；每一剑斩下，必是一声爆响。
<br/>　　而每一声轰鸣之下，亦必是一只体修傀儡被轰飞。
<br/>　　每一声爆响之下，必是一头绝世妖兽那庞然之躯被斩飞。
<br/>　　“破晓无极，破晓无极，破晓无极…”
<br/>　　萧逸心头连连低喝。
<br/>　　对，现今的他，正毫不间断地使用着破晓无极。
<br/>　　破晓无极，毫无疑问是极强的手段，亦是他现今的最强剑道手段。
<br/>　　只是，这一手段还未大成。
<br/>　　而且，如果能将这一手段，成为自己真正的剑式呢？
<br/>　　萧逸想试试，如果破晓无极，能一直萦绕剑身，会达到何等效果。
<br/>　　现在这个层次的压力，足以他剑道实力尽出，且异常勉强，甚至动辄有生命之虞了。
<br/>　　而在自己的控火一道实力，也就是自己如今的真正实力面前，这些压力，算不得什么。
<br/>　　也就是说，他可以肆无忌惮地压榨自己的剑道极限。
<br/>　　真有万一或者意外，他的实力足以碾压这里一切危险，故并不需担心。
<br/>　　“好猛。”
<br/>　　“好强。”
<br/>　　而这时，顾怜星、公孙火舞二人，已满脸惊色。
<br/>　　“好…好变态。”十数天骄，已然明白萧逸话中‘压过去’的意思。
<br/>　　在他们眼中，此刻，之前那些强大无比的体修傀儡与绝世妖兽，正不断地被轰飞。
<br/>　　而这些体修傀儡与绝世妖兽，本就身躯坚硬强大无比，如若两个铁疙瘩。
<br/>　　每一只的轰飞，都会如若一块巨石，反将它们身后的妖兽连连撞飞。
<br/>　　与其说萧逸现今的剑在劈斩，都不如说是在拍。
<br/>　　一只只妖兽、傀儡，被不断拍飞，沿途撞飞一连串妖兽。
<br/>　　此刻的萧逸，简直是在这条炼药通道上碾压着，一路开路。
<br/>　　轰…轰…轰…
<br/>　　萧逸一路劈剑，不过多久，已然带着众人前行了三分之一左右的距离。
<br/>　　萧逸一直在尝试着，在极限压榨着自己的剑道水准。
<br/>　　破晓无极这一手段，从出到收，间隔愈发小，也愈发延绵。
<br/>　　正常来说，他的破晓无极，足以将这些体修傀儡和绝世妖兽分尸。
<br/>　　但，他每每出剑，剑力只出9成，而剑身留有一成。
<br/>　　故，这些傀儡与妖兽，仅仅是被重重轰飞。
<br/>　　剑身上的一成剑力，便是他留着牵引破晓无极瞬间凝聚的手段。
<br/>　　此刻的他，不止在磨砺剑道水准，也是在磨砺破晓无极这一手段。
<br/>　　若是能由收获，再提升威力，自然是最好的。
<br/>　　另外，他也想试试，在进主局之前，能否让无上剑诀的剑气，与破晓无极完美契合。
<br/>　　这，才是他现今的目的。
<br/>　　……
<br/>　　时间，渐渐过去。
<br/>　　自萧逸开始将这些傀儡与妖兽拍飞开始，这条炼药通道，便已然再无法阻拦一行人的脚步。
<br/>　　仅仅是大半个时辰后，众人便已然来到了尽头边缘。
<br/>　　“呼。”萧逸重重地喘了口粗气。
<br/>　　“你们先出通道。”
<br/>　　“好。”众人满脸喜色。
<br/>　　15条通道，长度一样，尽头的位置，也是一样。
<br/>　　而15条通道的尽头，已然不是互相独立，而是完全接连融合。
<br/>　　也就是说，通道的尽头之外，便是各支队伍的再次汇合之地。
<br/>　　嗖…嗖…嗖…
<br/>　　众人连忙闪身出了炼药通道。
<br/>　　“破。”萧逸一声暴喝，一剑劈出，轰退了左右各方妖兽与傀儡阻拦。
<br/>　　身影一闪，同样出了通道。
<br/>　　吼…
<br/>　　一声声妖兽怒吼，充斥着愤怒以及嗜血。
<br/>　　可它们，显然都受古帝洞府之内的禁制操控。
<br/>　　故它们根本出不得这条通道。
<br/>　　一只只狰狞的妖兽，冷眼盯着通道之外的众人。
<br/>　　被如此多的凶兽虎视眈眈地盯着，众人不禁打了个冷颤。
<br/>　　而萧逸，则懒得理会，连忙盘膝坐下。
<br/>　　脸庞之上，此刻微微有些苍白；额头上，则已然汗水密布。
<br/>　　显然，将众人悉数安全护到这里，并不轻松。
<br/>　　一路碾压而过，萧逸的破晓无极，从未间断，他几乎消耗了自己体内过半的元力。
<br/>　　以他那庞大的小世界和气泉，元力耗费过半，可想而知那是何等庞大的概念。
<br/>　　“小贼，你没事吧？”公孙火舞紧张地问了一声。
<br/>　　萧逸摇了摇头，“无碍，只是元力耗费太大，歇息一会儿即可。”
<br/>　　萧逸边说着，边吞服下大堆丹药。
<br/>　　……
<br/>　　时间，渐渐过去。
<br/>　　炼药通道这边，萧逸一行人已出了通道。
<br/>　　但其余通道上，仍旧鏖战不断。
<br/>　　黑云学教队伍这边，莫悠一路势如破竹，轻松无比。
<br/>　　而青麟，则包揽了周遭其余较弱的妖兽与傀儡，护着学教队伍一行人。
<br/>　　两人联手下，虽比不上萧逸那碾压一般的速度，但也算得上是一路畅行。
<br/>　　不过半个时辰多些，一行人，亦成功出了通道。
<br/>　　各条通道之外，已然非是独立之处。
<br/>　　“萧逸师弟。”青麟一眼看到了盘膝坐下，脸色苍白的萧逸，连忙闪身至身旁。
<br/>　　“你可有大碍？”
<br/>　　青麟注意到萧逸胸膛上的数处狰狞伤口，又连忙问了一声。
<br/>　　“你说呢？”萧逸睁开眼睛，淡淡一笑。
<br/>　　“我的本事如何，你还不知道？不过是些皮外伤罢了。”
<br/>　　“嗯？”萧逸打量了青麟一眼，“浑身毫发无损，倒是不错。”
<br/>　　萧逸起身，拍了拍青麟的肩膀，笑道，“许久不见，果然大有长进。”
<br/>　　“那是。”青麟得意一笑。
<br/>　　“切。”恰在此时，一旁公孙火舞撇撇嘴，脸色不善。
<br/>　　“我没看错的话，刚才莫悠一路纵横，剑上白芒涌动，那就是传说中的剑帝本源吧。”
<br/>　　“难怪他如此强。”
<br/>　　“有他护着，你黑云学教一行人，自是安然无恙。”
<br/>　　气氛，霎时变得沉重。
<br/>　　青麟，脸色复杂，沉默了起来。
<br/>　　萧逸脸色淡然，“那都是小事，火舞姑娘便不要再提了。”
