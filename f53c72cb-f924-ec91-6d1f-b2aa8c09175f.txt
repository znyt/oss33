　　话语刚落，萧炎望着手中的碎片怔怔出神，他们萧族以前居然是大陆上最强的家族，那为何现在如此羸弱不堪？
<br/>　　既然是大陆上最强的家族，为何连加玛帝国一个小小的云岚宗都要惧怕？
<br/>　　“慕师兄的意思是说，用成为斗尊的资源换取我手中，陀舍古帝玉的碎片。”
<br/>　　“对，现在你清楚这块玉的来历和重要性，也知道了很多的秘辛，换不换由你决定。”
<br/>　　说罢，慕清霄语气变得严肃起来。
<br/>　　“我不会抢夺你手中的碎玉，但也不会插手各族的事情，如果有一日，你保不住手中的碎玉，我会出手拿走。”
<br/>　　萧炎咬牙道：“难道远古八族很强吗？并且都想要陀舍古帝玉碎片吗？”
<br/>　　“这个问题，老师比较有话语权。”
<br/>　　见萧炎目光落在自己身上，药尘叹息一声。
<br/>　　“为师出身就是八族其一药族的成员，远古八族除了已经衰落的萧族，皆是大陆上的庞然大物，一个比一个神秘，熏儿小丫头是古族的小公主，要论实力，八族中古族可排第二，强如斗尊，在他们眼中也是蝼蚁。”
<br/>　　想到此处，药尘凝重道：“如果让其他种族知道陀舍古帝玉的下落，绝对会不惜一切代价抢夺，这块碎玉极其烫手……”
<br/>　　其实，有一点药尘没有说明，原本他还不清楚，古薰儿为何会出现在萧炎身边，现在倒是有些明白了。
<br/>　　萧炎苦笑一声，道：“斗圣……慕师兄你是为陀舍古帝玉而来，也是远古八族中的一员吗？”
<br/>　　微微摇头，慕清霄道：“丹道上，你我师出同门，没有必要骗你，我并非远古八族任何一族，只是陀舍古帝洞府中有我想要的东西。”
<br/>　　萧炎不傻，他清楚自己的处境，若是陀舍古帝玉的信息传出去，他将被各族盯上。
<br/>　　“以慕师兄现在的实力，没有陀舍古帝玉，也无法进入洞府吗？”
<br/>　　“呵……破开斗帝的结界也并非不可能，只是我担心其中许多东西会被毁而已，那些可都是重要的资源。”
<br/>　　慕清霄也不觉得斗帝有多强，渡劫九重的圣姑他都能伤，别说区区一个斗帝洞府，但斗帝洞府是他突破渡劫的契机，用陀舍古帝玉进入其中，更加稳妥一些。
<br/>　　药尘飘在空中，并没有插足两名弟子的谈话。
<br/>　　今日知晓如此多的秘辛，连他心中都有些忐忑，与体制内一样的道理，不身在其中，根本不知道其中何其的危险。
<br/>　　稍有不慎，便是魂飞魄散。
<br/>　　其实，他打心底里不希望萧炎将如此危险的东西放在身上，更不希望牵扯到远古八族。
<br/>　　况且，现在的萧炎也没有保住陀舍古帝玉的能力。
<br/>　　此刻，清风送爽，但悬崖上的气氛却极其的压抑。
<br/>　　良久，萧炎洒脱一笑，将碎玉扔给了慕清霄。
<br/>　　“现在的我没有能力管什么斗帝洞府，我只想静心修炼，现在加玛帝国的萧家并非萧族，我更不想因此事，牵扯到我父亲一手建立的家族。”
<br/>　　“你倒是洒脱，选择很正确。”
<br/>　　接过碎玉，慕清霄也没有关注，直接取出一枚纳戒丢给了萧炎。
<br/>　　“其中，一阶到八阶灵药有数千株，一品到八品丹药有千颗，有老师辅助你，斗尊只是时间问题，用兽火进化焚决也是不错的想法。”
<br/>　　将纳戒戴在手中，查看了一下其中的东西，萧炎露出一抹笑意。
<br/>　　就在这时，慕清霄转身道：“偷听了这么久，是想让我请你们出来吗？”
<br/>　　话语刚落，药尘一惊，直接钻进黑色纳戒中，萧炎目光一凝，顺着慕清霄的视线望去。
<br/>　　只见悬崖后面，森林前面的空间一阵扭曲，旋即三道身影就凭空而现。
<br/>　　来者是两名身穿黑袍的老者，站在中央的是一名身穿迦南学院外院服饰的女子。
<br/>　　“熏儿……”
<br/>　　古薰儿目光温柔的看了一眼萧炎，道：“萧炎哥哥，此人非常危险，快来熏儿这里。”
<br/>　　萧炎瞥了一眼慕清霄，看向古薰儿的目光充满复杂之色。
<br/>　　“熏儿，你究竟瞒了我多少，难道接近萧家也是为了陀舍古帝玉的碎片？”
<br/>　　闻言，古薰儿脸色一白，不断摇头道：“萧炎哥哥，相信我，熏儿不可能对你有恶意的，此人接近萧家才是别有用心。”
<br/>　　“慕师兄想要夺走陀舍古帝玉的碎片，根本不需要如此大费周章，我自然知道熏儿你对我不可能有恶意，但古族不是每个人都如你一样。”
<br/>　　闻言，古薰儿目光不善的盯着慕清霄，萧炎对她的误解，都是因为面前这个面带人畜无害笑容的青年。
<br/>　　“你说自己不是远古八族，但你能撕开空间裂缝，进出自如，这明明就是太虚古龙族的血脉力量，你说自己能破开斗帝洞府，那你要陀舍古帝玉有何用？”
<br/>　　慕清霄脸上的笑容渐渐收敛。
<br/>　　“谁说能撕开裂缝的就一定是太虚古龙族，坐井观天，只是没想到你们古族的消息如此快，两名九星斗尊，是想抢陀舍古帝玉吗？”
<br/>　　慕清霄只是稍稍瞥了一眼两名老者，便能看出两人都是九星斗尊巅峰，如果联手足以对付半圣强者。
<br/>　　“小子，你竟敢对熏儿小姐无礼，听说太虚古龙族实力强横无比，老夫今日倒是要看看，你究竟有什么能耐。”
<br/>　　话语间，已经有一名黑袍老者上前一步。
<br/>　　慕清霄有些无语，自己施展的不过是《空间门》，没想到会被误认为是太虚古龙族，视线不由的看向萧炎。
<br/>　　“小子，知道自己的处境了吧，强者为尊，没有实力不会有人在意你，我不过出现短短几个时辰，古族就派两名九星斗尊来此，或许跟那小丫头没关系，但她的父亲可是古族族长。”
<br/>　　闻言，萧炎眼中闪过一抹苦涩，古族好大的手笔，一出手就是两名九星斗尊。
<br/>　　收回目光，慕清霄冷笑一声，道：“古族就派你们两个老东西来夺取陀舍古帝玉碎片，是不是太看不起我了？”
<br/>　　话语刚落，两名黑袍老者皆是上前，恐怖的气息席卷而出。
