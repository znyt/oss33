　　其余长老，在与萧逸寒碜了一番后，相继离去。
<br/>　　偌大个议事厅，只剩萧逸与大长老。
<br/>　　大长老率先开口道，“萧逸，听你刚才之言，是你萧家遇袭了？”
<br/>　　“嗯。”萧逸点了点头，道，“我来剑派，就是想看看这里有没有线索。”
<br/>　　大长老疑惑道，“我记得你以前与我说过，你萧家有猎妖令庇护。”
<br/>　　“什么人敢对付你萧家？”
<br/>　　萧逸沉声说道，“全是来自北山郡本土，以及周边几郡的亡命之徒。”
<br/>　　“亡命之徒？”大长老眉头一皱，道，“看来，是背后有人要对付你萧家。”
<br/>　　“这些亡命之徒，只是受雇佣的。”
<br/>　　大长老可是个人精，自然猜得出这些。
<br/>　　“可查到些什么了？”
<br/>　　大长老问道。
<br/>　　萧逸点点头，道，“大概能猜到些。”
<br/>　　“与我有仇的，也就那么几个势力。”
<br/>　　事实上，自从见到四长老出现在剑派，萧逸便已经能确定是哪个势力在对付他萧家了。
<br/>　　无非就是血雾谷。
<br/>　　以萧逸的心智，甚至已经大概将事情猜得八九不离十。
<br/>　　血雾谷想对付他，但不敢明着来，只得先对付他的家族。
<br/>　　而萧家有猎妖令庇护，血雾谷同样不敢直接对付。
<br/>　　雇佣亡命之徒出手，是他们唯一的办法。
<br/>　　当然，在这个办法之下，还有着一连串天衣无缝的计划。
<br/>　　先是让这些亡命之徒，出手对付萧家。
<br/>　　另一边，又遣人困住紫云城外的猎妖殿执法队。
<br/>　　这些亡命之徒在袭击萧家时，因没有执法队出来阻挠，便会不知道萧家有猎妖令庇护。
<br/>　　那些亡命之徒，以及一般的杀手势力。
<br/>　　自然是不可能知道哪个家族有猎妖令庇护的。
<br/>　　甚至等闲之辈，若以前没有遇到过猎妖令，根本认不出猎妖令。
<br/>　　猎妖殿，也不会到处宣扬哪个家族有猎妖令庇护。
<br/>　　自然的，在对上某个家族或个人时，是否有猎妖殿执法队出来阻挠。
<br/>　　是判定这个家族或者个人，有否猎妖令庇护的最主要手段。
<br/>　　任何一个被猎妖令庇护的家族，在受到攻击时，都会立刻有猎妖殿执法队出来阻挠并保护。
<br/>　　这是常识。
<br/>　　紫云城外的执法队被困。
<br/>　　那些亡命之徒，并不知萧家有猎妖令护身，才会有那个胆子对付萧家。
<br/>　　否则，就算他们视财如命，狠辣过人，也绝不敢承受猎妖殿的怒火。
<br/>　　另外一点。
<br/>　　萧家若仅仅是有伤亡，猎妖殿绝不会太过重视。
<br/>　　而如果真的完全被覆灭，猎妖殿就会立刻重视，并彻查。
<br/>　　如果猎妖殿通过这些亡命之徒，查到背后是血雾谷在对付萧家的话。
<br/>　　血雾谷同样逃不了干系，会遭到猎妖殿的报复。
<br/>　　所以，血雾谷很聪明，雇佣的，是北山郡本土，以及周边几郡的亡命之徒。
<br/>　　就算猎妖殿真的追查到他们身上。
<br/>　　他们将有大把的理由推脱干系。
<br/>　　例如，亡命之徒本就是猎妖殿以及官方追杀的对象，他们的话不可信。
<br/>　　例如，这只是一起单纯的亡命之徒袭击萧家之事。
<br/>　　总之，只要他们没有直接对上萧家。
<br/>　　他们就有无数的理由推脱。
<br/>　　他们事前能请动一位分殿主请求撤销猎妖令。
<br/>　　事后，同样能请动一位、两位，三位，乃至更多的分殿主，暗中替他们掩盖事情。
<br/>　　只要他们肯付出巨大的代价即可。
<br/>　　一般的猎妖令庇护家族被灭，都是分殿主出手解决罢了。
<br/>　　此事，只要不惊动殿主。
<br/>　　那么，数位分殿主的联手掩盖事情，足以让事情不了了之。
<br/>　　当然，这件事，必须取决于猎妖殿中无人愿意帮萧家这样的小家族。
<br/>　　简单来说，替萧家兑换猎妖令的那位分殿主，在主殿并没有什么能耐。
<br/>　　起码，在血雾谷看来。
<br/>　　萧逸虽是最强剑主，但不过是初入王都不足一年的毛头小子。
<br/>　　恐怕请动一位分殿主兑换猎妖令已经是极限了。
<br/>　　根本不可能认识猎妖殿内真正拥有实权的分殿主。
<br/>　　一切便都顺理成章了。
<br/>　　血雾谷，一边暗中雇佣亡命之徒对付萧家。
<br/>　　一边又派遣强者困住紫云城外的执法队。
<br/>　　而另一边，又有其他人拖住裂天剑派和郡王府。
<br/>　　萧家，孤立无援，单凭自身家族势力，足以轻松被那群势力也不怎么强的亡命之徒覆灭。
<br/>　　血雾谷另外一点的天衣无缝。
<br/>　　在于拖住裂天剑和郡王府的，都必定是理所当然出现的人物。
<br/>　　而并非他们血雾谷中人。
<br/>　　如剑宗四长老。
<br/>　　他是带宗主的手谕而来，名正言顺。
<br/>　　他要让裂天剑派脱离宗门，若成了，当然好。
<br/>　　若不成，也能拖住裂天剑派的强者，无法支援萧家。
<br/>　　一举两得。
<br/>　　郡王府那边，估计也是差不多的情况。
<br/>　　“呵。”萧逸淡淡一笑，但那笑容中，杀意凛然。
<br/>　　血雾谷，确实是天衣无缝。
<br/>　　却算漏了一点，并没有料到他萧逸会及时赶回北山郡。
<br/>　　更是在来到裂天剑派，看到四长老后，直接就将一切事情都连系了起来，确定了背后是血雾谷在对付萧家。
<br/>　　血雾谷要对付萧家。
<br/>　　无非就是他在短时间内连端了多个炼制血意丹的窝点。
<br/>　　阻挠了血雾谷炼制血意丹。
<br/>　　四长老，本就有份参与炼制血意丹之事。
<br/>　　既然他出现在裂天剑派，阻挠剑派的长老。
<br/>　　那么，结合诸多因素和推测，萧逸自然确定了这一切。
<br/>　　更确定了背后对付萧家之人是血雾谷。
<br/>　　不过，还有最重要的一点，也是导致血雾谷这个天衣无缝的计划，彻底失败的一点。
<br/>　　那便是，血雾谷并不知道萧逸的另一个身份。
<br/>　　更不知道，为萧家兑换猎妖令的，根本不是萧逸请动的某位分殿主。
<br/>　　而是他萧逸本人的另外一个身份，易霄分殿主。
<br/>　　这才在他们请动狂风分殿主，请求撤销那枚猎妖令时。
<br/>　　萧逸能因此立刻知晓有人对付萧家，并马上赶回萧家。
<br/>　　“呼。”萧逸深呼吸一口气。
<br/>　　他已经明白过来一切。
<br/>　　现在的他，越发庆幸自己一直没有暴露另外一个身份。
<br/>　　这个身份，处于暗处。
<br/>　　足以让敌人防不胜防，给予自己极大的便利。
<br/>　　若非他及时赶回了萧家，后果将不堪设想。
<br/>　　“血雾谷，给我等着。”萧逸的拳头，握得噼啪作响。
<br/>　　半晌，他收敛下眼中杀意。
<br/>　　他今日来裂天剑派，还有别的目的。
<br/>　　“大长老。”萧逸看向大长老，沉声道，“我今日来，是有事问您。”
<br/>　　“何事？”大长老疑惑问道。
<br/>　　萧逸手中光芒一闪，取出了八龙焚火炉。
<br/>　　“这个炼药炉，大长老是知道的吧。”萧逸认真问道。
<br/>　　“八龙焚火炉。”大长老眉头一皱。
<br/>　　“你既问我这个问题，证明你去王都果真一直都在调查十一的事。”
<br/>　　“而且，已经调查得极其深入了。”
<br/>　　大长老的语气，变得无比凝重。
