　　像是归墟天，是在大罗天星域之中的。
<br/>　　按理来说，魔族一般都不会深入到武者的领地之中。
<br/>　　毕竟双方可是不死不休的仇敌，可这一次，九域居然是吸引了如此之多的魔族到来。
<br/>　　看来这归墟天内，的确是有着什么，在吸引着魔族。
<br/>　　“那魔族强者，是什么实力？”
<br/>　　楚风眠追问道。
<br/>　　现在楚风眠的实力，云老的了解的，可按照云老话中的意思。
<br/>　　只怕是这魔族强者，会给楚风眠带来极大的麻烦，才会在这个时候出言来提醒楚风眠。
<br/>　　“至少是一位九劫古帝，甚至可能是半步天人境，而且此人的地位极高，跟他同行的，数位都是九劫古帝，却是对此人毕恭毕敬。”
<br/>　　云老回忆的一下，开口道。
<br/>　　“九劫古帝…半步天人境？”
<br/>　　楚风眠听到云老的话，眉头一皱，九劫古帝，就已经不是楚风眠对付的了的了。
<br/>　　虽然现在楚风眠的实力在杀了陈家众人之后，又有增长，看楚风眠最大的自信，也只是在九劫古帝之下自保。
<br/>　　至于跟九劫古帝一战，楚风眠心里没底，九劫古帝如此，如果这魔族强者，是一位半步天人境的话，那可就太麻烦了。
<br/>　　楚风眠虽然没有见识过真正的半步天人境，但是现在九劫古帝楚风眠都难以对付，而半步天人境，又是比起九劫古帝还要强的多。
<br/>　　而且听云老话中的意思，在归墟天之内的魔族，九劫古帝，都不止一位。
<br/>　　“半步天人境？地位极高，难道是少帝？”
<br/>　　楚风眠脑海之中突然显现出了一道身影，曾经在幽鬼域的时候，楚风眠所见到的少帝。
<br/>　　少帝的随意的一道化身，就足以轰杀古帝，他的真正实力，只怕是也要接近半步天人境了。
<br/>　　而且在魔族之中，少帝的地位也是极高。
<br/>　　“如果真是少帝，可就麻烦了。”
<br/>　　想到这里，楚风眠都感觉到了无比棘手，如果说素不相识的魔族还好。
<br/>　　但如果是少帝的话。
<br/>　　楚风眠跟少帝之间的仇怨，可以说是不死不休，楚风眠可是从少帝的手中夺走了青铜天魔骸骨，这魔族至宝。
<br/>　　如果说真的碰到少帝，只怕是少帝，会不惜一切代价，杀了楚风眠。
<br/>　　但这归墟天，楚风眠是不得不去，他不可能因为担心，而不去，毕竟实力才是一切解决的办法。
<br/>　　躲得了一时躲不了一世，只有真正的踏入帝尊境，楚风眠才有跟少帝抗衡的资本。
<br/>　　“门主还打算去？”
<br/>　　云老也在注意这楚风眠眼神之中的变化，看出了楚风眠下定了决心。
<br/>　　“不错。”
<br/>　　楚风眠点了点头道。
<br/>　　突破帝尊境，归墟天他是必须要去的，如果不去归墟天，楚风眠想要突破还要耽误很久的时间。
<br/>　　现在的楚风眠，可耽误不起这么久的时间。
<br/>　　无论如何，半年之后，楚风眠也要面对一位九劫古帝的怒火。
<br/>　　楚风眠毁了执法堂，定然会让执法堂的上一任堂主，法理古帝暴怒，这可是一位老牌的九劫古帝。
<br/>　　倒时候，圣龙秘境的高层都闭关而出，龙千绝也不可能护得住楚风眠。
<br/>　　唯有楚风眠踏入帝尊境，才能自保，不管如何，楚风眠也没有时间耽搁下去了。
<br/>　　“我们陪你去。”
<br/>　　洛汐听到楚风眠的回答，开口道。
<br/>　　“我们三人联手，也足以抗衡一下九劫古帝。”
<br/>　　寒月漓也是开口道。
<br/>　　“不，这一次去归墟天，我一个人去。”
<br/>　　良久，楚风眠看着洛汐跟寒月漓，开口道。
<br/>　　如果真的是少帝的话，楚风眠一个人前去，就无比危险了，他不可能让寒月漓跟洛汐，也置身于危险之中。
<br/>　　而且楚风眠虽然实力不如九劫古帝，但是凭借他的半步不朽真身，始祖天龙的血脉，逃命的手段不在九劫古帝之下。
<br/>　　楚风眠一个人前去，才是最好的选择。
<br/>　　“你们二人，都是刚刚突破，如今正好在剑道门内巩固境界，而且剑道门，也不能无人看守。”
<br/>　　楚风眠沉声说道。
<br/>　　“九域动荡，必须要抓紧时间提升实力才行。”
<br/>　　洛汐跟寒月漓，都是沉默了一下。
<br/>　　楚风眠如此不让他们二人一同前去的话，肯定是有着其中的考虑。
<br/>　　而且楚风眠话的意思也没错，现在九域无比动荡，已经是越发的混乱了，提升实力，才是重点。
<br/>　　“小心。”
<br/>　　许久，寒月漓才开口道。
<br/>　　“遇到危险，就用玉符通知我们。”
<br/>　　洛汐也是开口道。
<br/>　　“好。”
<br/>　　楚风眠点了点头道。
<br/>　　“放心吧，以我的手段，天下间也没人能杀了我。”
<br/>　　楚风眠笑了笑，随即身躯一闪，化为了一道遁光，冲天而起，离开了剑道门，一路向着归墟天的方向飞去。
<br/>　　之前楚风眠要回到归墟天，通过云羽天宫便可，不过现在云羽天宫已经是来到了剑道门。
<br/>　　楚风眠在想要去归墟天，就必须要跟第一次前去一样，先去西域之中的归墟城，随后通过空间裂缝，进入归墟天。
<br/>　　一路上，北域的动荡，倒是不多，北域在九域之中本就算是中下，而且势力不多。
<br/>　　只前也就只有寒雪阁跟北邙学院之间的争斗。
<br/>　　现在寒雪阁跟北邙学院，合二为一，都成为了剑道门的一部分，这一点争斗也消失了，如今平静无比。
<br/>　　不过刚刚踏出北域，来到西域，楚风眠就感觉到了一阵一阵的灵力波动。
<br/>　　显然是就在附近，就有着不少的战斗，而且空气之中，一直是弥漫着一种血腥味。
<br/>　　这种血腥味，可不止是杀了一人，两人，可以出现的，至少是数万人，甚至是数十万人的死。
<br/>　　才会让空气之中弥漫出如此浓郁的血腥味。
<br/>　　“九域浩劫。”
<br/>　　楚风眠感叹一声，遁光却是没有停下，一直是向着归墟城的方向飞去。
<br/>　　“好浓的血腥味！”
<br/>　　楚风眠越是接近归墟城，就越是感觉到周围的血腥味越浓，显然是在这里附近，刚刚是死了不少人。
<br/>　　随即一股楚风眠灵识一扫，就在不远处，一群红衣人，正在大肆屠杀一群武者。
<br/>　　这一群武者的衣着并不相同，看来是属于不同势力的人，可无一例外，都被这一群红衣人所追杀。
