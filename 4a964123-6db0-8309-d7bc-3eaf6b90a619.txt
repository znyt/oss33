　　古帝境界强者，最为明显的一点，便是举手投足之间都蕴含澎湃的天地之力。
<br/>　　踏入到了古帝境界，楚风眠才明白了，为何古帝身上的天地之力会那般滂湃。
<br/>　　一旦是踏入了古帝境界，楚风眠感觉到他现在的一举一动之间，都足以调动天地之力。
<br/>　　真帝级别的武者，只有是在催动天地法则的时候才可以动用天地之力。
<br/>　　而古帝级别的强者，却是不需要催动任何的手段，随手就足以凝集出大量的天地之力。
<br/>　　因此古帝跟真帝之间的差距，无比巨大，宛如天沟，无法跨越。
<br/>　　现在楚风眠的境界已经是踏入到了八劫古帝的层次，在跟古帝交手的时候，境界之上便是再也没有任何的劣势。
<br/>　　虽然楚风眠现在只是刚刚踏入了八劫古帝，但是现在楚风眠的一身实力，捏死八劫古帝巅峰级别的强者，就跟捏死一只蝼蚁一样的简单。
<br/>　　甚至是在碰到九劫古帝，楚风眠都毫不惧怕了。
<br/>　　万龙帝，法理古帝……！
<br/>　　楚风眠眼神之中露出几分凶芒，他踏入到了八劫古帝，第一件事就是要去解决万龙堂，执法堂的余孽了。
<br/>　　只怕是他们都想不到，楚风眠的实力会增长如此之快，那法理古帝，还打算出关就来追杀楚风眠。
<br/>　　等到他出关的一刻，楚风眠也要好好给他一个惊喜。
<br/>　　那万龙帝之前出手，差一点杀了楚风眠，要不是云尊的玉符，还有龙千绝的出现，楚风眠就危险无比了。
<br/>　　这个仇怨，楚风眠可是记得的。
<br/>　　现在楚风眠踏入了八劫古帝境界，实力暴涨，也该是楚风眠报仇的时候了。
<br/>　　“战龙决第二重的功法出现了。”
<br/>　　就在楚风眠思考之际，在他的脑海之中，战龙决第二重的功法，突然显现出来。
<br/>　　现在楚风眠已经是将战龙决的第一重彻底的修炼完成，十二万九千六百滴远古战龙精血已经是全部凝聚完成。
<br/>　　现在的楚风眠，可以称得上是一龙之力。
<br/>　　不过楚风眠的真正实力，早已经是超越了一条远古战龙，凭借他的一身奇遇，比起远古战龙一族的龙王，也还要更加强大了。
<br/>　　这战龙决的第二重……
<br/>　　楚风眠脑海之中，看着那战龙决三个字，突然变换着。
<br/>　　随即化为三个金光闪闪的龙帝决。
<br/>　　“龙帝之力？这战龙决第二重的极限，居然可以达到一尊龙帝的实力？”
<br/>　　楚风眠看了一眼这第二重，龙帝决的内容，眼神之中露出了几分震惊。
<br/>　　这战龙决的第一重，修炼完成后，便是足以拥有一条远古战龙的力量。
<br/>　　这种功法，整个九域都寻找不到，一人之力，凝炼龙血，创造出这等功法的人，绝对是天人境的存在。
<br/>　　而现在这第二重，更是夸张，龙帝决，修炼到极限，便是拥有一尊龙帝的实力。
<br/>　　这龙帝，值得可不是如今大罗天星域之中所说的所谓龙帝，而是远古时代的龙帝。
<br/>　　远古时代，唯有踏入天人境的龙族，才有资格称为龙帝。
<br/>　　现在那万龙堂的万龙帝，实际上不过是自封为帝罢了，根本不配被称为真正的龙帝。
<br/>　　修炼这龙帝决，达到极致，以血脉之力，就足以拥有天人境的实力。
<br/>　　而且，这还只是战龙决的第二重。
<br/>　　楚风眠隐隐约约还看得见，这战龙决，还有第三重的存在。
<br/>　　“这战龙决，到底是什么存在创造出来的？”
<br/>　　现在楚风眠修炼战龙决的时间越久，也就越是足以看的出这战龙决的玄妙之处。
<br/>　　以人力，修炼强行凝聚龙血，看起来是容易，但是真正尝试起来，十有八九都会被龙血占据身躯，龙化成龙。
<br/>　　而战龙决，只要是正常修炼，就没有这样的烦恼。
<br/>　　武者最弱的，便是血脉的力量，如果纯粹以肉身对战，武者远不如妖神，魔族。
<br/>　　但是武者也有强大的一点，便是各种武技，各种灵器，凭借武者灵器，武者才可以跟妖神，魔族抗衡。
<br/>　　这战龙决的出现，却是打破了其中的平衡，一名武者的肉身血脉，足以跟妖神平起平坐，在加上武技，灵器，直接回超越妖神，魔族。
<br/>　　这也是楚风眠实力强大的根本，他就算是没有任何的奇遇，单单凭借这血脉，都算得上是一位天才强者了。

<br/>　　毒龙晶被楚风眠所吞噬殆尽，在这天元商会遗迹之中的毒雾，也是逐渐散去了。
