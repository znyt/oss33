　　　　清浅，这个就是蓝队收寻到的选手，虽然如今才国服钻1水准。
<br/>　　但是这家伙的天赋却是和杨小艺差不了多少，他也才仅仅接触游戏一个月而已，而且他今年仅仅只有17岁，虽然比杨小艺要大上一些，但是他的职业寿命还是很长的。
<br/>　　最重要的是清浅这家伙是一个打辅助位置打到如今的成绩的，所以他一加入蓝队，秦霄便是将他当成了自己的接班人来培养。
<br/>　　“你们可以啊，居然真的被你们找到一个好苗子！”看到清浅打了一局排位赛，叶帝也是不由的感叹出声。
<br/>　　“哈哈，怕了吧，明年我们中辅双剑合璧，绝对要打爆你们白队。”蒋念兴奋的开口喊道。
<br/>　　“切，你别做梦了好么，明年我们中上双剑合璧，你们根本就没有机会的好么，而且我们茄子和萝卜两人到了明年肯定会更加的老练的，到时候哼哼！”香蕉不屑的开口说着。
<br/>　　“我们这边也有小南和飞刀，明年绝对虐你们，到时候别哭鼻子就好了。”队伍里有楚南还有飞刀，蒋念自然是丝毫不怂。
<br/>　　“还是把今年打好在说吧，夏季赛的冠军我们白队可是势在必得的，你们可是要小心了啊。”叶帝这个时候突然笑着开口说道。
<br/>　　而在荣耀白蓝两队这样良好的氛围下，尤其是有着香蕉这个中二少年在，原本还有些生疏的清浅，很快便是和大家熟悉了起来，脸上也是笑容不断。
<br/>　　日子一天天的走着，很快距离夏季赛开赛的时间便是已经不足7天了，而这段时间，叶帝他们都已经将自己的状态调整了过来。
<br/>　　“小叶子，你可有一段时间没来直播间直播了啊，赶紧抽时间在比赛之前来一次直播。”就在距离夏季赛开赛5天前的时候，叶帝的二叔却是给他来了一个电话。
<br/>　　听着二叔的话，叶帝也是有些哭笑不得，自己这段时间确实很忙，倒是真的已经许久没有直播了。
<br/>　　不过二叔既然都这么说了，那叶帝自然是要打打直播，为华夏的火神直播拉拉人气的。
<br/>　　其实经历过前面那些大型的赛事，以及十王者战，如今华夏火神的直播人气已经是当之无愧的成为了no.1的存在。
<br/>　　无论在任何时段，华夏的观众人数都是最多的。
<br/>　　这一现象对叶天江来说自然是极好的，单单火神直播就为华夏创造了如此丰厚的利益，可想而知接下来叶天江的仕途恐怕就算是没有叶天正的帮助，也会是畅通无阻的。
<br/>　　“行，明天我就直播一次。”叶帝笑着开口应道。
<br/>　　两人又聊了一会后，便是挂断了电话。
<br/>　　挂断了和二叔的电话后，叶帝拿起电话直接向着苏沐婉那边打了过去，这个暑期因为十王者战的事情，倒是耽误了他很长时间。
<br/>　　虽然十王者战结束后，他回到北京陪了苏沐婉两天，但是回来后还是抑制不住他对苏沐婉的想念，尤其是看到苏沐婉的肚子变大的样子之后。
<br/>　　按照叶家的推算，苏沐婉生孩子的时间应该恰好是今年领土战的时候，倒是也赶巧。
<br/>　　结束了和苏沐婉的通话，叶帝便是思索起来明天要如何直播，直播打王者排位这肯定是不行的，版本要在三天后才改动，才会增加到200个王者名额，现在的排队时间还是很长的。
<br/>　　打低分段那就更没意思了，除了有点娱乐效果外，对自己没有丝毫的帮助，而且直播间里的粉丝也不一定愿意看自己去虐低分段，毕竟自己不是白银三杰。
<br/>　　“只能创作几首歌曲了，就剩下一天的时间了，想要创作其他的歌曲也没那么多时间，就弹几首吉他曲吧。”拿定注意后，看了看时间，洗漱过后，叶帝便是回到了自己的床上休息了下去。
<br/>　　而在楼下，此刻杨小艺那孤单的身影依旧是在那里一个人打着排位赛，虽然叶帝已经跟她说了让她白天去打排位赛，但是杨小艺却依旧是喜欢晚上一个人在这边打游戏。
<br/>　　有时候看着杨小艺那孤单的身影，叶帝心里也挺心疼的，杨小艺之所以这样，可能跟他这么多年的生活有关系吧，可能她已经习惯了孤独，习惯了黑暗。
<br/>　　第二天一大早，叶帝一行人依旧是早早的起床，包括清浅在内，他也是跟着大家一起出来锻炼来了。
<br/>　　当然了唯一的例外当然是杨小艺，听林雅楠说杨小艺5点钟才回到房间去休息，也就是他们刚起来的那个时候。
<br/>　　“小艺这样也不行啊，白天就吃一顿饭，晚上又熬夜这么久，这孩子真让人心疼。”林雅楠开口说道，这一段时间也是破天荒的起来跟着锻炼来了。
<br/>　　“已经跟他说了很多次了，让她晚上不要打游戏，白天和大家一起玩，她就是不听。”蒋念也很无奈，对杨小艺大家都是打心眼喜欢。
<br/>　　“还是有些生，等大家在熟悉熟悉就好了，今天晚上让阿姨多给小意备些夜宵吧，让他晚上也吃点东西，要不然白天只吃一顿饭，就算是神仙也受不了的。”叶帝跟着开口说道。
<br/>　　既然杨小意这么喜欢晚上打游戏，那他也没必要去强行更改对方的喜好，等到哪天杨小意想白天打游戏，她自然会出来的。
<br/>　　而且在叶帝的心里他这个时候已经想到一个要为杨小艺改时差的办法了，只不过这个还要在等上几天，相信到了那个时候杨小艺就算是不改，那也由不得她了。
<br/>　　而就在当天中午，叶帝接到了一个意想不到的电话，来电话的不是别人正是他们京都大学的校花柳清月。
<br/>　　他和柳清月还真的很久没有见面了，当然了私下里大家还是经常沟通的，主要还是柳清月在请教叶帝如何上分。
<br/>　　而叶帝私下里也是会经常去看柳清月的直播的，所以虽然很长时间没见，但是大家的感情却是丝毫不见生疏。
<br/>　　想想如今的电竞三美，居然都在自己身边，叶帝这心里也是美滋滋。
