　　面对慕容飞提出的这第三个要求，赵国邦没有立刻答应下来，沉默了片刻对慕容飞说道：“你应该知道，龙隐部队的所有行动都是绝密，甚至其中有相当一部分是没有解密期限，所以你的这个要求……”
<br/>　　“龙隐部队的规矩我自然是知道的，我也不会破坏龙隐部队的规矩。”慕容飞解释道：“我所指的告诉我父母，是让他们知道他们的儿子是一个正直的人，我是他们眼中的骄傲，我不希望他们老两口在被人的冷言冷语和指指点点中抬不起头来。”
<br/>　　“这个我可以答应你。”赵国邦重重的点头，说道：“如果你在行动发生意外，我会亲自去跟你的父母去解释清楚，你是他们的骄傲，一直都是，从来没有改变过。”
<br/>　　“谢谢你，野狼。”慕容飞对赵国邦说道。
<br/>　　“你不需要感谢我，这些都是你应得的。”赵国邦对慕容飞说道：“你家里面不需要担心，我会帮你照顾好，不管是你的父母，还是佟千雪父母都会打点好。来吧，再陪我打几把游戏，好久没玩了，玩了两把瘾上来了，这次得好好过过瘾。”
<br/>　　“来。”
<br/>　　慕容飞这次没有跟赵国邦继续在局域网里玩，而是登陆了一个游戏对战平台，毕竟CS早就没有了当年的火爆，不会再看到网吧里一排坐着打CS的情景，所以要想玩个痛快就只能去游戏对战平台上玩。
<br/>　　玩了个把小时，赵国邦跟慕容飞两个人一前一后走出了网咖，慕容飞站在街边拦下一辆出租车，返回快捷酒店；而赵国邦则是走到对面马路，钻进一辆停在路边的黑色奥迪A6L中，赵国邦上车后，黑色奥迪A6L立刻亮起左闪灯，从路边驶离，驶入车道。
<br/>　　坐在副驾驶上的云岳扭头对赵国邦问道：“野狼，事情怎么样？”
<br/>　　“他答应了。”
<br/>　　赵国邦顺带把慕容飞提出的三个要求也一并跟云岳说了，神色严肃的说道：“‘追魂’行动可以启动了，这次非要从小丑身上扒下一层皮来。”
<br/>　　“明白！”
<br/>　　……
<br/>　　慕容飞坐着出租车在快捷酒店前停下的时候，时间已经是快要十一点半了，服了车费，慕容飞推门走下出租车，抬头看了眼在夜色下快捷酒店灯牌呼出一口气，白雾被呼出口，很快就被风吹散，慕容飞竖起衣领迈步走进快捷酒店。
<br/>　　前台值班看到慕容飞，微笑着向慕容飞打招呼，慕容飞点头跟对方示意了一下，按下电梯，用自己的房卡在电梯上刷了一下，按下楼层，返回自己的客房。
<br/>　　用房卡打开房门，慕容飞担心吵醒佟千雪刻意放轻了动作，可是关上门转头却看到佟千雪站在自己身后。
<br/>　　“把你吵醒了吗？”
<br/>　　慕容飞脱下自己的外套，挂在衣架上。
<br/>　　“你去哪儿了？”佟千雪对慕容飞问道。
<br/>　　“部队的一个战友在东海市，来了总要去见一面，就出去跟他见了一面。”慕容飞回答道。
<br/>　　“真的只是去见战友？”佟千雪皱眉，有些怀疑的看着慕容飞，女人天生敏锐的直觉，让佟千雪觉得慕容飞似乎是有着什么事情在瞒着自己。
<br/>　　“真的只是去见战友。”慕容飞把佟千雪推进房间，说道：“我对这里又不熟悉，你以为我能跑到哪儿去？明天早上还要早去，赶紧去休息。”
<br/>　　佟千雪被慕容飞推着回到房间的路上，又是不放心的跟慕容飞确认了一遍，知道慕容飞一再肯定的向佟千雪保证，自己只是出去见了战友，才把她说服回到床上。
<br/>　　这一夜，慕容飞失眠了。
<br/>　　中间，他数次强迫着自己睡去，可是刚把眼睛闭上，从他被赵国邦选中带入龙隐部队开始，就像是电影播放一样，一幕一幕出现在慕容飞眼前，“赤鸦行动”，泰勒……一切都是那么的清晰，就像是在昨天刚发生一样。
<br/>　　扭头看了眼自己身边睡得正熟的佟千雪，轻叹一口气，他很舍不得佟千雪，舍不得他现在的家庭，纵然有万般不舍，他还是必须要放下这些“不舍”，他是“无常”，他是龙隐部队的兵！
<br/>　　临近天亮的时候，慕容飞才迷迷糊糊个把小时，被街道上清扫车的音乐声吵醒，起床去浴室用冷水洗了把脸，让自己清醒了一下。
<br/>　　因为不知道会不会有影响，所以慕容飞和佟千雪两个人没有吃早饭就退了房间直奔司徒空说得地方，出租车上慕容飞只是说了要去的地方，出租车司机没有仔细问地址就直接按下了“空车”牌子，汇入到车流中，显然是对这个地方很了解。
<br/>　　原本慕容飞以为在西医横行的今天，大家对中医的关注度会很低；可是到了地方，慕容飞才发现自己错了，即便现在才刚过八点，可是在门口就已经排起了长队。
<br/>　　不是中医不行了，而是好的中医太少，更多的是打着中医招摇撞骗的骗子，加上有心人在背后的推波助澜，才使得越来越多的人戴着有色眼镜去看中医。
<br/>　　可是如果有真正好的中医，它的受欢迎程度要远超西医，毕竟这是老祖宗传承了几千年的东西！
<br/>　　慕容飞和佟千雪刚走近队伍，就有黄牛上前来兜售排队号，并且声称如果不买他的排队号，就算是排到下午都未必能轮得上。慕容飞没有理睬黄牛，直接带着佟千雪往前走去，对站在门口穿着布衣，引导就诊的年轻女子说明了自己的身份。
<br/>　　“原来是慕容先生和佟小姐。”年轻女子对慕容飞和佟千雪说道：“两位请跟我来，大掌柜已经吩咐过我了，你们一到就带你们去见他。本来我还好奇早就已经不出诊的大掌柜为什么今天会突然出诊，原来你们是小少爷的朋友，这就难怪了。”
<br/>　　年轻女子没有带着慕容飞和佟千雪进入正厅，而是推开大门旁的一扇小门，带着两个人走了进去。
<br/>　　“大掌柜，慕容先生和佟小姐到了。”年轻女子带着慕容飞和佟千雪走到一间古色古香的厢房前，在门上敲了敲说道。
