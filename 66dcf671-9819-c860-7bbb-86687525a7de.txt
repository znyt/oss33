　　“不要把她交给钟馗他们，亲手杀了星月奴，为道渊报仇！”张无生哑着嗓子说道。
<br/>　　叶少阳捏紧拳头，用力点了点头，“师叔你放心，我一定手刃她！”
<br/>　　“好了好了，不就是死一回呢，都别哭丧着脸了，来办正事吧，正事要紧。”
<br/>　　张无生走到自己的尸体边上，看了一眼，说道：“这几天我一直说自己心口疼你们还记得吧，那就是因为，我隐约能感觉到身体有东西，八成就是星月奴的洞府了，就在我胃里，你们把我肚子剖开，找到它！”
<br/>　　“这……”叶少阳很是为难。
<br/>　　人家才死，就要做出毁坏人家尸体的事，叶少阳有点干不出来。
<br/>　　张无生也意识到这点，白了他一眼道：“得了，我还是送佛送到西吧。”自己魂归尸身，但因为经脉震断，活是活不成了，他成了临时的鬼尸。
<br/>　　让曲波送了把刀子过来，张无生握住刀柄，插进自己肚子里。
<br/>　　叶少阳几人都转过头去，虽然知道他现在是鬼，只是控制自己的尸体，并不会感觉到疼，但这一幕看上去还是有点惊悚和残忍。
<br/>　　等了一会，突然一只手从后面搭在自己肩膀上，叶少阳转头一看，是“张无生”站了起来，一只手从自己胸前的伤口里插进去，很专注地摸索着。
<br/>　　叶少阳脸上的肌肉抽了两下。
<br/>　　随着他的手血淋淋地拔出来，大家惊讶发现，有一股半透明的气体从他身体里不断流出来，如同烟雾一样朝四周散去。
<br/>　　真气！
<br/>　　这是失去了控制的真气，在自行散去，回归自然。
<br/>　　三人大惊。
<br/>　　“师叔，你这……你没阴阳转逆？”
<br/>　　阴阳转逆是法术界的一个传统：有些法师，实力强大到一定地步之后，只要是自然死亡——意外死亡的话，只要不是当场就死，参考青云子——都是可以用这“阴阳转逆术”，将一身法力转化成为鬼魂的魂力，稍微炼化一番，直接就是修为了。
<br/>　　虽然转化率跟实力成正比，不能全部转化，之后也不能再修炼人间法术，但还是可以学习鬼术，提升自己的实力。毕竟，很多人修炼了一辈子，才积攒了这些真气，很多人死后也不想放弃。
<br/>　　阴阳转逆很容易，对张无生这样的大佬来说根本不是事，而他……居然了自己一身修为，任凭这些真气失去了依附之后，自行散去了。这让叶少阳三人怎能不吃惊。
<br/>　　“我累了，我做了一辈子法师，死后不想再干这个了。”张无生淡淡说道。
<br/>　　为了规范人间秩序，阴司要求法师们以身作则，一旦死后用阴阳转逆术来保持修为的法师，必须去阴司报道，而且只能在阴司供职。
<br/>　　“那你……”
<br/>　　“去轮回，好好改造，重新做人。”张无生轻描淡写。
<br/>　　叶少阳却是听者有心，沉吟起来。
<br/>　　张无生一边在自己身体里摸索，一边跟大伙淡定地聊天，这场面……实在有点惊悚。
<br/>　　“好了好了，赶紧办正事。”张无生把那只手从身体里拿出来，血还是热的，手上沾的全是。张无生当众把那只鲜血淋漓的手在三人面前摊开，在一团血垢之中，有一个亮晶晶的东西显露出来。
<br/>　　九星阁？
<br/>　　叶少阳按捺着激动，把这东西拿过来，转身找到洗脸盆，在水中洗干净，拿到大家面前。
<br/>　　有拳头那么大，形状像个梭子，黄中泛绿，晶莹剔透的，一眼看进去，里面有几个珍珠一般的小珠子，菱形排列，看上去极为精致。
<br/>　　“这就是九星阁？”叶少阳喃喃自语。
<br/>　　“她总不能放个定时炸弹在我身上吧。”张无生拍了一下他的肩膀，“你茅山的阴阳镜呢？”
<br/>　　“在我这，咋了？”
<br/>　　叶少阳掏出阴阳镜，张无生接过去，目光在三人脸上扫了一遍，道：“好了，我任务完成了，剩下的你们年轻人干吧，我今天起就在里头不出来了，什么时候你们回到人间，再把我放出来，到时候我就跟老钟去阴司。”
<br/>　　叶少阳苦笑，“你就这么信任我们，能活着回去？”
<br/>　　“你要死了，人间法术界就彻底沦陷了。”
<br/>　　一句话说得叶少阳亚历山大。
<br/>　　“我尽力吧！”叶少阳冲他笑了笑。
<br/>　　张无生魂魄离开身体，钻进了阴阳镜中，尸体立刻倒了下去。
<br/>　　曲波含着泪把尸体搬到床上躺着，自己跪在边上磕了三个响头，转过头来询问三人，尸体怎么办。
<br/>　　林三生提议一把火烧了，带着骨灰回去，大家也都同意。于是三人一起来到外面，叶少阳找来宫里管事的太监，让他们去准备柴火，很快，一座柴堆就在花园的空地里搭起来，张无生的尸体摆在最上面，林三生已经把捉鬼联盟所有人都叫来了，在柴堆前站成一排。
<br/>　　曲波上前点火，其余人表情肃穆地观礼。因为张无生魂魄就在阴阳镜中，没必要念往生咒，大家就静静地看着尸体焚化。
<br/>　　这一刻，叶少阳的心里不是悲伤，反而热血澎湃。
<br/>　　他想到了人间法师的传承，从古至今，一代代传承下来，靠的不光是法术、法器，而是这种慷慨赴死的勇气。
<br/>　　每个时代，都有厉鬼大妖，纵横作乱，祸害人间；每个时代，都有降魔法师，斩妖除魔，捍卫人间太平。
<br/>　　道高一尺，魔高一丈。
<br/>　　收不完的恶鬼，杀不光的厉妖，斩不尽的死僵，除不尽的凶灵。
<br/>　　人间法师，就是这么一代一代传到今天。
<br/>　　如今，这种传承落在了自己头上。
<br/>　　天降大任于斯世人……叶少阳没这种感觉，他体会到的只是一种责任，作为法师的责任。
<br/>　　张无生的尸体被烧成了一把骨灰，被曲波装在了一口坛子里，小心地收起来。
<br/>　　叶少阳带着大家回到皇宫里，商讨战术计划。
<br/>　　“战前会议”自然是林三生主持，召集了十几个将军，让他们出去把外面的生灵全召集进城，在皇宫外面等候消息。
<br/>　　至于进入九星阁作战，自然是用不上他们的，兵贵精不贵多，人太多会乱，难以照顾，真正能上去打的，还是捉鬼联盟这些人。再加上一个钟馗，小马也选了几个在这里收的手下，还有林三生带来的一些人。
