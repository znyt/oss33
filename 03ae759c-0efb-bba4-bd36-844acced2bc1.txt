厉嫣然迅速拽了一下拳，她真的不知道夏夕绾给陆子羡喝了什么迷魂汤，竟能让陆子羡这么喜欢袒护她。
<br/>
<br/>        范恬当然也听说了昨晚夏家五十大寿上发生的事情，帝都四大豪门厉陆徐都为夏夕绾抬轿，竟然让夏夕绾从那段不好的流言蜚语里成功脱身而出，而她本人更一跃成为了清冷风华陆子羡陆院士的未婚妻，范恬心里又开始不安。
<br/>
<br/>        这时有人道，“快擦擦口水，夜老来了！”
<br/>
<br/>        大家迅速从陆子羡消失的方向收回了花痴星星眼，只见科学院院长夜老从前方走了过来。
<br/>
<br/>        “夜老好！”
<br/>
<br/>        夜老一身中山装，精神矍铄，一双浑浊的双目里闪烁着利光，他走过来，慈爱的看着自己的外孙女厉嫣然，“嫣然，从今天开始你就正式加入科学院了，欢迎你。”
<br/>
<br/>        这么多人围着，厉嫣然没有叫外公，她甜美的笑道，“夜老，谢谢你，我一定不会辜负你的期待的。”
<br/>
<br/>        “恩。”夜老满意的点了点头，“嫣然，这一次你们三个人从选拔赛中脱颖而出，现在我就有一个任务交给你们。”
<br/>
<br/>        现在就有任务了？
<br/>
<br/>        今天是厉嫣然范恬她们第一天进入科学院，夜老就委派了任务，足以可见对她们的重视和锻炼。
<br/>
<br/>        “夜老，什么任务？”厉嫣然连忙正色的问道。
<br/>
<br/>        “我们科学院和科研中心为了响应全民心理健康建设的这一个理念，所以打算开两个电台，准备两档晚间情感主播专栏，这其中一个电台，我要交给嫣然你们三个人。”夜老道。
<br/>
<br/>        厉嫣然知道自己这是被委以重任了，她进科学院的第一个任务就是成为晚间情感电台的主播。
<br/>
<br/>        “夜老，那另一个电台交给谁，是不是由科研中心的人员负责？”厉嫣然问。
<br/>
<br/>        说起科学院和科研中心，这其中是有一段故事的。
<br/>
<br/>        五年前的帝都是没有科研中心的，只有科学院，夜老一人独大。
<br/>
<br/>        但是大概五年前，帝都爆发了一场病毒，当时病毒蔓延，科学院得到了第一手的消息，所有的院士携带着自己手下的医学团队参与了和这场病毒的抗争。
<br/>
<br/>        当时的陆子羡作为科学院最闪耀的那颗星星，却十分不巧的在国外参与学术研讨会，消息封锁导致他没能回来。
<br/>
<br/>        当时最有希望在第一时间攻克这个病毒的就应该是院长夜老，所有人都将希望放在了他的身上。
<br/>
<br/>        但是结果令人大跌眼睛，第一时间攻克这个病毒的是T大校长邱远邱院士，听说当时邱院士的团队里出现了一个惊才绝艳的女孩，那个女孩首先将病毒株进行了医学分离，找到了病毒的起源，成功遏制了病毒的扩散。
<br/>
<br/>        当时那个惊才绝艳的女孩轰动了整个医学界，所有人都想一堵她的真容，但是邱远邱院士却摇了摇头，没有将她曝光，而且封存了所有关于她的资料，让她消失在了聚光灯里。
<br/>
<br/>        大家到现在对那个女孩都一无所知，只知道她代号X。
<br/>
<br/>        X，代表着希望和力量。
