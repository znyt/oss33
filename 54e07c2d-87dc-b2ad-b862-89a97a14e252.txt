　　阴寒域，忘川城。
<br/>　　此时，四道身影漫步在忘川城外碎石铺砌的道路上，道路两旁绽放着鲜血一样的绚烂鲜花。
<br/>　　彼岸花！
<br/>　　彼岸花有许多种颜色，红色彼岸花盛开于地狱，白色彼岸花绽放于天堂。
<br/>　　在传说中，彼岸花的花香传拥有魔力，能唤起死者生前的记忆，此花只开于黄泉，是黄泉路上唯一的风景。
<br/>　　望着漫山遍野的彼岸花，慕清霄目光颇为诧异，没想到传说中的彼岸花，在鬼界中却是如此的泛滥。
<br/>　　“彼岸花，当真有唤起死者生前记忆的效果？”
<br/>　　话语刚落，天鬼皇的声音响起：“彼岸花确实拥有这种效果，无论是神界，魔界，鬼界都生长着彼岸花……”
<br/>　　慕清霄了然的点了点头，在世俗中，彼岸花是传说中的存在，只不过传说比较离谱而已。
<br/>　　在传说当中，彼岸花能够将死人复活，但在各界中，彼岸花只是一种价值微乎其微的药材罢了。
<br/>　　唤灵丹，由彼岸花为主药炼制而成，能够唤出死者生前的记忆。
<br/>　　“阁下，前面就是忘川城了，乃是阴寒域的主城，双煞鬼王就在其中。”天鬼皇语气严肃的说道。
<br/>　　不多时，一座庞大的城市便出现在众人的视线中。
<br/>　　城市占地面积很辽阔，最起码比酆都大上十倍不止。
<br/>　　漆黑如墨的城墙望不到边角，城门匾额上，印着猩红的两个大字‘忘川’，盯着两字望了片刻，一股阴寒血腥的气息就扑面而来。
<br/>　　慕清霄体内灵气轻微波动，阴寒血腥的气息就被抵消掉。
<br/>　　城门口由两名阴寒域的鬼兵把守，两名金丹级别的鬼兵来把守城门，若是放在世俗中，那是不可想象的。
<br/>　　进入忘川城的瞬间，两名鬼兵下意识的瞥了慕清霄等人一眼。
<br/>　　毕竟，四人的身影都笼罩在黑袍中，不过在鬼界这种奇怪的鬼修并不少，鬼兵瞥了一眼过后便收回目光。
<br/>　　漫步于忘川城的街道之上，令慕清霄有种行走在世俗街道上的错觉。
<br/>　　虽说是错觉，但也不见得。
<br/>　　因为，各种城池中，生活的也是鬼族后裔，他们是鬼族中的普通鬼混，与世俗中有着相同的生活规律。
<br/>　　“走在你们鬼族的城池中，真是令我觉得有些古怪。”
<br/>　　黑袍底下，慕清霄的视线在街道上扫视着，一幕幕繁荣昌盛的景象映入他的眼帘中。
<br/>　　“本座当年去你们人族都城的时候，也曾有过相同的感受。”
<br/>　　说着，天鬼皇不禁回忆起，当年在人族世俗中的生活，虽说那段记忆中有着颇多不美好的事情，但对他来说却是非常的珍贵。
<br/>　　慕清霄淡然一笑，他已经感受到，在忘川城的正前方，有着两股很强的气息。
<br/>　　两名强大的鬼修，比天鬼皇要强上一些，合体八重中期，若是不出意外，估计就是双煞鬼王兄弟了。
<br/>　　“双煞鬼王，都比你高出一重境界。”
<br/>　　天鬼皇面色一动，可见慕清霄已经找到了双煞鬼王所在地点，深不可测的实力，着实令他一惊。
<br/>　　“他们修为上比本座高一个小境界，但本座是天鬼一族，血脉上就占据优势，并不惧他两人。”
<br/>　　鬼族中也存在血脉一说，而天鬼则是鬼族中颇为高贵的血脉，只是在天鬼皇这一代，并未将血脉发挥到极致而已。
<br/>　　拥有天鬼一族血脉的天鬼皇自然比其他鬼修占据优势。
<br/>　　“其中一名鬼王似乎在破境，现在不动手更待何时？”
<br/>　　说罢，周围空间一阵扭曲，慕清霄的身影就率先消失，天鬼皇对天魑皇和魉妹吩咐一声后，当即追了上去。
<br/>　　片刻后，慕清霄出现在忘川城尽头的一座庙宇上空，不多时天鬼皇也追了上来。
<br/>　　望着下方的庙宇，慕清霄古怪的瞥了眼天鬼皇，脑海中不禁想起酆都金碧辉煌的宫殿，除了实力，爱慕虚荣，也算是天鬼皇最大的弱点了吧。
<br/>　　瞥了眼脚下的庙宇，不知何时，天鬼皇手中已经出现一柄赤红色长刀。
<br/>　　鬼王庙，在忘川城中的地位，相当于世俗中的皇宫，在普通人眼中，乃是圣地一般的存在。
<br/>　　鬼王庙占地面积很大，其中金碧辉煌，前殿还能见到有鬼族后裔上香如同拜佛一般，偌大的后院却是空荡荡一片。
<br/>　　而两股强大的气息，就隐藏在鬼王庙后院最深处。
<br/>　　当天鬼皇取出自己的武器后，合道七重的气息便自黑袍中席卷而出，一时间风云色变。
<br/>　　慕清霄双手结印，一股无形的屏障将整个鬼王庙都笼罩在其中，原本在上香的鬼族后裔都莫名的出现在了庙宇外面。
<br/>　　对此，鬼族后裔并未察觉到任何的不对劲，一时间也想不起来，刚才自己等人究竟在干什么。
<br/>　　空间结界布下后，一名容貌阴柔的黑衣男子，抱着一柄长剑从后院中走出。
<br/>　　看到黑衣男子的瞬间，慕清霄面色平淡，天鬼皇则面色浓重了一分，抬起赤红色长刀，杀意凌然。
<br/>　　“天鬼皇，你知道自己在做什么吗？”
<br/>　　阴柔男子缓缓睁开双眼，长剑抱在胸口，纹丝不动，双眼中漆黑如墨，看不出任何一丝波澜。
<br/>　　慕清霄目光一凝，没想到面前的鬼王居然是一名剑道强者。
<br/>　　原本，阴柔男子闭着双眼时，慕清霄没有察觉到，但睁开双眼的一瞬间，他对锐利的感觉非常敏感。
<br/>　　鬼王与鬼王，刀与剑。
<br/>　　嘴角牵起一抹弧度，慕清霄身影隐没于空中，道：“真是有意思，能不能占领阴寒域，就看你自己了。”
<br/>　　另外一名鬼王没有出现，慕清霄也没有出手的必要。
<br/>　　如果天鬼皇连其中一名鬼王都无法解决，那么他占领阴寒域后，最终也唯有走向破灭而已。
<br/>　　立于高空，天鬼皇手持赤红长刀，长刀所指，霸气测漏：“本座当然知道，鬼冥，你认为本座有心情跟你开玩笑吗？”
<br/>　　“看来你的野心不小啊，想染指阴寒域？给你一次机会，立刻滚出忘川城，否则，你们天鬼一族也没必要存在了。”
<br/>　　天鬼皇眼中杀意猛增，赤红色长剑一挥，一道血红刀气宛若要劈开天地，席卷而出。
