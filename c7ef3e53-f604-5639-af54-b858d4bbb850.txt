　　第411章
<br/>　　卫兵的脚步声咚咚响起，距离越来越近，想也知道，守着院子的卫兵已听到响声，正向大殿跑来。
<br/>　　白元飞此刻已顾不得许多，一把将李行哉推进地洞中。
<br/>　　“快点爬！”
<br/>　　李行哉眼前一黑，什么都看不到，身前只有一条深邃看不到底的洞。
<br/>　　如此狭窄的空间，似乎也只能爬了。
<br/>　　地洞一直从城外通向这里，开掘的空间并不大，险险只能容一人手足并用前进。随之，巨大的恐惧感汹涌而来，李行哉感觉地洞的四壁都向自己压来一般，然后就是窒息般的紧张。
<br/>　　“快点，你想我们都死在这里么！”
<br/>　　身后传来王德的声音，他在用力推李行哉。和李行哉不同，王德经常生活在地下，有时候深更半夜，都是他一人探入古墓中，何况是今天。但李行哉却并没有这样的生活经历，这时头一遭遭遇，他才发现自己对黑暗如此恐惧。
<br/>　　然而王德说得不错，留给他们的时间并不多。假若再耽搁下去，三人谁也走不了。如果被抓住，李行哉自然不会有事，但却会害死前来救自己的白元飞和王德。
<br/>　　他咬破舌尖，让疼痛使自己清醒过来，随即用双肘前进。
<br/>　　此时此刻，卫兵已经冲进房间，眼前这一幕令他们大惊失色。只见大殿倒着几具尸体，而李行哉已不翼而飞，只有白元飞立在哪里。
<br/>　　“什么人！”
<br/>　　“拿下他！”
<br/>　　卫兵的反应可以说是半点不慢。这些日子李行哉的待遇虽然提了上去，但对他的监视却是似松实紧，能派来把守院子的卫兵也同样是凉州城的精英。
<br/>　　白元飞立在原地嘿嘿笑笑，忽地从身上拿出一包物事，挥手点燃，冲卫兵砸了过去。
<br/>　　“火药，是火药！”卫兵惊得大叫。
<br/>　　轰！
<br/>　　爆炸声响起，大殿被砸塌了一角，本来安静如湖面的城主府，随着这声巨响被惊动。
<br/>　　宋伯康此时还没有入睡，正在书房与陆洛、戴白议事。三人表情都很严肃，可见所讨论事情的重要性。
<br/>　　此刻，突然传来的巨响令窗棂都震动起来，三人同时看向窗外，脸露困惑神色。
<br/>　　一名手下急匆匆跑进房间，道：“大人，大人，不好啦！”
<br/>　　“发生什么事了？”宋伯康问。
<br/>　　“有贼人挖掘地道潜入城主府，将六皇子劫走了！”
<br/>　　“什么！”
<br/>　　陆洛和戴白同时起身，表情十分震惊。李行哉的存在对他们而言极其重要，涉及到以下一步步计划，却是没想到，在城主府内他竟然生生丢了。
<br/>　　宋伯康却稍显镇定些：“派人去追，一定要活着把殿下救回来。”
<br/>　　“已经有人进入地洞去追了。”
<br/>　　宋伯康和陆洛、戴白三人急匆匆赶到事发现场，看到了大殿中央的地洞。
<br/>　　“大人，人就是从这里逃走的，已经让卫兵进去追，现在还没动静。”
<br/>　　看着地上的黑洞，宋伯康脸黑得难看。很快，进入洞口的卫兵便被堵住了，又过了很久，从里面拖了一具尸体出来。
<br/>　　对方的死相十分难看，是被锐器贯穿头骨而死。地洞狭窄，一个人在里面转身都不能，而尸体一旦堵住路，就是千军万马也挤不进去。
<br/>　　宋伯康深吸一口气，压抑住内心的怒火。
<br/>　　“全城出动，派人到城外搜捕，尽快找到地洞的出口。”
<br/>　　“是！”戴白干脆答应。
<br/>　　凉州城有十万精兵，宋伯康说的全城出动，便是这十万人倾巢而动。甚至还包括他自己，凉州城所有城门打开，宋伯康骑了一匹快马出城。十万兵丁，沿着城墙开始拉网式搜捕。
<br/>　　十万人不是一个小数目，但对于今天的任务来说，人手依旧算不上充裕。
<br/>　　宋伯康对这一点心知肚明，于是心情更加焦躁，但现在他能做的，却也是尽自己的最大力量，把李行哉留住。
<br/>　　至于究竟能不能留住，也只能尽人事听天命。
<br/>　　李行哉觉得此生都没有如此痛苦过，地道内狭窄、潮湿、寂静，还有各种叫不上名字来的小虫子。闷热的空气让李行哉恨不得将自己的胸膛撕扯开。关键是，这种烦闷窒息的情绪竟然无一人可倾诉，寂静从来没有如此可怕。
<br/>　　再往前行空间渐渐开阔起来，但这种开阔也仅仅是到能够转身的程度。紧跟着，李行哉面临了人生从未有过的挑战。
<br/>　　地面开始积水，各种腥臭扑鼻而来，时不时还有老鼠从身前爬过。
<br/>　　“这是什么！”李行哉吓得尖叫。
<br/>　　“别废话了，快往前爬吧。”王德催促到。
<br/>　　“究竟还有多久才能离开这鬼地方。”如果可以的话，李行哉恨不得立刻把头顶炸开，让新鲜空气透进来。
<br/>　　“再有一个时辰吧，如果你的速度够快的话。”王德道：“你可千万别吐了，很恶心的。”
<br/>　　“你还会觉得恶心。”
<br/>　　李行哉强忍着反胃的情绪，一点一点跋涉过阴暗恶臭的地洞。
<br/>　　想要冲城外挖掘出来一条通向城内的地道，其实并不容易。过程中会遇到各种各样的问题，好在王德是这方面的内行，至少塌方的事情并未在工程中出现。在进入城中后，城内的排水设施发挥了作用。自然，这不是一件很愉快的差事，却是最省力的一种方式。
<br/>　　如今李行哉遭遇的就是阴沟中的污水，其中甚至夹杂着人兽的粪便。就算李行哉这些年一直在外流浪，过惯了苦日子，可这样的事情却是头一次。
<br/>　　城外，宋伯康催马疾驰，一名手下匆匆过来，道：“大人，找到地道的出口了。”
<br/>　　宋伯康眼睛一亮：“走，快带我去看看！”
<br/>　　在城门偏东方向，上面盖着枯枝稻草，打开后就可以看大一个地洞。里面夹着树木，还有挖掘的痕迹。
<br/>　　宋伯康一颗心落了踏实，总算是找到了。只要守住这里，李行哉定然是插翅难逃。不过程大雷还真是狡猾，蛤蟆城在凉州城西北方向，他们竟然从东边挖掘地洞。
<br/>　　“挖，把地洞挖开！”宋伯康恶狠狠的下令：“找到他们！”
<br/>　　在找到地洞入口后，凉州兵就向这里集结，于此同时，李行哉还在艰难跋涉。
<br/>(本章完)
