        难道，现在在房间里的这个四爷不是大哥吗？
<br/>
<br/>        童筱颖正疑虑之际，突然有东西在她的背脊上移动，像是在写着什么字：有监听器。
<br/>
<br/>        整个神经紧绷了一会儿，得知房内有监听器后，童筱颖讷讷地开口道：“四爷，你可不可以把枪放下来，我只是来给你送早餐的。”
<br/>
<br/>        她这样说话，应该不会引起监听他们的人的怀疑了吧！
<br/>
<br/>        “放下早餐就滚。”身后传来男人冷淡的呵斥声。
<br/>
<br/>        童筱颖转过身，只见傅夜沉早已穿戴整齐，并戴上了假面眼镜。
<br/>
<br/>        说来也奇怪，这八爷和十爷也是戴着假面眼镜的，似乎都不愿意让别人看到他们的真容。
<br/>
<br/>        比如说，哪天他们夜惑里的十位爷，没戴假面眼镜在大街上相遇，说不定彼此都不认识。
<br/>
<br/>        童筱颖心不在焉地将早餐放在了一旁的矮柜上，瞥了傅夜沉一眼，准备离开的时候，傅夜沉又命令道：“回来，给我换药。”
<br/>
<br/>        诶？！
<br/>
<br/>        大哥受伤了吗？
<br/>
<br/>        童筱颖下意识地刚要开口关问，就被傅夜沉再次先声夺人：“药在那边的角柜里。”
<br/>
<br/>        “是！”童筱颖弱弱地应了声，走到那边的角柜前，将药去了过来。
<br/>
<br/>        她眼睁睁地看着他在床边坐下，脱下身上的黑色的衬衫，然后朝她背过身去，他后背上靠左肩的地方有块纱布。
<br/>
<br/>        童筱颖凑过去小心翼翼地替他拆纱布，看到一个血肉模糊的枪伤口，心疼得鼻子一酸，眼泪瞬间就从眼眶里冒了出来。
<br/>
<br/>        她一边无声地流泪，一边小心翼翼地替他处理伤口，包扎完后，又贴心地帮他把衬衫套上。
<br/>
<br/>        在他转过身来时，她便偷偷地把眼泪抹掉了。
<br/>
<br/>        可是，已经哭红的眼睛，还是出卖了她。
<br/>
<br/>        “你不用这么怕我，我不会杀了你，至少我还想留着你伺候我。”傅夜沉抬起手，捧着童筱颖的脸颊，表里不一地说。
<br/>
<br/>        他很想说几句宽慰她的话，但这些话不得不憋在心里。
<br/>
<br/>        童筱颖对上傅夜沉的眼睛，从他的眸子里看到了少有的温柔，嘴角情不自禁地微微上扬，应道：“四爷好好休息，我先下去了。”
<br/>
<br/>        傅夜沉微笑着点了点头。
<br/>
<br/>        另一边，古堡里的监听室里，八爷取下了耳麦，看向坐在落地窗旁的单人沙发上抽着雪茄的十爷冷逸峰。
<br/>
<br/>        “这个阿四，并没有什么异常。你为什么会认为，他是冒先生派来暗中做掉我们的？”八爷纳闷地问道，毕竟，五年前，阿四救过他的命，虽然，他们见面的次数屈指可数，但是，私底下，他有个时候还是会称他为“四哥”。
<br/>
<br/>        “不知道为什么，最近我这心里总不太踏实。总觉得，我们都快要玩完了。”十爷深沉地回答。
<br/>
<br/>        八爷不以为然地说道：“玩完？怎么可能！等冒先生那边，只要把解药做出来。到时候，我们就能大赚一笔。”
<br/>
<br/>        “对了，傅夜沉是不是你花钱请阿四替你杀的？”十爷转开话题，随口问道。
<br/>
<br/>        八爷微微耸肩，回答道：“我是跟傅夜沉有仇，但还没到想要他的命的地步。再说，他怎么着也是你的堂侄。不看佛面看僧面，我也不会轻易顾杀手去取他的性命。”
<br/>
<br/>        “不是你的话，难道是一爷？不对！难道是冒先生？应该不可能……”十爷若有所思起来，总觉得四爷暗杀傅夜沉这事，有些蹊跷。
<br/>
<br/>        或许是怕引起十爷和八爷的怀疑，傅夜沉跟童筱颖并未有太多交集，打照面的时候，也没有过多的交流。
<br/>
<br/>        只不过，她有发现，傅夜沉似乎是在这古堡里搜寻着什么东西。
<br/>
<br/>        一个星期后，这天中午，或许因为每天起得太早，所以导致她午后便会犯困。
<br/>
<br/>        童筱颖刚准备回自己的房间休息，才进房门，把房门关上，就被人给捂住了嘴巴。
<br/>
<br/>        正当她心想着会不会是大哥的时候，一个男人的声音，小心翼翼地在她耳畔响起：“筱颖，是我，阿旭。”
<br/>
<br/>        因为嘴巴还被他捂着，童筱颖没有出声，也没有挣扎。
<br/>
<br/>        何明旭握住童筱颖垂在腿边的手，将自己手里的东西，塞到了童筱颖的手心里。
<br/>
<br/>        “如果，我没能活过今晚，你就想办法离开这里，帮我把这东西备份了之后再交给警察。”何明旭小声而急促地说完，便放开了童筱颖。
<br/>
<br/>        他没再跟她多说一些不该说的废话，而是直接拉开了房门，蹑手蹑脚地走了出去。
<br/>
<br/>        童筱颖凝望着何明旭离去的背影，直到他的背影在门边的拐角消失，她才缓缓地抬起手来，看了看自己手心里的东西。
