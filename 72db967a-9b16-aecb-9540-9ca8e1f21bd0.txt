
<br/>　　<br/>　　    六个人头骨外加一些凌乱的骨架被摆在破庙的地上，也就是说这至少得是六条人命了。<br/>　　<br/>　　    潘利明和郭茜茜都是一愣，狐疑的眼神落在了王惊蛰的身上，第一个念头就是，他怎么知道这地下埋了这么多的人骨，郭茜茜第二个念头，手就伸向了腰间，握在了枪套上。<br/>　　<br/>　　    看着她的动作，潘利明示意别动，然后蹲下了身子拿起了一根骨头，几十年的办案经验不是白给的，他几乎几眼就看出来一些细节了。<br/>　　<br/>　　    “这些人至少死了得有几十年了……”潘利明说道。<br/>　　<br/>　　    郭茜茜这时在松了口气，死了几十年的人，自然不可能跟王惊蛰有关了，他的年纪才不过二十几岁。<br/>　　<br/>　　    潘利明狐疑的看着这些骨架，特别是那些头骨，他很迷惑的说道：“奇了怪了，这些骨头上的血迹，怎么好像刚沾上不久呢？”<br/>　　<br/>　　    六个头骨上，最顶端的位置，都有一簇血迹十分明显，呈现暗紫色，就好像是被人滴上去后顺着头骨往四周流淌过一样。<br/>　　<br/>　　    新鲜的血自然是呈现鲜红色的，如果入土一段时间就会是暗紫色，但这些人明显死了几十年了，就算头骨上留有血迹的话，也早该被土壤给稀释没了，万万不可能还带有什么色的，这肯定是有点解释不通了。<br/>　　<br/>　　    “你怎么知道这里会埋有人骨的？”潘利明抬头问道。<br/>　　<br/>　　    王惊蛰挠了挠鼻子，有点不知道怎么解释了：“呃，感觉？”<br/>　　<br/>　　    潘利明顿时皱眉：“这个借口太烂了，换一个吧”<br/>　　<br/>　　    王惊蛰两手一摊，无奈的说道：“我要是用我的解释，你们肯定理解不了，所以你干脆就认为这是一种感觉得了，反正这些骨头都被埋了几十年，也不可能和我有什么关系吧？”<br/>　　<br/>　　    郭茜茜冷笑道：“你这么说的话，我倒是还真信，感觉是有可能的，这就跟警犬嗅出来的一样。”<br/>　　<br/>　　    王惊蛰：“……”<br/>　　<br/>　　    潘利明摆了摆手，随意问道：“你是怎么找出来的我就不问了，我就想知道，你找出这些人骨，和现在的案子有什么关系？”<br/>　　<br/>　　    关系肯定是有，但王惊蛰却没办法跟这两个人民公仆解释，因为解释了他们也不懂。<br/>　　<br/>　　    这六个头骨自然就是那一伙妖和尚的，早年间他们被人打死在了破庙里，然后直接就胡乱的埋在了大殿下面，几个妖和尚作恶多年，恶贯满盈，死了也没进去阴曹地府而是留在了阳间，成了孤魂厉鬼，继续盘踞在这破庙中，仍旧为恶。<br/>　　<br/>　　    那个出租车司机说的没错，这庙确实很邪门，原因自然在这几个死了的妖和尚身上，有人过来拆庙那就等于是拆了他们的老巢，那能愿意么？<br/>　　<br/>　　    自然就是，谁拆就祸害谁了。<br/>　　<br/>　　    几十年前，倒霉的那些村民是这么回事，前段时间死了的三个工人，还有凌虚子师徒也是这么回事，这就能解释明白为啥这庙拆不了了。<br/>　　<br/>　　    按理来说，案子到这里就该结束了，因为人骨被挖出来后，只要王惊蛰开坛处理一下，这庙明天就能拆了。<br/>　　<br/>　　    不过这案子算是能处理了，但范二哥交代的那个事还真准了，确实是有人故意在搞事情，原本这几个妖和尚成为孤魂厉鬼的年头太短了，都没过百年左右，是没有多大道行的，顶天了是能祸害一下人，远达不到索命的态度，就拿那三个冤死在庙外的工人来说吧，他们是真够倒霉的了，因为离庙还有几十米远呢，又是大白天的，这几个孤魂野鬼哪里会有那个本事，来了个隔空索命啊。<br/>　　<br/>　　    再一个就是凌虚子师徒，虽然是走江湖的骗子，但崔浩和张堂在招摇撞骗上也是有模有样的，那天晚上他们搬进庙里的东西，有狗血，黑驴蹄子，桃木剑等一系列的辟邪物品，单单要是撞见几个妖和尚，哪怕他们就是一点道术都不会，一股脑的把这些东西全砸出去也有脱身的可能，而远不会死的那么惨。<br/>　　<br/>　　    毕竟，人是骗子，但他俩带来的那些东西，是真能辟邪的。<br/>　　<br/>　　    这就有意思了，明显是有人在背后为这几个孤魂厉鬼来辅助攻击啊，原因就在他们头骨上的那处淡紫色的血迹。<br/>　　<br/>　　    “啪，啪”王惊蛰拍了拍手上的尘土站了起来，说道：“什么关系呢，我暂时还捋不明白，过两天再说吧，那个什么，嗯，今天就到这吧”<br/>　　<br/>　　    王惊蛰说完就有要走的意思，潘利明顿时拦了他一下，皱眉说道：“你到底什么意思？”<br/>　　<br/>　　    “我什么意思你不用问，但我可以告诉你，我的出发点跟你们一样，处理这个案子，解决这个庙”王惊蛰耸了耸肩膀，说道：“不过呢，我们的思路不在一条线上，所以剩下的我不多说了”<br/>　　<br/>　　    撂下一句话，王惊蛰转身就走，但他转过身子的时候，眼神在郭茜茜身上一扫而过时，轻声说道：“记住我刚才那句话，床底下，蕾丝内衣……”<br/>　　<br/>　　    “唰”王惊蛰夹着小碎步，跑了。<br/>　　<br/>　　    郭茜茜一脸呆傻。<br/>　　<br/>　　    潘利明狐疑的看着王惊蛰的背景，又瞅着警花说道：“不是，你俩有啥故事啊，怎么还整出床和内衣来了呢？”<br/>　　<br/>　　    郭茜茜崩溃的说道：“他神经病……”<br/>　　<br/>　　    “神经病怎么能看出你今天穿了身红色的蕾丝内衣？”<br/>　　<br/>　　    “你怎么知道的？”郭茜茜顿时磨了磨牙。<br/>　　<br/>　　    潘利明咽了口唾沫，心里哎呀了一声，嘴他么有点快了，他只得硬着头皮说道：“早上你上班，在更衣室换警服的时候，门没关紧，我正好瞥了一眼”<br/>　　<br/>　　    “你们男人没一个好东西！”郭茜茜愤愤的瞪着他，扭着小腰就出去了。<br/>　　<br/>　　    潘利明接着喊道：“我知道是红色的，是因为我看过现场，不是，那他是怎么知道的呢，还有，床底下是怎么回事啊，你都给我整糊涂了”<br/>　　<br/>　　    “我俩有奸情，行了吧……”
