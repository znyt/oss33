　　第470章
<br/>　　马做的卢飞快，弓如霹雳弦惊。
<br/>　　三千龙字军，扑过来就冲进中军，硬生生趟开一条路，程大雷身边的敌人，一刀一个被他们宰了。三千龙骑，将程大雷护在中央。
<br/>　　赵子龙在马上和程大雷相会，程大雷难免吃惊：“你怎么来了？”
<br/>　　“哈，是刘大哥放心不下大当家，派我们来接应大当家，还请大当家莫怪。”赵子龙有些心虚。
<br/>　　“来得好，来得好。”程大雷看到面前义字军的尸体，轻轻叹了口气：“可惜，来得迟了。”
<br/>　　“三千龙字军都在，另外秦大哥，刘大哥，关二哥，张三哥，高家兄弟带人正在路上，算时间，也快到来。眼前情形怎样，一切听大当家吩咐。”
<br/>　　程大雷心头无名火起，手中大斧指着薛丁山位置：“与我宰了他。”
<br/>　　“得令！”
<br/>　　赵子龙一骑当先，冲着薛丁山扑去，千军万马之中，本就是他施展本事的舞台。身后三千龙字军，也是人人威武，给各路诸侯的帝国军狠狠上了一课。
<br/>　　出色的指挥能力，完美的配合，不弱戎族的骑术。他们以长矛短刀为武器，割头如割草，只杀得各个胆寒，人人心慌。
<br/>　　陆承看见这一幕，只吓得心惊胆战，口中不停道：“他们是哪里来的，他们是哪里来的……”
<br/>　　不怪陆承不明白，其实程大雷心里也糊涂着呢，不过他现在被仇恨冲昏了头脑，眼前有人就好，其他事也不去想他。
<br/>　　事实上，程大雷刚走不久，赵子龙这帮人就出发了。不然凉州并州相距甚远，赵子龙也不能说来就来。
<br/>　　程大雷离开蛤蟆城后，留守的人都知道长安那边打得如火如荼，一帮武夫人人思战，急得抓耳挠腮，上蹿下跳，就如同一群和尚看见美女一般。最后大家聚在一起一合计：大当家单枪匹马出发，万一遇到危险怎么办，不如我们去接应大当家……
<br/>　　说是如此说，其实都想着过来一会天下英雄。
<br/>　　几乎是一拍即合，连一个持反对意见的都无。留下一些人看守山寨，其他人奔赴并州。赵子龙的龙字军是轻骑兵，因为机动性强，三千人全员出动。另外还有秦蛮的蛮字军，关鱼的鱼字军，张肥的张字军，高飞豹的豹字军，高飞虎的虎字军，一共点起一万人，顺着草原就趟了过来。
<br/>　　在进入并州草原后，查探到帝国军的调动，赵子龙率先领着队伍过来，刚好解了程大雷的危机，至于其他人，如今还在赶来的路上。
<br/>　　白平等人的阵亡，程大雷很难过，但他同时不得不承认，用起来，还是自己训练出来的队伍得心应手。
<br/>　　这是成百上千次的摸爬滚打中训练出来的，训练过程中，有人受伤，有人牺牲，所为的不就是今日么。
<br/>　　同样三千人，帝国军根本形不成有效的抵抗力。
<br/>　　在程大雷下达命令后，直扑薛丁山而来。
<br/>　　薛丁山扫了陆承一眼，心中十分不屑，果然百无一用是书生。
<br/>　　“慌什么！我们兵力占优，稳住阵脚，必然能击溃敌军。”
<br/>　　如果战场之上，只是简单的兵力对比，那世间也就没有战争了。
<br/>　　陆承点着头，心道，你说得对，很有道理，可你快想个办法啊。忽然，他一指远方，道：“你看那是什么？”
<br/>　　远处，一支大军感到，战旗如飞，长矛如林，在这个时候，秦蛮等人率领的步兵也赶到了。
<br/>　　“传我军令，杀无赦！”程大雷恶狠狠道。
<br/>　　十几个骑兵冲杀出去，口中高喊着程大雷的命令。
<br/>　　“大当家令，格杀勿论，一个都不许放过。”
<br/>　　这声音一传十，十传百，瞬间传遍全军，喊杀声震天。
<br/>　　其实不用程大雷吩咐，秦蛮等人也会这么做。他们像是三个月没洗澡的人突然见到水一般，口中直呼痛快。
<br/>　　形势发生逆转，一开始是帝国军将程大雷逼在河边，此刻，却是蛤蟆军将帝国军堵在河边。
<br/>　　骑兵在平原上的确有许多优势，但步兵也不是一点作用也无，不然所有人一起攀骑兵的科技树好了，其他兵种也就完全可以抛弃了。
<br/>　　秦蛮这些人来得慢些，一方面是因为机动力不如赵子龙。另外一方面，也是因为他们携带了许多物资。在这些物资中，武器占很大一部分。
<br/>　　有些武器，是帝国从未出现过的。如今的山寨，已经可以自己配置火药，在火药配置成功后，由因火药衍生了一些其他武器，火雷就是其中一种。
<br/>　　每一颗都有拳头大小，外面是烧制好的陶器，中间填上铁弹，石块……在使用时，用明火点燃。
<br/>　　在蛤蟆城的几只队伍中，张肥的张字军是使用火雷用得最好的。
<br/>　　轰轰的爆炸声响起，先不提这些火雷的杀伤力，声势是特别大的。帝国军没遇见过这些东西，首先便被吓破了胆。
<br/>　　张肥手持丈八蛇矛，可以说长驱直入。
<br/>　　秦蛮的蛮字军也杀了进来，梅花阵摆开，几人组成一个小团体，成了乱军之中攻不破的堡垒。
<br/>　　关鱼、高飞豹、高飞虎的人堵住了外面的漏子，来一个杀一个，杀一对杀一双。
<br/>　　陆承最后一点心理防线被击溃，一瞬间面如土色，目光看向薛丁山，将所有希望都寄托在他身上。
<br/>　　“狗贼，纳命来！”
<br/>　　正此时，一员女将杀了过来，座下胭脂马，手中绣绒刀，一刀劈向薛丁山面门。
<br/>　　薛丁山举起银戟反抗，踉跄挡了两招，心中暗暗震惊：这人虽然是一介女流之辈，却半点不弱自己，论招式甚至还在自己之上。
<br/>　　不过他也是豪横之辈，虽惊不惧，片刻便稳住了座下战马，手持银戟攻击，口中高喊一声：“来得好！”
<br/>　　二人打得难解难分，不分胜负，两匹马车轮旋转，却是谁也奈何不得对方。
<br/>　　“啊呀呀！”
<br/>　　忽然一声高喝，在薛丁山头顶响起，薛丁山下意识抬起头来，只见一柄大斧从天而降，从上到下将他劈成两半。
<br/>　　樊梨花吓了一跳，口中喝道：“你着什么急，我马上就能杀了他。”
<br/>　　“那个……你没有和他说话吧？”
<br/>(本章完)
