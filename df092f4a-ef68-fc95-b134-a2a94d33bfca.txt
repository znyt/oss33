　　叶少阳沉默半晌，说道：“道风，你说我们能打赢吗？”
<br/>　　“这问题你不必问，也不必去想，现在唯一可靠的，就是我们自己。”
<br/>　　“是吧。但毕竟还是有压力啊。”叶少阳深吸一口气，吐出去，道，“对了，你上次跟我说，世上没有轩辕上帝，这是真的？”
<br/>　　“司命星君亲口说的，还能有假？”
<br/>　　上次龙虎山之战后，两人短暂交谈时，道风说起这件事，轩辕上帝，只是一个符号，早在开辟了轩辕山不久，他就分解了肉身元神，修为释放在轩辕山境内，润泽草木，形成了一个内部闭环的循环系统，因此才会有远比别的空间都更充裕的灵力。
<br/>　　轩辕上帝羽化之前，在山脉深处闭关修炼，只有他几个弟子能见到他，后来他羽化了，但这消息被他几个弟子封锁，并努力维持轩辕上帝还在的假象，是为了迷惑众生，让外人不敢打轩辕山的主意，不过最重要的，是星月奴可以假借轩辕上帝之口给自己下令，在行为上找个正当的理由，而且让人不敢反抗。
<br/>　　所谓统治人间之类的，都是她自己的决定，最多是和清长风合计的，跟轩辕上帝没关系。
<br/>　　这消息叶少阳知道之后，心中很多疑惑也就解决了：轩辕上帝要真在意人间，就不必开辟轩辕山，就算他后悔了想要收回人间，自己亲自过来把所有人打服就完事了，何必让星月奴费事地折腾几百年也没搞成。
<br/>　　东岳帝君，轩辕上帝，东皇太一，这些之前大家都以为还存在的超级大佬，其实都不在了，最多留下来一些法宝或者精元之类的，为后代弟子提供点帮助。
<br/>　　“不说这个了，你研究了那么久的各派秘籍，可有收获？”
<br/>　　“嗯，我突破了。”叶少阳如实回答，这么多天的苦修，积累到了一定程度，本来差的就是一个契机，结果几天前元宵节楼顶的聚会上，他解开了心结，又多了一番新的感悟，瞬间就悟道了。
<br/>　　造化境界。
<br/>　　从天师到地仙，是一个坎，天师是普通道士修行的大成，到了地仙，虽然不是真的仙人，但体内浊气排空，等于开启了炼气化神的境界，之后历经灵仙、到了上仙巅峰，炼气化神也到了大成，然后又是一个坎。
<br/>　　这个坎迈过去，便等同于炼神返虚，硬实力上没什么变化，但是对于气息的运用已经到了极限，开始修炼元神……
<br/>　　其实到这一步，已经脱离了常人理解的那种法术的范畴，有点像是古代修真之士的那一套，十分玄妙，难以用言语来形容。
<br/>　　叶少阳元神之力本来就极强，在元神修炼上进展神速，才几天功夫，可能就有了别人数年的进展（天才真的没法比），实际上，叶少阳的天赋下限低、但上限很高：最开始的时候，他修炼速度不过比普通人快一些，跟人间那些罕见的天才（如凌宇轩）接近，并未拉开太大实力差距。
<br/>　　随着实力不断提升，他的天赋才更多地显现出来，跟普通天才的差距也越拉越大，尤其是在抵达瓶颈时，他极强的悟性让他总是能够找到突破的契机。
<br/>　　五岁开始修道，修了差不多二十年，到了大部分法师听都没听说过的造化境界，起初他自己都觉得大概是到了瓶颈了，谁知道天赋到了这里才真正显现。别人随着实力提升，进步都是越来越慢，他却是越来越快。
<br/>　　他进一步提升实力的速度，古今罕见。
<br/>　　道风确定这件事后，大为震惊，立刻就拉他去了外面，找了个平坦的地方，跟他斗法。
<br/>　　一边打斗，一边给他示范、指点不足之处，最重要的是，将自己炼神返虚的收获通过斗法一点点传给他。
<br/>　　有道风这个绝代天才言传身教，叶少阳进步神速。
<br/>　　起初，经常打着打着，叶少阳就被干趴下了，只能起来重新来过，后来坚持的时间越来越长，从一分钟到五分钟，再到十分钟。
<br/>　　到了黎明十分，叶少阳已经能逼得道风使出九成法力的情况下而坚持近半小时不败。
<br/>　　“差不多了，我能教你的，都给你了。”又一次把叶少阳打趴下之后，道风说道。
<br/>　　叶少阳抹了抹脸上的泥土，爬起来，气喘吁吁说道：“我还没打赢你呢。”
<br/>　　道风笑道：“就算你彻底开悟，想用一晚上时间就打败我，天方夜谭。”
<br/>　　“那你觉得我多久能打败你。”
<br/>　　道风想了一下说道：“一年到三十年吧。”
<br/>　　叶少阳笑哭。“这什么跨度，太夸张了吧！”
<br/>　　“我是炼神返虚大成，你要打赢我，我想，至少得到无极境界吧。以你修炼速度，也可能一年就够了，也有可能卡个几十年，谁又知道。”
<br/>　　叶少阳觉得也是，机缘这个东西，真的不好说，而且天赋强度也是一个问题。
<br/>　　道风望着他，微笑道：“我也很想看看，你的天赋能支持你走到什么地步。”
<br/>　　叶少阳吸了一口气，吹了一下头发，说道：“但愿能活到那时候。”
<br/>　　随后又问：“那我现在实力算几流了？”
<br/>　　道风想了下说道：“如果大帝鬼王这种算超一流，那你勉强能算一流，或准一流。”
<br/>　　一流强者，虽然还有个准字，但感觉也不错了。
<br/>　　想到道风之前的评价，像他自己、鸠摩罗什、司命星君、钟馗、崔府君等等这一干人算是一流强者，自己而今居然能跟他们站在同一个层次，这……就算是一年前，自己也想都不敢想的啊！
<br/>　　道风看他得意的模样，泼冷水道：“你别高兴太早，你忘了你下一个对手是鬼王，你这实力，十个你也不是他对手。”
<br/>　　叶少阳这才回到现实，深呼吸，说道：“也是最后一个对手了。不管结果怎样，我也算对得起自己了。”
<br/>　　突然又想到一个人，道：“对了，我现在是后卿的对手吗？”
<br/>　　道风想了想，“之前的后卿，是一流高手，比你现在稍强一些，你努把力能打过。你现在……加上你那些手段，还有龙泉剑，与女魃能有一战。”
<br/>(本章完)
