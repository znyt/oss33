<br/>				车子越过高坡之后，前方果然出现了一个灯火通明的小镇，由于地势很高，犹如在天上云端一般，十分显眼。<br/>　　<br/>　　离近了，传来一阵阵躁杂的声音，有音响的声音，有疯狂的呐喊声，偶尔还夹杂着一道道惨叫声。<br/>　　<br/>　　古纯阳道：<br/>　　<br/>　　“这云间小镇，据说以前是麦氏土司的地盘，虽然建国后土司制度已经废除，但是他们的后代仍然以土司自居，再加上这里实在偏僻贫瘠，没有什么资源，渐渐的就展成为了一个自辖区，成了一个三不管地带……”<br/>　　<br/>　　古纯阳故意说这么多，显然是在解释给叶尘听。<br/>　　<br/>　　不料，郭师傅忽然插嘴道：<br/>　　<br/>　　“古老兄有所不知，你说的那些，都已经是老黄历啦！现在的云间小镇，早已经不是麦氏当家做主，而是鬼灵宗的势力范围啦！”<br/>　　<br/>　　“鬼灵宗！”<br/>　　<br/>　　听到这话，不光古纯阳大吃一惊，就连一旁的楚飞烟，也一下子坐直了身体，呼吸也变得急促起来。<br/>　　<br/>　　“没想到那灵智上人，竟然把势力伸到了这里……”<br/>　　<br/>　　古纯阳低声喃喃了一句。<br/>　　<br/>　　郭师傅将两人的神情看在眼中，继续道：<br/>　　<br/>　　“古老兄和桑兄都是道门中人，对这个鬼灵宗，应该比我要熟悉吧？听说那鬼灵宗的宗主灵智上人，有神鬼莫测之能，是神仙级别的人物！不知道比两位如何？”<br/>　　<br/>　　桑木苦笑道：<br/>　　<br/>　　“郭师傅莫要取笑我们！那灵智上人乃是神通境的高人，我们如何能比得了？就算我和古老鬼联手，也远远不是人家的对手！”<br/>　　<br/>　　郭师傅闻言，顿时脸色一肃，<br/>　　<br/>　　“照你们这么说，那位灵智上人，难道比我们武道界的化劲宗师还要厉害？”<br/>　　<br/>　　“非也！”<br/>　　<br/>　　桑木道长继续解释道：<br/>　　<br/>　　“我们修道者只修炼真元，不修炼肉体，所以近战袭杀能力远不及你们武者！就拿咱们现在来说，如果郭师傅突然对我们出手，我们两个自然万万阻挡不了，可是如果让我们做好准备，借助法器和符箓之力，却也未必会输给郭师傅你！”<br/>　　<br/>　　郭师傅听完桑木道长的解释，顿时一脸恍然，原本凝重的神色，也大大缓和了许多。<br/>　　<br/>　　不料，古纯阳忽然开口道：<br/>　　<br/>　　“其实如果我们修道者，能够法武双修，弥补自身的缺陷，实力便能大大提升！”<br/>　　<br/>　　说到这里，古纯阳不由得悄悄望了一眼旁边的叶尘。<br/>　　<br/>　　之前他曾经听叶尘讲过，其实叶尘的修为，比他高不出太多，但是战斗力却有着天壤之别，甚至连于成雄那样的化劲大宗师，都死在了叶尘的手上。<br/>　　<br/>　　当然了，古纯阳又哪里知道，叶尘所修炼的功法，哪怕是在修真界也是最顶级的，而且又有八百年修真记忆，地球上的这些低级传承，自然无法相提并论。<br/>　　<br/>　　桑木道长听到古纯阳这话，不由得笑道：<br/>　　<br/>　　“你说的是上古修道者吧？可惜那些传承早就已经断绝，现如今哪里还有那样的人物？”<br/>　　<br/>　　古纯阳还欲反驳，这时车子忽然停了下来，却是已经来到小镇的入口处。<br/>　　<br/>　　众人下了车，郭师傅还有意犹未尽，<br/>　　<br/>　　“桑兄，古兄，这旁边有一家茶楼，泡茶的手艺很不错，不如让我徒弟他们先去订房间，咱们哥仨再就之前的话题，好好探讨一番，如何？”<br/>　　<br/>　　既然郭师傅都提出来了，古纯阳和桑木也不好拒绝，于是七人便兵分两路，古纯阳、桑木道长和郭师傅，去一旁的茶楼聊天，大壮和阿虎两师兄弟，则带着叶尘和楚飞烟，去酒店订房。<br/>　　<br/>　　……<br/>　　<br/>　　“楚小姐，待会等订好酒店，我们哥俩带你们去玩玩如何？反正师父他们一时半会也回不来！”<br/>　　<br/>　　大壮一边走着，一边笑着对楚飞烟和叶尘开口道，两眼不漏痕迹的，在楚飞烟那妙曼的身子的瞄了一眼。<br/>　　<br/>　　相比之下，那个阿虎就直接多了，眼睛几乎都没有从楚飞烟的身上挪开过，口水都快要流出来了，听到大壮的提议，连忙附和道：<br/>　　<br/>　　“就是就是！我们哥俩也算是这里的常客了，知道很多好玩的地方，保证让你们大开眼界！”<br/>　　<br/>　　楚飞烟自然也察觉到了两人的色心，不由得冷冷一哼，面色冰冷道：<br/>　　<br/>　　“没兴趣！”<br/>　　<br/>　　两人见楚飞烟态度如此冰冷，顿时一阵气馁。<br/>　　<br/>　　这时，叶尘忽然开口了，<br/>　　<br/>　　“我倒是很想见识见识！”<br/>　　<br/>　　反正左右也无事，坐在酒店里实在太无聊了，不如四处走走，而且他对之前古纯阳三人所说的鬼灵宗，也挺感兴趣，这让他想起了修真界的某个宗门。<br/>　　<br/>　　大壮和阿虎顿时一脸嫌弃，他们看中的是楚飞烟，才没有兴趣陪叶尘玩。<br/>　　<br/>　　不料，楚飞烟听到叶尘这么说，当即也跟着道：<br/>　　<br/>　　“既然叶师要去，那我也去！”<br/>　　<br/>　　大壮和阿虎先是一愣，继而大喜，虽然他们不知道楚飞烟为何这么快改变主意，但是只要结果是他们想要的就够了。<br/>　　<br/>　　至于楚飞烟称呼叶尘为叶师，他们误以为是叶尘的名字，所以并没有放在心上。<br/>　　<br/>　　……<br/>　　<br/>　　酒店就在小镇入口附近，四人很快订好了房间之后，在大壮和阿虎的带领下，便朝小镇深处走去。<br/>　　<br/>　　别看这云间小镇不大，而且又是在荒山野岭之中，但这镇子上，茶楼、酒店、酒吧，甚至还有地下赌场、地下擂台等各种非法娱乐场所。<br/>　　<br/>　　不过叶尘对这些地方，实在提不起任何兴趣，大致看了几眼之后，便准备返回酒店了。<br/>　　<br/>　　“没什么好看的，回去吧！”<br/>　　<br/>　　“哦！”<br/>　　<br/>　　楚飞烟自然无不应允，当即乖巧的应了一声，便要跟随叶尘返回酒店，大壮和阿虎一听，顿时气得牙根直痒痒。<br/>　　<br/>　　他们现在算是看出来了，虽然不知道是什么原因，但是这位楚大美女，对眼前这个穿着普通的少年，可以说是言听计从，他们想要多和这个大美女接触，就必须先搞定这个小屁孩才行！<br/>　　<br/>　　“别呀！叶兄弟，真正好玩的地方都还没有去呢！这么着急回去做什么？”<br/>　　<br/>　　阿虎搂着叶尘的肩头，一脸假笑道。<br/>　　<br/>　　大壮也跟着笑道：<br/>　　<br/>　　“是呀！我知道北边有一家地下拍卖场，经常拍卖一些你们修道之人使用的物件，不知道叶兄弟感不感兴趣？”<br/>　　<br/>　　叶尘听到这话，不由得眉头一挑，<br/>　　<br/>　　“还有这种地方？去看看！”<br/>　　<br/>　　(本章完)<br/>　　<br/>　　//<br/>　　<br/>　　天才一秒记住本站地址：。手机版阅读网址：<br/>
