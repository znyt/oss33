
    王惊蛰倒在床上就睡死了过去，今天这酒喝的彻底让他迷失了方向，再喝点都容易喝出酒精中毒了，现在肯定是处于断片的程度了。<br/>　　<br/>　　    另外一张床上的丁武躺下去后眼睛“唰”的一下就睁开了，目视着天花板，眼神呆滞的愣了一会后，丁武翻身起床来到卫生间洗了把脸，再出来后人就精神不少了。<br/>　　<br/>　　    “对你来说，解决问题可能有一百种方式，但对我来说都不如第一百零一种来的管用和干脆，不好意思了惊蛰，我可能要让你失望了”丁武轻声嘀咕了一句后，悄悄的就从房间里出来了，然后关上房门就往外走了。<br/>　　<br/>　　    几分钟后丁武从酒店里出来，站在门口抽着烟等着拦一辆出租车，楼上一间亮着的房间里，林羡鱼刚冲了个澡穿着睡衣敷着面膜准备上床刷会手机就睡觉。<br/>　　<br/>　　    “哗啦”林羡鱼打开空调，来到窗口这拉开窗帘准备把窗户关上，就看见楼下街边站着道身影一手夹着烟，一手招呼着过往的车。<br/>　　<br/>　　    “不是喝的人事不省了么，这么快就醒酒了，他这是要干什么去呢？”林羡鱼嘟囔了一句后就“哎呀”一声，说道：“真让那个碎嘴子说准了？他说让我盯着他点，他果然就偷偷的自己跑了？肿么办啊，我到底要不要看着他啊，唉，好纠结呢”<br/>　　<br/>　　    林羡鱼脑袋里的念头左右徘徊了两圈之后，她扯下面膜快速的给自己重新穿上了衣服：“妈蛋，就折腾姐姐，大半夜的还得去抓男人回来，我是真能糟践自己啊”<br/>　　<br/>　　    “咣当”林羡鱼穿上衣服后就火急火燎的拉开房门跑了出去，等她从酒店里追出来后丁武刚好拦了辆出租车走了，林羡鱼连忙上了自己的车，倒出来之后就追了过去，远远的看见丁武坐的那辆和自己有四五个车位远。<br/>　　<br/>　　    坐上出租车，丁武搓了搓疲惫的脸，跟司机说了一声去顺义卫星路之后，就倒在座椅上眯着睡了过去，这一路得将近一个小时他正好睡一会养养精神，后面跟着的林羡鱼正在逐渐靠近出租车，她本来打算想开过去把出租截停了的，但这个念头却被脑子里的好奇给掐断了。<br/>　　<br/>　　    “妈蛋的，好奇害死猫啊，特别是一个女人的好奇心要是起来了，八匹马是都拦住的······”<br/>　　<br/>　　    此时，就王惊蛰正倒在酒店的床上呼呼大睡，他也完全都没想到丁武今晚是奔着把他灌醉的结果去的，因为两人的酒量都差不多少，王惊蛰开始想的是我喝多了你也肯定就喝多了，但他哪里想到快喝完了的时候丁武跑到厕所里，把肚子里的酒都给抠吐出来了。<br/>　　<br/>　　    他是喝多了，丁武却是清醒的。<br/>　　<br/>　　    时间一晃过了午夜十二点，出租车开到了顺义卫星路附近，司机就把丁武给叫醒了，同时后面的林羡鱼也尾随了过来，看见丁武下车后就熄了车灯停在路边远远的注视着。<br/>　　<br/>　　    丁武下了出租车，瞅了眼附近的门牌号，然后拿出手机拨了个号码，跟那边说了一声自己到了后就挂断了，等了大概不到五分钟左右，一辆黑色的桑塔纳就开了过来，停到丁武的附近，车里面一个年轻人跟他打着招呼。<br/>　　<br/>　　    “出来了，武哥？”<br/>　　<br/>　　    丁武嗯了一声，拉开车门就坐到了副驾驶上，开车的青年扭头说道：“我本打算去接你的呢”<br/>　　<br/>　　    “那倒不用了，我有朋友等着呢，刚把他安顿好”丁武打了个哈欠后，抻着懒腰问道：“我要的东西带来了吧？”<br/>　　<br/>　　    “都在这里呢，给你”开车的人从后座上拎起一个包递给了丁武。<br/>　　<br/>　　    这个青年叫李天鹏，是丁武的狱友，两人在一个监区里蹲着，李天鹏出来的要比他早一年多，在监狱里的时候李天鹏就是给丁武伺候槽子的，他对丁武是一百个服气和佩服，监狱里面的人都拉帮结伙，但唯独丁武事独来独往的，然后不管多牛逼的犯人见到丁武都不敢欺负他，听说是丁武进来后没多久，碰见两个牢霸要收编他，丁武直接就给牢霸送到医院去了。<br/>　　<br/>　　    丁武这人就是，你别欺负我别给我找麻烦，大家相安无事，你要是给我惹毛了，我过不好的话你也别想过安生了。<br/>　　<br/>　　    丁武打开包，看了两眼就有点皱眉了，包里放着一把尖刀还有一捆绳子和两副手套，他抬头问道：“就这个，没有响啊？”<br/>　　<br/>　　    李天鹏摇头说道：“武哥别闹，这是哪？是京城啊，天子脚下，谁敢在这卖枪啊，在这里被抓到都得比别的地方重判一倍，而且枪响了性质也不同，你借我几个胆子也不敢这么干啊”<br/>　　<br/>　　    “行啊，对付用吧，都是杀人夺命的家伙”丁武拉上包之后，又问道：“我之前让你出来后帮我打听的事，打听到了吧？”<br/>　　<br/>　　    “嗯呢，问到了，这个陆程一也不难打听······”<br/>　　<br/>　　    陆程一就是几年前租给丁武他们家房子的人，以前卫星路这一片还没拆迁呢，现在是拆完了之后盖的住宅小区，外面临街是一排的门市房，其中有几间就是陆程一的。<br/>　　<br/>　　    李天鹏打听到的消息是这些年陆程一也没搬走，还是就混在这一片，一是靠收租金生活，二是他和别人经营卖建材，短短五六年的时间积攒了不少的家底，在京城有房有车还有几百万的钱放在银行里，这日子已经算是相当不错的了。<br/>　　<br/>　　    “这边三间门市是他的，再往前走卫星路到头之后有个建材市场，他在里面有个档口卖建材的，白天他基本呆在档口里，晚上就回家了，住在这小区里面，是当年拆迁后给他的安置房，家里有个老婆还有两孩子”<br/>　　<br/>　　    “今晚呢？”丁武淡淡的问道。<br/>　　<br/>　　    “也在家，中午我接到你电话后，就过来蹲点了，晚上跟着他回来的，然后人就再也没出去了，估计现在都睡着了吧”李天鹏挺直接的问道：“武哥，你是打算今天晚上······”<br/>　　<br/>　　    “换个时间吧，他老婆孩子都在家呢，不方便”丁武寻思下后说道：“天棚你回去吧，剩下的我自己来就行了”<br/>　　<br/>　　    李天鹏笑了：“武哥，这车我买两月了，是一辆黑车手续什么的都是假的，我就寻思你可能用得到然后就给买下来了，武哥我给你开个车打个下手都行，让我跟着你呗？”<br/>　　<br/>　　    丁武皱眉说道：“扯淡，你跟着我干嘛？我要干啥你又不是不知道，你都出来了放着好日子不过，和我掺和干什么，听话，回去！”<br/>　　<br/>　　    想和更多志同道合的人一起聊《天命赊刀人》，微信关注“热度网文或者rdww444”与更多书友一起聊喜欢的书
