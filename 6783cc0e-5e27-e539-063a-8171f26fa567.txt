　　无为大道，天知人情，
<br/>　　无为窈冥，鬼见人形，
<br/>　　心言意语，鬼闻人声，
<br/>　　犯禁满盈，地收人魂。
<br/>　　规则，就是规则。
<br/>　　悟了，彻底悟了。但是元神中最后一点神念，也在这一刻消失了。
<br/>　　元神泯灭，化作精魄，在空中散开，又汇聚在一起，朝着轩辕之门飞去。
<br/>　　那几个轩辕山的大佬，还有北斗观那些没死的弟子，都默默地看着精魄升天，表情一个个都是呆滞的。
<br/>　　纵横三界的风之谷主道风，真的……死了？
<br/>　　山山禅师在山谷之中，望着飞舞的精魄，宣了声佛号，又一次念起了往生咒。
<br/>　　精魄飞到半山腰时，清长风猛然回过神来，急忙作法，制造了一道结界，将精魄拦截起来。
<br/>　　“北斗真人，这是为何啊？”
<br/>　　声如洪钟，出自一个白衣老翁之口。
<br/>　　清长风抬头看去，是之前看热闹的那个白衣白袍的老翁，从山顶走下来。
<br/>　　“师叔，我困住这道风精魄，免得他飞走，万一遇到什么机缘，还有聚魂重生的可能。”
<br/>　　精魄是生灵最原始的组成，基本上是溶解不了的，不过，却可以被人轻松困住。
<br/>　　白衣老翁笑道：“哪里有这样的机缘，纵然有，那便是他命不该绝，你又何必做这绝然之事，为自己平生业障。”
<br/>　　清长风不快道：“因为这厮，我北斗观弟子几乎被斩杀殆尽，难道这仇不该报么？”
<br/>　　“人已死，你不是已经报仇了吗？”白衣老翁并不生气，依旧笑着。
<br/>　　“人间有句话，叫斩草除根，万一他有什么机缘，精魄重生，又当如何？”
<br/>　　白衣老翁往前走了几步，道：“一码归一码，他今日已死，便一切全销，北斗你这么做，不是我轩辕山的行径。”
<br/>　　清长风怔了一下，干脆不理他，继续作法，将被困住的道风的精魄朝自己身边拉过来。
<br/>　　白衣老翁轻笑着摇了摇头，单手结印，朝天一指，一道清波像水柱般飞出，将清长风布置的结界打穿，精魄飞出，向着轩辕之门飞去了。
<br/>　　“师叔，你！”清长风震怒。
<br/>　　白衣老翁笑而不语。
<br/>　　眼看着精魄飞走，清长风还想作法阻拦，耳中听见白衣老翁说道：“话不可说尽，势不可用尽，凡事太尽，缘分势必早尽……北斗，不可如此。你还是快去看看南音的肉身吧。”
<br/>　　清长风心头一紧，这要是当务之急，而且看这白衣老翁的意思，如果自己再出手，他还是要阻拦。跟这种长辈真的没法沟通，也没法跟他动真格的，当即冷哼一声，纵身上山去。
<br/>　　所幸，星月奴的肉身完好无缺，清长风让几个弟子搬来几座上佳的法器，摆在修所附近，又安排了一些最得意的弟子，在前后轮班把守，这才稍稍放心，回到正殿去，派人去打理战场，不久之后，有弟子来回报，北斗观有近大半的弟子，都被道风杀了，而且都是他嫡传的内门弟子。
<br/>　　“砰！”清长风一拳砸在桌子上，连香炉都震飞起来。
<br/>　　“贼子，我北斗观多年基业，几乎被这厮刨了个底朝天！”
<br/>　　这时有弟子来报，白泽来访。
<br/>　　清长风只能有请。
<br/>　　白泽跟清长风地位相当，关系也不错，进来之后，也没怎么客气，直接坐在了客座上。
<br/>　　清长风招呼上茶，很快有人捧了两只青瓷茶杯上来，白泽结果一个，揭开杯盖，一团雾气立刻翻腾起来，白泽东贪婪地吸了一大口，雾气被吸走后，能看到杯里有半杯水，里面浸泡着一块咖啡色的石头，光泽饱满，最顶级的血晶石。
<br/>　　这东西在人间也有，遇水之后，能不断产生灵气，但是因为稀少，十分珍贵，在人间要是有人人品爆发得到一块极品血晶石，是法师都会用来镶嵌法器，或者用来炼制外丹，是邪物就放在洞府中，再找一堆灵石之类的搭配起来，成为邪阵，用来修炼，这样才能最大程度发挥血晶石的灵力，不会造成浪费。
<br/>　　但在清长风这里，只是用来招待贵客的“茶叶”。
<br/>　　清长风还沉浸在之前的战斗中，情绪低落，端着茶杯，一口接一口地吞着灵气，一言不发。
<br/>　　“我之前也在山中观战，见证了事件经过。”白泽开口，看了清长风一眼，“真人你损失了不少弟子，甚是可惜，不过……你却也诛杀了道风，这极为难得啊，算起来，总归是不亏的。”
<br/>　　清长风叹道：“亏倒是不亏，能杀道风这厮，的确也解气，不过这也分从什么角度去看，我师妹那里，自然是开心的，她只损伤了一个徒弟，换了道风这个强敌，自然欢喜，可我北斗观，说到底只是个帮忙的，却有如此损失，实不甘心，但道风已诛，却也只能了结，咽下这口气。”
<br/>　　白泽笑道：“岂能一无所获，道风那两件逆天法器，不是落在真人手中了吗？”
<br/>　　清长风一听，随即从身后椅子上摸出了打神鞭和番天印，眼中也是露出了欢喜，握住打神鞭，灌入真气，立刻遇到强烈的抵抗——这是道风专属法器，在里面灌入了极强的一缕念力，神器早就通灵，就算是他，也无法炼化，换了番天印也一样。
<br/>　　但这并没让他感到不快：反正自己有的是时间，十年，二十年，再不行就一百年，总有一天能炼化成功。
<br/>　　虽说以他的地位，身边也不乏极品法器，但打神鞭和番天印毕竟不同，普通的九段光法器根本没法比。
<br/>　　“我到现在有点不敢相信，道风真的死了……”清长风摇着头，感慨地说道，突然想到什么，问白泽：“仙长来找我，是有事？”
<br/>　　“闲聊而已。真人方才怔忡发呆，似乎有心事？”
<br/>　　“我是在想……道风为什么要这么做？”清长风露出沉思之色，“他今日之死，完全是自找的，他为什么要真么做呢？”
<br/>　　“自然是为了刺杀星君。他与那叶少阳是一体的，当初杀了星君六大弟子，也是他所为，双方水火不容，他得知星君肉身在这，有机可趁，正好那假叶少阳又去撩拨，他将计就计，前来行刺，完全有理由。”
<br/>(本章完)
