<br/>				    此时的段夜，手中紧紧的握着剑，唇嘴紧紧的抿着，一张娃娃脸上溅满了鲜血，目光泛红，死死的盯着那被数名仙尊围攻着的血罗。<br/>　　<br/>　　    “别杀他！我要他活着！”段夜冰冷的声音传了出来，听得那仙尊一愣。<br/>　　<br/>　　    “段夜，你受了不轻的伤，先到一旁坐下再说。”云琼仙宗的仙宗说着，想扶他到一旁去坐下，谁知，他却紧紧的抓住他的手。<br/>　　<br/>　　    “师叔。”段夜看着他，一手用力的抓着他的手，双眼泛红的看着他，声音有着一丝的哽咽。<br/>　　<br/>　　    看到段夜这神色，云琼仙宗的仙尊不由的脸色凝重，问：“你怎么了？告诉师叔。”段夜他是知道的，他师兄座下最得意的弟子，可居然眼红了，而且声音还哽咽了？到底出什么事了？<br/>　　<br/>　　    “他打开了黑暗之门，把凤九弄没了。”段夜心头血气翻滚着，此时的他，又害怕，又担心，又自责。<br/>　　<br/>　　    他害怕凤九会因此而死，他担心她会再也回不来了，他更自责，若不是为了救他，她也不会被那旋涡吸进去。<br/>　　<br/>　　    那仙尊听到这话，眉头一跳，他看着段夜，语气中带着连他自己也没察觉的小心翼翼：“凤、凤九？应该不会是那鬼医凤九吧？”<br/>　　<br/>　　    “就是她。”段夜说着，一句话，三个字，让那仙尊整个人一僵，声音也不由的提高了起来。<br/>　　<br/>　　    “什么！竟真是她？她、她被吸进了那黑暗之门？完了完了，这回完了。”他整个人脸色大变，急得在原地打转着。<br/>　　<br/>　　    他们就是为鬼医凤九而来的，原就是为了找她，想去百川城天丹楼的，可谁知，竟在这半道上遇到这事了？那血罗的黑暗之门……这、这怎么得了！<br/>　　<br/>　　    然，就在他心下大乱之时，只听一声惨叫声响起，他猛的反应过来抬头一看，正好看到那几位仙尊砍杀了那血罗的画面，看到这一幕，他心一抖，喃喃的道：“这回就真的完了……出大事了……”<br/>　　<br/>　　    段夜也看到了那血罗被砍杀的一幕，嘴唇微动，却是半句话说不出来。几位仙尊的速度太快，几人连手，没几下便将那血罗给砍杀了，快得让他反应都反应不过来。<br/>　　<br/>　　    眼下，他只关心着，凤九被吸进了黑暗之门会被传送到哪里去？他只关心着，进了那地方，到底是死是活？她还能活着回来吗？又或许，他们能去找她吗？<br/>　　<br/>　　    “哼！这魔族的人真是越发的放肆了！竟敢出来作乱，真当我们四大仙宗的人都是摆设吗？”一位仙尊说着，以他们的实力联手将那血罗砍杀，根本就不费吹灰之力。<br/>　　<br/>　　    “只可惜，我们是来得晚了一步，让这血罗杀了这么多的修士，若是能早到一柱香或者半柱香时间，也许，就不会有这么多人死在血罗的手里了。”另一名仙尊叹了一声，看这一地的尸体摇了摇头。<br/>　　<br/>　　    其他的几人听了则沉默着，看了一地的鲜血，心下也是一叹，这时，想起了什么一般，朝那云琼仙宗的仙尊看去。<br/>
